LibreCAD-Integration in ProNest

LibreCAD für ProNest und ProNest bilden eine integrierte Lösung, sodass man mühelos vom CAD-System zum Schneidsystem wechseln kann. LibreCAD kann genutzt werden als:

  • eigenständige CAD-Anwendung, oder
  • Editor zur Änderung vorhandener Zeichnungen und Teile aus ProNest.

LibreCAD für ProNest besitzt ein kontextabhängiges Hilfesystem, das genauso funktioniert wie die ProNest-Hilfe. Durch Drücken der F1-Taste bei geöffneter Anwendung oder in einem bestimmten Modus bekommen Sie weitere Informationen zur Verwendung von LibreCAD.

 

Jedes in ProNest importierte Teil kann nach dem Import in LibreCAD geöffnet und bearbeitet werden. DWG- und DXF-Dateien können direkt von LibreCAD aus bearbeitet werden. Sie müssen dazu nicht erst das Fenster „Teileliste bearbeiten“ in ProNest aufrufen.

 

Wenn eine Datei in LibreCAD geöffnet und geändert wurde, muss sie als DXF-Datei gespeichert werden.

Installation

LibreCAD for ProNest ist ein CAD-Programm exklusiv für Hypertherm und enthält besondere Funktionen wie „Text formen“ und „Übermaß zuschneiden“. Es wird bei der Installation bzw. beim Update von ProNest automatisch installiert. Wenn das Basisprogramm LibreCAD (nicht LibreCAD für ProNest) bereits auf Ihrem PC installiert ist, wird diese vorhandene Installation vom ProNest-Installationsprogramm nicht überschrieben, sondern es wird ein separates Programm installiert. In diesem Fall können Sie LibreCAD und LibreCAD für ProNest separat verwenden.

Zeichnungen aus ProNest erstellen und bearbeiten

Eine neue Zeichnung aus ProNest erstellen

Klicken Sie im Fenster Teileliste bearbeiten auf die Schaltfläche LibreCAD für ProNest (  ), um das Programm zu starten. Dort können Sie dann eine neue Zeichnung erstellen.

Wenn Sie fertig sind, können Sie entweder mit der Schaltfläche Beenden zu ProNest zurückkehren oder mit der Schaltfläche In ProNest exportieren (  ) eine Zeichnung in ProNest importieren, ohne LibreCAD zu verlassen. Bei der Verwendung der Schaltfläche „In ProNest exportieren“ ist es am besten, wenn das Programm ProNest bereits auf Ihrem Computer läuft.

Öffnen von CAD-Dateien in LibreCAD vor dem Import von Teilen

  1. Wählen Sie im Fenster Teileliste bearbeiten im Abschnitt Teilequellen eine CAD-Datei aus.
    • Sie können mehrere Dateien gleichzeitig auswählen, die in LibreCAD übernommen werden sollen. Halten Sie dafür die Strg-Taste auf Ihrer Tastatur gedrückt und klicken Sie mehrere Dateien an.
  2. Klicken Sie in der Vorschau im Dropdown-Menü auf LibreCAD.
    • Machen Sie die Änderungen an Ihrer Zeichnung und verlassen Sie LibreCAD. Erst dann können Sie in ProNest weiterarbeiten.
  3. Führen Sie in LibreCAD die erforderlichen Änderungen an der CAD-Datei durch (Geometrie ändern, Eigenschaften der Ebenen ändern, Blöcke konfigurieren, Bemaßungen festlegen usw.).
  4. Klicken Sie auf Beenden. Sie werden aufgefordert, Ihre Zeichnung zu speichern, und kehren dann zu ProNest zurück.
  • Wenn Sie ein Teil nach dem Import in LibreCAD ändern, gehen alle vorhandenen Fasen verloren, die dem Teil in ProNest hinzugefügt wurden.
  • Teile, die verkettet, überbrückt oder mit einem „Common Line“-Schnitt geschnitten wurden, können in LibreCAD nicht bearbeitet werden.
  • Wenn Sie mehrere Mengen eines Teils importiert haben (zum Beispiel 10 erforderliche Mengen der Bodenplatte), werden alle Instanzen des Teils bearbeitet. Wenn nur eine Instanz eines Teils bearbeitet werden soll, müssen Sie dieses Teil einzeln in die Teileliste aufnehmen.

Öffnen von CAD-Dateien in LibreCAD nach dem Import von Teilen

Sie können an einem Teil noch einige Änderungen vornehmen, nachdem es in ProNest importiert wurde, zum Beispiel Änderungen an Geometrie, Text, Bemaßungen und „Nur anzeigen“-Informationen.

Ebenendaten können geändert werden, aber es können nur solche neuen Ebenen hinzugefügt werden, die in ProNest bei den Einstellungen für das Importieren von Teilen angegeben sind (ProNest > Einstellungen > Teile importieren > Registerkarte CAD-Import). Neue Ebenen, die in eine Zeichnung aufgenommen werden, aber vor dem Import des Teils nicht in ProNest definiert wurden, werden verworfen.

  1. Klicken Sie mit der rechten Maustaste in der Teileliste oder der Verschachtelung auf ein oder mehrere Teile.
  2. Klicken Sie auf In LibreCAD für ProNest bearbeiten (  ).
  3. Führen Sie in LibreCAD die erforderlichen Änderungen an der CAD-Datei durch.
  4. Klicken Sie auf Beenden. Sie werden aufgefordert, Ihre Zeichnung zu speichern, und kehren dann zu ProNest zurück.
