Zeichnungen nach ProNest exportieren

Zeichnungen können direkt in ProNest importiert werden. Dazu muss das Fenster „Teileliste bearbeiten“ nicht geöffnet werden.

  • Wenn Sie bereit sind, die Zeichnung zu importieren, klicken Sie auf die Schaltfläche In ProNest exportieren (  ).

Öffnen Sie ProNest und öffnen Sie anschließend LibreCAD, damit LibreCAD Ihre Einstellungen zum Import von Teilen (ProNest > Einstellungen > Teile importieren) für die Informationen zu CAD-Ebenen und andere Importeigenschaften verwendet. Dies muss nur einmal durchgeführt werden, solange Sie keine Änderungen an Ihren Einstellungen zum Import von Teilen vornehmen.
