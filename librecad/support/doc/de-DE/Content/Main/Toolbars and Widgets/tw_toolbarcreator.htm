Widget „Symbolleisten-Ersteller“

Mit LibreCAD können Sie für den schnellen Zugriff auf Ihre bevorzugten Werkzeuge benutzerdefinierte Symbolleisten erstellen und andocken.

Neue benutzerdefinierte Symbolleiste erstellen

  1. Öffnen Sie Widgets > Symbolleisten-Ersteller. Das Fenster des Symbolleisten-Erstellers erscheint.
  2. Klicken Sie im linken Feld auf ein beliebiges Werkzeug und anschließend auf die nach rechts weisende spitze Klammer (>). Damit wird es in die Liste für die Symbolleiste im rechten Feld aufgenommen. Wenn Sie im linken Dropdown-Menü eine Kategorie auswählen, werden die im linken Feld erscheinenden Werkzeuge entsprechend gefiltert.
    • Im rechten Feld können Sie die Elemente Ihrer neuen Symbolleiste durch Ziehen und Ablegen neu anordnen.
    • Wenn Sie Elemente aus Ihrer neuen Symbolleiste entfernen möchten, wählen Sie das Werkzeugelement und anschließend die nach links weisende spitze Klammer (<) aus.
  3. Wenn Sie alle gewünschten Werkzeuge hinzugefügt haben, geben Sie Ihrer neuen Symbolleiste im Feld Name einen Namen.
  4. Klicken Sie auf Erstellen.

Eventuell müssen Sie LibreCAD neu starten, damit Ihre neue Symbolleiste in den Listen „Widgets > Symbolleisten“ sowie „Rechtsklick > Symbolleisten“ erscheint.

Eine vorhandene benutzerdefinierte Symbolleiste ändern

  1. Öffnen Sie Widgets > Symbolleisten-Ersteller. Das Fenster des Symbolleisten-Erstellers erscheint.
  2. Wählen Sie aus dem Dropdown-Menü auf der rechten Seite die benutzerdefinierte Symbolleiste aus, die geändert werden soll.
  3. Nehmen Sie die gewünschten Änderungen an den Elementen Ihrer Symbolleiste bzw. ihrer Anordnung in der Symbolleiste vor.
  4. Klicken Sie auf Erstellen, wenn Sie Ihre Änderungen speichern möchten, oder verlassen Sie das Fenster, wenn Sie Ihre Änderungen verwerfen möchten.

Eine benutzerdefinierte Symbolleiste löschen

  1. Öffnen Sie Widgets > Symbolleisten-Ersteller. Das Fenster des Symbolleisten-Erstellers erscheint.
  2. Wählen Sie aus dem Dropdown-Menü auf der rechten Seite die benutzerdefinierte Symbolleiste aus, die gelöscht werden soll.
  3. Klicken Sie auf Zerstören.
