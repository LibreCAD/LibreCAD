Ebenenlisten-Andockwidget

Das Ebenenlisten-Andockwidget bietet eine Liste der Ebenen in der aktuellen Zeichnung und Funktionen zur Verwaltung von Ebenen. In der Eingabezeile oben im Andockwidget können Sie nach Ebenenname filtern (zum Beispiel werden mit dem Filter „*01“ alle Namen angezeigt, die auf „01“ enden).

Ebenenlisten-Funktionen

Werkzeug Symbol Beschreibung
Alle Ebenen anzeigen Macht alle Ebenen in der aktuellen Zeichnung sichtbar.
Alle Ebenen ausblenden Blendet alle Ebenen in der aktiven Zeichnung aus.
Ebene hinzufügen Fügt eine neue Ebene hinzu.
Aktuelle Ebene entfernen Entfernt die ausgewählte Ebene.
Ebenen-Attribute ändern Zur Änderung von Attributen der Ebene einschließlich Name der Ebene.

Funktionen für einzelne Ebenen

Symbol Beschreibung
/ Ebene ein- oder ausblenden.
/ Ebene zur Bearbeitung sperren/freischalten.
Zulassen, dass die Ebene in die gedruckte Zeichnung aufgenommen wird.
/

Konstruktionslinien ein-/ausschalten. Wenn diese Option aktiviert ist, erscheinen alle Linien und Liniensegmente so, als wären sie unendlich lang.
