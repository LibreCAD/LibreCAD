Blocklisten-Andockwidget

Das Blocklisten-Andockwidget bietet Funktionen zur Verwaltung einer Liste von Blöcken in der Zeichendatei.

 

Im unteren Bereich des Andockwidgets wird eine Liste von Blöcken in der aktuellen Zeichnung angezeigt. Die Blöcke im obigen Beispiel tragen die Namen „MyBlock1“, „MyBlock2“ und „MyBlock3“.

Blockfunktionen

Einige dieser Elemente sind auch durch Klicken mit der rechten Maustaste auf eine Ebene erreichbar.

Werkzeug Symbol Beschreibung
Alle Blöcke anzeigen/Fixierung aller Blöcke aufheben Macht alle Blöcke in der aktuellen Zeichnung sichtbar.
Alle Blöcke ausblenden/fixieren Blendet alle Blöcke in der aktiven Zeichnung aus.
Einen Block erstellen Erstellt einen Block aus den ausgewählten Elementen.
Leeren Block hinzufügen Erstellt einen leeren Block, der dann in einem separaten Zeichenfenster bzw. einer Registerkarte bearbeitet werden kann.
Aktiven Block entfernen Löscht den hervorgehobenen Block.
Aktiven Block umbenennen Benennt den hervorgehobenen Block um.
Aktiven Block in separatem Fenster bearbeiten Öffnet ein neues Zeichenfenster zur Bearbeitung eines neuen oder vorhandenen Blocks.
Aktiven Block in einer Datei speichern Speichert den hervorgehobenen Block in einer Datei.
Aktiven Block einfügen Fügt den hervorgehobenen Block in der aktuellen Zeichnung am angegebenen Referenzpunkt ein.
