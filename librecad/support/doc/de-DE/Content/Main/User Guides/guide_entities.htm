Arbeiten mit Objekten

Mithilfe von Werkzeugen können Sie Linien-, Bogen-, Text-, Punkt- und Schraffur-Objekte erstellen:

  • Linien
  • Kreise
  • Kurven
  • Ellipsen
  • Polylinien
  • Bemaßungen
  • Einzeiliger Text
  • Mehrzeiliger Text
  • Punkte
  • Schraffurmuster

Bei manchen Werkzeugen müssen Sie Parameter angeben, zum Beispiel wird beim Werkzeug „Mittendurchmesser“ nach der Eingabe eines Durchmessers gefragt. Bei anderen Werkzeugen können Sie freihändig zeichnen, zum Beispiel beim Rechteck-Werkzeug. Normalerweise können Sie für ein Objekt entweder Koordinaten angeben oder Objekte durch Zeigen und Klicken mit der Maus zeichnen.

 

Im Abschnitt Werkzeuge finden Sie für jedes Werkzeug Informationen zum Nachschlagen.

 

Kreise

Ändern der Kreisgröße

  1. Klicken Sie auf das Kreisobjekt.
  2. Klicken Sie auf einen blauen Anfasser in irgendeinem Quadranten und halten Sie die Maustaste gedrückt. Ziehen Sie den Anfasser zur Veränderung der Kreisgröße und lassen Sie die Maustaste los.

– oder –

  1. Klicken Sie auf Eigenschaften (  ) oder geben Sie prop in der Befehlszeile ein.
  2. Klicken Sie auf das Kreisobjekt.
  3. Ändern Sie die Werte für Mittelpunkt (x), Mittelpunkt (y) und Radius nach Bedarf.
  4. Klicken Sie auf OK.

– oder –

  • Verwenden Sie das Werkzeug Ändern – Skalieren.

Kreise verschieben

  1. Klicken Sie auf das Kreisobjekt.
  2. Klicken Sie auf den blauen Anfasser in der Mitte des Kreises und halten Sie die Maustaste gedrückt. Ziehen Sie den Kreis an eine neue Position und lassen Sie die Maustaste los.

– oder –

  • Verwenden Sie das Werkzeug Ändern – Verschieben/Kopieren.

Kurven

Kurven ändern

  1. Klicken Sie auf das Kurvenobjekt.
  2. Klicken Sie auf einen blauen Anfasser an einem der beiden Enden und halten Sie die Maustaste gedrückt. Ziehen Sie den Anfasser zur Verschiebung des Kurvenendes und lassen Sie die Maustaste los.

– oder –

  1. Klicken Sie auf Eigenschaften (  ) oder geben Sie prop in der Befehlszeile ein.
  2. Klicken Sie auf das Kurvenobjekt.
  3. Ändern Sie die Werte für Mittelpunkt (x), Mittelpunkt (y), Radius, Startwinkel und Endwinkel nach Bedarf. Bei Bedarf können Sie durch Anklicken der Option Umgekehrt die Kurvenrichtung ändern.
  4. Klicken Sie auf OK.

– oder –

  • Verwenden Sie das Werkzeug Ändern – Skalieren.

Kurven verschieben

  1. Klicken Sie auf das Kurvenobjekt.
  2. Klicken Sie auf den blauen Anfasser in der Mitte der Kurve und halten Sie die Maustaste gedrückt. Ziehen Sie die Kurve an eine neue Position und lassen Sie die Maustaste los.

– oder –

  • Verwenden Sie das Werkzeug Ändern – Verschieben/Kopieren.

Ellipsen

Anfasser an einer Ellipse

A. Mittelpunkt

B. Anfasser zur Änderung der Ellipsenlänge (Hauptachse), -breite (Nebenachse) und -orientierung

C. Anfasser zum Zeichnen eines Ellipsenausschnitts als Bogen

D. Anfasser zur Änderung der Breite der Ellipse (Nebenachse)

E. Anfasser zur Änderung der Ellipsenlänge, -breite und -orientierung

 

Ellipsen umformen

  1. Klicken Sie auf das Ellipsenobjekt.
  2. Klicken Sie entweder auf Anfasser B oder E und halten Sie die Maustaste gedrückt. Ziehen Sie den Anfasser zur Umformung der Ellipse und lassen Sie die Maustaste los.

Ellipsenausschnitte erstellen

  1. Klicken Sie auf das Ellipsenobjekt.
  2. Klicken Sie auf Anfasser C und halten Sie die Maustaste gedrückt. Ziehen Sie den Anfasser entlang der Ellipse, bis der korrekte gewünschte Bogen in der Vorschau dargestellt wird.
  3. Lassen Sie die Maustaste los und klicken Sie woanders hin, um den Bogen zu zeichnen.

Ellipsen skalieren

  • Verwenden Sie das Werkzeug Ändern – Skalieren.

Ellipsen verschieben

  1. Klicken Sie auf das Ellipsenobjekt.
  2. Klicken Sie auf den Mittelpunkt bei Anfasser A und halten Sie die Maustaste gedrückt. Ziehen Sie die Ellipse an eine neue Position und lassen Sie die Maustaste los.

– oder –

  • Verwenden Sie das Werkzeug Ändern – Verschieben/Kopieren.

Linien

Länge oder Orientierung einer Linie ändern

  1. Klicken Sie auf das Linienobjekt.
  2. Klicken Sie auf einen blauen Anfasser an einem der beiden Enden und halten Sie die Maustaste gedrückt. Ziehen Sie den Anfasser zur Verschiebung des Linienendes und lassen Sie die Maustaste los.

Alle Linien in einem Polygon werden als aufgebrochen betrachtet. Wenn Sie die Größe eines ganzen Polygons ändern möchten, benutzen Sie „Ändern – Skalieren“. Wenn Sie die Position eines ganzen Polygons ändern möchten, benutzen Sie „Ändern – Verschieben/Kopieren“ oder Strg+C und Strg+V zum Kopieren und Einfügen. Weitere Informationen über Änderungswerkzeuge finden Sie hier.

Linien verschieben

  1. Klicken Sie auf das Linienobjekt.
  2. Klicken Sie in die Nähe der Linienmitte und halten Sie die Maustaste gedrückt. Ziehen Sie die Linie an eine neue Position und lassen Sie die Maustaste los.

– oder –

  • Verwenden Sie das Werkzeug Ändern – Verschieben/Kopieren.

Polylinien

Polylinien ändern

  1. Klicken Sie auf das Polylinienobjekt.
  2. Klicken Sie auf einen blauen Anfasser und halten Sie die Maustaste gedrückt. Ziehen Sie den Anfasser zur Verschiebung des Linienendes und lassen Sie die Maustaste los.

Alle Linien einer Polylinie werden einzeln geändert. Wenn Sie die Größe eines ganzen Polygons ändern möchten, benutzen Sie Ändern – Skalieren.

Polylinien verschieben

  1. Klicken Sie auf das Linienobjekt.
  2. Klicken Sie irgendwo auf die Polylinie, aber nicht auf die blauen Anfasser, und halten Sie die Maustaste gedrückt. Ziehen Sie die Polylinie an eine neue Position und lassen Sie die Maustaste los.

– oder –

  • Verwenden Sie das Werkzeug Ändern – Verschieben/Kopieren.

Punkte

Punkte verschieben

  • Klicken Sie auf den Punkt und halten Sie die Maustaste gedrückt. Ziehen Sie den Punkt an eine neue Position und lassen Sie die Maustaste los.

– oder –

  1. Klicken Sie auf Eigenschaften (  ) oder geben Sie prop in der Befehlszeile ein.
  2. Klicken Sie auf den Punkt.
  3. Geben Sie unter Geometrie – Position (x) und Position (y) die gewünschten Werte ein, um den Punkt an diese Position zu verschieben.

Schraffur

Eine Fläche schraffieren

  1. Klicken Sie auf mindestens ein Objekt. Die besten Ergebnisse erzielen Sie, wenn Sie eine geschlossene Form wie einen Kreis oder ein Polygon auswählen.
  2. Klicken Sie im Werkzeugmenü auf Schraffur (  ) oder geben Sie in der Befehlszeile hatch ein.
  3. Wählen Sie aus dem Dropdown-Menü ein Schraffurmuster aus und geben Sie einen Maßstab an. Optional können Sie einen Winkel für das Schraffurmuster angeben.
  4. Klicken Sie auf OK.

Ein Schraffurmuster ändern

  1. Klicken Sie auf Eigenschaften (  ) oder geben Sie prop in der Befehlszeile ein.
  2. Klicken Sie in der Zeichnung auf das Schraffurmuster.
  3. Ändern Sie die Attribute des Schraffurmusters.
  4. Klicken Sie auf OK, wenn Sie alle Änderungen vorgenommen haben.

Ein Schraffurmuster entfernen

  1. Klicken Sie auf das Schraffurmuster.
  2. Wählen Sie die Taste „Löschen“ aus.
