Neue Schriftarten erstellen

Mit LibreCAD können Sie mühelos neue Schriftarten erstellen. Als Vorlage dafür verwenden Sie eine vorhandene Schriftart. Jedes Zeichen einer Schriftart in LibreCAD ist ein Block, und jeder Block trägt den Namen des entsprechenden Unicode-Zeichencodes.

Eine Schriftart aus einer Vorlage erstellen

Schritt 1

Wählen Sie eine vorhandene Schriftart aus, die als Vorlage für Ihre neue Schriftart dienen soll. Verwenden Sie dabei eine Schriftart, die der gewünschten neuen Schriftart möglichst ähnlich sieht, oder eine Standard-Schriftart wie Times New Roman.

Wenn Sie sich bei der Auswahl der Schriftart nicht sicher sind, können Sie sich im Dialogfenster Schriftart konvertieren eine Vorschau der Schriftarten ansehen. Sie erreichen dieses Dialogfenster unter Werkzeuge > MText > Schriftart konvertieren (  ).

 

Schritt 2

Suchen Sie die Schriftart im Datei-Explorer auf Ihrem PC heraus.

 

Wenn Sie eine Schriftart verwenden, die Sie aus AutoCAD oder Microsoft konvertiert haben, finden Sie die Datei für diese Schriftart in dem Pfad, der unter Anwendungseinstellungen > Pfade > Schriftarten angegeben ist.

 

Wenn Sie eine LibreCAD-Schriftart verwenden, finden Sie die Schriftart in folgendem Ordner: C:\Program Files (x86)\Hypertherm CAM\LibreCAD\resources\fonts

 

Schritt 3

Öffnen Sie die Schriftart-Datei (*.LFF) in LibreCAD. Wählen Sie dazu Datei > Öffnen und stellen Sie den Dateityp LFF Font (*.lff) ein. Wechseln Sie dann zum entsprechenden Ordner und klicken Sie die Schriftart-Datei an, die Sie öffnen möchten.

 

Schritt 4

Die Schriftart-Datei wird nun in LibreCAD geöffnet. Beachten Sie, dass jedes Zeichen mit dem entsprechenden Unicode darunter in einer Zeile angezeigt wird. Beachten Sie außerdem, dass in der Blockliste jedes Zeichen unter seinem Unicode-Namen als einzelner Block aufgeführt wird.

 

So bearbeiten Sie ein Zeichen:

  1. Suchen Sie in der Blockliste den Block für das entsprechende Zeichen heraus.
  2. Klicken Sie in der Blockliste auf den Block.
  3. Klicken Sie auf Aktiven Block in separatem Fenster bearbeiten(  ).

Der aktive Block wird nun in einem separaten Fenster geöffnet.

 

Schritt 5

Nehmen Sie die erforderlichen Änderungen an dem Zeichen vor. Verlassen Sie die Registerkarte nach der Bearbeitung des Zeichens wieder. Dadurch gelangen Sie zurück in die gesamte LFF-Datei für die Schriftart. Sie müssen den Block nicht in einer einzelnen Datei speichern. Der Block wird stattdessen in der LFF-Datei gespeichert.

 

Wiederholen Sie diesen Vorgang, bis alle Zeichen wunschgemäß geändert wurden.

 

Schritt 6

Wir empfehlen, die geänderte Schriftart in einer anderen Datei zu speichern, sodass der Zustand der Original-Schriftart erhalten bleibt.

 

So speichern Sie die Schriftart als andere Datei:

  1. Klicken Sie auf Datei.
  2. Klicken Sie auf Speichern unter.
  3. Wechseln Sie zu dem Ordner, in dem die Schriftart-Datei gespeichert werden soll. Wir empfehlen folgenden Ordner: C:\Program Files (x86)\Hypertherm CAM\LibreCAD\resources\fonts.
  4. Wählen Sie aus der Liste der Dateitypen LFF-Datei (*.lff) aus.
  5. Geben Sie einen Dateinamen an, aber keine Dateierweiterung. Verwenden Sie also beispielsweise „MeineSchriftart“, aber nicht „MeineSchriftart.lff“.
  6. Klicken Sie auf Speichern.

Wenn Sie die Schriftart im LibreCAD-Schriftarten-Ordner speichern möchten (siehe Schritt 6.3), müssen Sie dazu möglicherweise die Ordnerberechtigungen ändern, damit Sie Dateien in diesem Ordner speichern können. Sollte dies nicht möglich sein, speichern Sie die Schriftart-Datei in dem Ordner, den Sie in Anwendungseinstellungen > Pfade > Schriftarten festgelegt haben.

 

Beispiel

 

Schritt 1 – Italict.lff ist die Schriftart, die als Vorlage verwendet werden soll.

 

Schritt 2 – Italict.lff ist eine Standardschriftart von LibreCAD und in folgendem Ordner gespeichert: C:\Programme (x86)\Hypertherm CAM\LibreCAD\resources\fonts.

 

Schritt 3 – Die Schriftart wird geöffnet.

 

 

Schritt 4 – Zur Bearbeitung eines Zeichens wird ein einzelner Zeichenblock ausgewählt und in einem neuen Fenster bzw. einer neuen Registerkarte bearbeitet. Die folgende Abbildung zeigt die Registerkarte mit der Blockzeichnung für das Zeichen „A“.

 

 

Schritt 5 – Ändern Sie den Zeichenblock nach Bedarf und verlassen Sie dann die Registerkarte.

 

Schritt 6 – Speichern Sie die Schriftart in einer neuen Datei. (Nicht vergessen: Die Dateierweiterung (.lff) darf nicht am Ende des Dateinamens angegeben werden; wählen Sie stattdessen einfach aus der Dropdownliste mit den Dateitypen „LFF“ aus.)

 

Die Schriftart steht nun zur Verwendung in einer Zeichnung zur Verfügung.
