Arbeiten mit Ebenen

Ebenen erstellen

  1. Klicken Sie im Andockwidget „Ebenenliste“ auf Ebene hinzufügen (  ).
  2. Geben Sie einen Namen für die Ebene ein. Geben Sie außerdem an, ob die Ebene eine Konstruktionsebene ist, und passen Sie die Standard-Stiftoptionen nach Bedarf an.
  3. Klicken Sie auf OK.

Ebenen entfernen

Wenn eine Ebene entfernt wird, werden mit der Ebene auch sämtliche Geometrien auf dieser Ebene entfernt.

  1. Klicken Sie im Andockwidget „Ebenenliste“ auf die Ebene, die entfernt werden soll.
  2. Klicken Sie auf Ebene entfernen (  ) oder öffnen Sie mit der rechten Maustaste das Kontextmenü und klicken Sie dort auf „Ebene entfernen“.
  3. Klicken Sie auf OK.

Eine Ebene zur aktiven Ebene machen

  • Klicken Sie im Andockwidget „Ebenenliste“ einfach auf die Ebene, damit sie aktiv wird.

Ändern der Farbe und anderer Attribute einer Ebene

  1. Klicken Sie die Ebene an, die geändert werden soll.
  2. Klicken Sie auf Ebene ändern (  ).
  3. Nehmen Sie die erforderlichen Änderungen an den Einstellungen der Ebene vor.
  4. Klicken Sie auf OK.

Die Ebene eines Objekts ändern

  1. Klicken Sie ein oder mehrere Objekte an.
  2. Klicken Sie bei den Änderungswerkzeugen entweder auf Attribute (  ) oder auf Eigenschaften (  ). 
  3. Klicken Sie im Dropdown-Menü Ebene eine andere Ebene an.
  4. Klicken Sie auf OK.

Ebenen sortieren

Die Ebenen werden in der Ebenenliste in alphanumerischer Reihenfolge angezeigt. Die Reihenfolge der Ebenen hat jedoch nichts mit der Reihenfolge zu tun, in der die Objekte in der Zeichnung erscheinen. Mit den Befehlen unter Änderungswerkzeuge > Schaltfläche Sortieren (  ) können Sie Objekte nach oben oder unten verschieben.

Aktion Taste Ergebnis
Nach oben verschieben [Pos1] Das ausgewählte Objekt wird an die oberste Position verschoben.
Nach unten verschieben [Ende] Das ausgewählte Objekt wird an die unterste Position verschoben.
Nach oben, über Objekt [Bild auf] Das ausgewählte Objekt wird um eine relative Position nach oben verschoben.
Nach unten, über Objekt [Bild ab] Das ausgewählte Objekt wird um eine relative Position nach unten verschoben.

Konstruktionsebenen erstellen

Um eine vorhandene Ebene zur Konstruktionsebene zu machen, gehen Sie so vor:

  • Klicken Sie neben dem Namen der Ebene auf Konstruktionsebene ( ).

Wenn die Schaltfläche grün dargestellt wird, heißt das, dass die Ebene als Konstruktionsebene eingerichtet ist. Wenn die Schaltfläche ausgegraut dargestellt wird, ist die Ebene keine Konstruktionsebene.

Neu erstellte Ebenen, die als Konstruktionsebenen verwendet werden sollen, dürfen keinen zugeordneten Prozess in ProNest haben. Damit wird vermieden, dass unerwünschte Geometrien aus Konstruktionsebenen in ProNest importiert werden.

Wenn beispielsweise eine Schneidebene als Konstruktionsebene verwendet wird, um Konstruktionslinien zu erstellen, werden die Linienobjekte, aus denen die Konstruktionslinien bestehen, in ProNest als Geometrie erkannt. Wenn hingegen eine neue Ebene verwendet wird, die ProNest nicht erkennt, werden die Linien, aus denen die Konstruktionslinien bestehen, von ProNest ignoriert.

Konstruktionslinien erstellen

So erstellen Sie Konstruktionslinien unbegrenzter Länge:

  1. Vergewissern Sie sich, dass die Konstruktionsebene (  ) aktiv ist.
  2. Erstellen Sie eine Linie mithilfe eines beliebigen Linien- oder Polylinien-Werkzeugs. Zur Vorschau wird über dieser Linie eine neue Linie unbegrenzter Länge gezeichnet.

    3. Wenn Sie Konstruktionslinien auf einer Ebene platzieren, der ein Prozess in ProNest zugeordnet ist, müssen Sie die Linienobjekte, aus denen die Konstruktionslinien bestehen, vor dem Import der Zeichnung in ProNest löschen. Wir empfehlen, neue Ebenen zu erstellen, denen in ProNest kein Prozess zugeordnet ist, und diese Ebenen als Konstruktionsebenen zu verwenden. So vermeiden Sie, dass unerwünschte Geometrie aus Konstruktionsebenen in ProNest importiert wird.

  3. Passen Sie die Linie an und klicken Sie dann an irgendeine Stelle in der Zeichnung, um die Linie zu platzieren. Diese Auswahl wird zum Referenzpunkt für die Erstellung einer weiteren Konstruktionslinie.

  4. Platzieren Sie nun beliebig viele weitere Konstruktionslinien durch Wiederholung von Schritt 3.

  5. Wenn Sie mit der Zeichnung von Konstruktionslinien fertig sind, verlassen Sie den Modus zur Erstellung von Konstruktionslinien mit einem Rechtsklick oder der Esc-Taste.

Konstruktionslinien verbergen

Um Konstruktionslinien unsichtbar zu machen, während sämtliche sonstige Geometrie auf einer Ebene sichtbar bleibt, gehen Sie so vor:

  • Klicken Sie neben dem Namen der Ebene auf Konstruktionsebene ( ).

Die Schaltfläche „Konstruktionsebene“ wird ausgegraut und alle Konstruktionslinien werden ausgeblendet.

Die von Ihnen erstellten Konstruktionslinien werden durch Betätigung der Schaltfläche „Konstruktionsebene“ nicht entfernt, sondern nur ausgeblendet.

Konstruktionslinien entfernen

  1. Klicken Sie eine oder mehrere Konstruktionslinien an.
  2. Wählen Sie die Taste „Löschen“ aus.

Durch diesen Vorgang werden alle unbegrenzten Konstruktionslinien gelöscht, nicht aber die Geometrie, mit der die Konstruktionslinien erstellt wurden (Linien und Polylinien).
