Arbeiten mit Vorlagen

Vorlagen sind Prototypen von Zeichnungen, bei denen man grundlegende Parameter und Einstellungen speichern kann, sodass man nicht jede neu erstellte Zeichnung wieder neu konfigurieren muss. Zu diesen Parametern und Einstellungen gehören die Zeichnungseinstellungen wie das Papierformat, die Haupt-Maßeinheit und das Hauptformat sowie das Format für Bemaßungen. Vorlagen können außerdem Ebenen und Ebenen-Konfigurationen, Linienarten und -stärke, Stiftfarbe und weitere Zeichnungselemente wie Rahmen enthalten. Diese Einstellungen werden bei allen Zeichnungen übernommen, die mit der Vorlage erstellt werden.

 

Sie können eine vorhandene Zeichnung als Vorlage benutzen oder die Standard-Leerzeichnung als Vorlage benutzen.

Einen Titelblock in die leere Standard-Zeichnungsvorlage aufnehmen

Sie können einen Titelblock oder andere Objekte in die leere Standardzeichnung aufnehmen, die beim Start von LibreCAD bzw. bei der Erstellung einer neuen Zeichnung erscheint. Dies ist dann sinnvoll, wenn bestimmte Objekte in allen Ihren CAD-Zeichnungen vorhanden sein müssen.

  1. Öffnen Sie die DXF-Standardvorlagendatei in LibreCAD: C:\Users\[Windows Username]\AppData\Local\Hypertherm CAM\ProNest 2019\ProNestTemplate.DXF
  2. Nehmen Sie die erforderlichen Änderungen an der Zeichnungsvorlage vor.
  3. Klicken Sie auf Speichern (  ).

Die Vorlagendatei wurde nun aktualisiert. Erstellen Sie zur Überprüfung eine neue Zeichnung.

Wenn Sie die Vorlagendatei nicht finden können: Die Standard-Zeichenvorlage wird bei der Erstellung einer neuen Zeichnung in der Befehlszeile angezeigt.

Eine vorhandene Zeichnung als Vorlage verwenden

  1. Klicken Sie auf Datei.
  2. Klicken Sie auf Neu von Vorlage.
  3. Wählen Sie die Zeichnung aus, die Sie als Vorlage verwenden möchten.
  4. Klicken Sie auf Öffnen.

Bearbeiten Sie die Zeichnung nach Bedarf. Beachten Sie dabei, dass die Zeichnungsvorlage nicht geändert wird. Sie müssen die neue Zeichnung als separate Datei speichern.
