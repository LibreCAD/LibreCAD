Mehrfach-Befehlseingaben mit Variablendateien (erweitert)

Man kann Befehlseingaben kombinieren und innerhalb einer einzelnen Zeile eingeben. Dazu werden die Befehle und andere Eingaben mit Semikolons getrennt. Wenn man in der Kommandozeile li;0,0;10..0;0..10;-10..0;c;k eingibt, wird ein Quadrat der Größe 10 × 10 gezeichnet. Man kann eine Liste von Befehlen (getrennt durch Zeilenschaltungen) kopieren und über die Kommandozeilen-Schaltfläche mit „Mehrere Befehle einfügen“ wieder einfügen (unten rechts im Kommandozeilen-Andockwidget).

 

Befehlseingaben können auch aus Textdateien geladen werden. Tragen Sie die Befehle und andere Eingaben in eine Textdatei ein, immer durch eine Zeilenschaltung getrennt. Legen Sie beispielsweise eine Textdatei an und tragen Sie folgende Befehle ein:

li

0,0

@10,0

@0,10

@-10,0

c

k

Mit diesen Befehlen wird ein Quadrat der Größe 10 × 10 gezeichnet.

Mehrfach-Befehlseingaben verwenden

  1. Erstellen Sie eine Textdatei, die die Befehlseingaben enthält.
  2. Wählen Sie in LibreCAD die Befehlszeilen-Schaltfläche aus (  ) und klicken Sie im Dropdown-Menü auf Befehlsdatei laden.
  3. Suchen Sie die Datei heraus, klicken Sie sie an und anschließend auf Öffnen.

Die in der Textdatei spezifizierte Geometrie erscheint dann in der aktuellen Zeichnung.

 

Variablen in Mehrfach-Befehlseingaben

Mehrfach-Befehlseingaben können einer Variablen zugewiesen werden. Außerdem können Variablen andere Variablen enthalten, diese werden dann rekursiv gelesen:

a=ci;0,0;10

b=ci;10,0;10

c=\a;\b;kill

\c

Geben Sie die einzelnen Zeilen des obigen Textes in der Befehlszeile ein. Wenn Sie \c eingeben, werden zwei Kreise mit einem Radius von 10 gezeichnet, die sich überschneiden. Das Zeichen \ ist ein Escape-Zeichen, durch das die Befehlszeile den Variablennamen als Aktion interpretieren kann. Im obigen Beispiel wird \c aufgelöst zu ci;0,0;10;ci;10,0;10;kill.

Variablen mit Mehrfach-Befehlseingaben verwenden

  1. Erstellen Sie eine Textdatei mit variablen Mehrfach-Befehlseingaben.
  2. Laden Sie die Datei. Wählen Sie dazu Optionen aus.
  3. Klicken Sie auf Anwendungseinstellungen.
  4. Klicken Sie auf die Pfade.
  5. Klicken Sie auf Durchsuchen (  ) und suchen Sie die Variablendatei heraus.
  6. Klicken Sie auf OK.
  7. Klicken Sie noch einmal auf OK, um die Anwendungseinstellungen zu speichern.
  8. Starten Sie LibreCAD neu.
  9. Geben Sie \c in der Befehlszeile ein.

Die Datei wird nun geladen.
