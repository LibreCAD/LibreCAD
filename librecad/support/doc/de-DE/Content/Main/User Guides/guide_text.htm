Arbeiten mit Text

Man kann Text in künstlerische Zeichnungen und Zeichen integrieren und damit Informationen und Anmerkungen einschließlich Titelblöcken darstellen. Mit den Werkzeugen Text oder MText können Sie ein- oder mehrzeiligen Text erstellen.

  • Bei der Verwendung von Text erzielen Sie die besten Ergebnisse, wenn Sie (mit dem Befehl explode) Text in Geometrie aufbrechen, bevor Sie eine Zeichnung an ProNest senden. Sie sollten Text an der Grundlinie links (  ) ausrichten und MText an Unten links, damit beim Senden einer Zeichnung an ProNest keine Geometrieprobleme auftreten.
  • Beachten Sie, dass Text in ProNest immer in Geometrie umgewandelt wird, wenn er sich nicht auf einer Beschriftungsebene befindet, aber die Ergebnisse sind besser, wenn Text vor dem Export nach ProNest in LibreCAD aufgebrochen wird.
  • In ProNest sollten Sie vor dem Import eines Teils aus LibreCAD auf jeden Fall die Glättung aktivieren, damit stufige Liniensegmente in Bögen und glatte Profile umgewandelt werden (ProNest > Einstellungen > Teile importieren > Objekte glätten aktivieren. Versuchen Sie es anfangs mit einem Wert von 0,001 bis 0,005 für die Glättungstoleranzen).

Einzeiligen Text erstellen

  1. Klicken Sie im Werkzeugmenü auf Text ().
  2. Geben Sie im Feld Text den Text ein, der in der Zeichnung erscheinen soll.
  3. Ändern Sie die Schriftart-Einstellungen, Ausrichtung und Symbole und konvertieren Sie Microsoft-Schriftarten bei Bedarf.
  4. Klicken Sie auf OK.
  5. Klicken Sie auf die Zeichnung, um den Text zu platzieren, oder geben Sie die Platzierungskoordinaten in die Befehlszeile ein (die Textausrichtung wird bei der Eingabe der Koordinaten als Einfügepunkt verwendet).

Mehrzeiligen Text erstellen

  1. Klicken Sie im Werkzeugmenü auf MText ().
  2. Geben Sie im Feld Text den Text ein, der in der Zeichnung erscheinen soll. Mit dem Dropdown-Menü „Symbol einfügen“ können Sie Sonderzeichen in den Text einfügen, zum Beispiel die Symbole für Grad oder Copyright.
  3. Konvertieren Sie bei Bedarf Microsoft-Schriftarten und legen Sie die Abstandswerte fest.
  4. Stellen Sie die Ausrichtung und den Einfügewinkel ein (der Winkel kann leer bleiben, wenn der Text in einem Winkel von 180° eingefügt werden soll).
  5. Klicken Sie auf OK.
  6. Klicken Sie auf die Zeichnung, um den Text zu platzieren, oder geben Sie die Platzierungskoordinaten in die Befehlszeile ein (die Textausrichtung wird bei der Eingabe der Koordinaten als Einfügepunkt verwendet).

Text ändern

  1. Klicken Sie im Werkzeug „Ändern“ auf Eigenschaften (  ). Das Text-Dialogfenster erscheint erneut, und Sie können den Text und die Ebenen- und Farbinformationen ändern.
  2. Klicken Sie den Text an, der geändert werden soll.
  3. Ändern Sie die relevanten Eigenschaften.
  4. Klicken Sie auf OK.

Text um ein Objekt herum formen

Exklusiv für Hypertherm gibt es die Funktion Text formen in LibreCAD, mit der Sie Text um ein einzelnes Linien-, Bogen- oder Kreisobjekt herum formen können.

  1. Zeichnen Sie eine Form.
  2. Platzieren Sie Text mit den Werkzeugen Text oder MText. Geben Sie im Dialogfenster „Text“ bzw. „MText“ auf jeden Fall eine Ausrichtung an. Wenn Sie sich bei der Ausrichtung nicht sicher sind, versuchen Sie es mit der Option „Unten links“.
  3. Aktivieren Sie den Modus „Text formen“. Dazu klicken Sie entweder bei den Änderungswerkzeugen auf Text formen (  ) oder geben in der Befehlszeile st ein.
  4. Klicken Sie auf den Text, den Sie formen möchten.
  5. Bewegen Sie den Cursor um eine Form herum. Ändern Sie bei Bedarf den Wert für den Versatz oder über das Kontrollkästchen die Textrichtung. Wenn der Text sich an der gewünschten Position befindet, klicken Sie, um ihn zu platzieren.

Um den geformten Text noch genauer steuern zu können, empfehlen wir, die Zeichenketten mit dem Änderungswerkzeug Text in Buchstaben aufbrechen (  ) in einzelne Buchstaben aufzubrechen.

Geformten Text durch die Bereichsgrenzen des Objekts ausrichten

  1. Zeichnen Sie eine Form.
  2. Platzieren Sie Text mit den Werkzeugen Text oder MText. Geben Sie im Dialogfenster „Text“ bzw. „MText“ auf jeden Fall die Ausrichtung „Mitte zentriert“ an.
  3. Aktivieren Sie den Modus „Text formen“. Dazu wählen Sie entweder bei den Änderungswerkzeugen Text formen (  ) aus oder geben in der Befehlszeile st ein.
  4. Stellen Sie in der Symbolleiste den Versatz für den Modus „Text formen“ ein.
    •  Bei einzeiligem Text sollten Sie die Hälfte der Texthöhe als negativen Wert als Versatz verwenden. Bei einer Texthöhe von 1 sollte also beispielsweise der Versatzwert –0,5 betragen.
    • Der Versatz kann durch Eingabe eines Wertes und Auswahl der Enter-Taste eingestellt und aktualisiert werden.
  5. Klicken Sie auf den Text, den Sie formen möchten.
  6. Bewegen Sie den Cursor um eine Form herum. Wenn der Text sich an der gewünschten Position befindet, klicken Sie, um ihn zu platzieren.

 

Mit dem Änderungswerkzeug Übermaß zuschneiden (  ) können Sie unerwünschte Geometrie schnell mit einem Klick entfernen.
