Arbeiten mit der Befehlszeile

Die Befehlszeile ist eine Alternative zum Arbeiten mit der Maus, wenn man Werkzeuge auswählen und zeichnen möchte. Bei der Erstellung von Zeichnungen kann man mit der Tastatur schneller und präziser Werkzeuge auswählen und Koordinaten eingeben.

 

Die Befehlszeile bietet noch weitere nützliche Funktionen, die mit der Maus nicht erreichbar sind:

  • Mehrfach-Befehlseingaben
  • Ein Rechner

Die Befehlszeile wird in einem eigenen Andockwidget dargestellt und besteht aus drei Komponenten:

  1. Eingabeaufforderung.
  2. Befehls-Ausgabefenster.
  3. Einer Schaltfläche (  ), über die ein Dropdown-Menü mit folgenden Einträgen angezeigt wird:
    • Tastencode-Modus ein/aus.
    • Eine Befehlsdatei laden.
    • Mehrere Befehle einfügen.

 

Im Anhang finden Sie eine vollständige Liste der Befehlszeilen-Befehle.

Funktionen in der Befehlszeile eingeben

Die Befehle, die zur Verfügung stehen, werden auf den Referenzseiten für jede Werkzeugkategorie angezeigt. Die Befehlszeile kann auf unterschiedliche Arten aktiviert werden:

  • Beginnen Sie einfach mit der Eingabe eines Befehls (zum Beispiel rect) und drücken Sie dann „Enter“.
  • Drücken Sie die Leertaste, geben Sie einen beliebigen Befehl ein und drücken Sie dann „Enter“.
  • Drücken Sie Strg+M, geben Sie einen beliebigen Befehl ein und drücken Sie dann „Enter“.
  • Drücken Sie die Umschalttaste, geben Sie einen beliebigen Befehl ein und drücken Sie dann „Enter“.
  • Geben Sie bei eingeschaltetem Tastencode-Modus einen aus zwei Buchstaben bestehenden Befehl ein (zum Beispiel ci). Es muss nicht „Enter“ gedrückt werden.
  • Bei der Eingabe von Befehlen in der Befehlszeile können Sie die Tabulatortaste zur automatischen Vervollständigung verwenden. Geben Sie einen Teil eines Befehls ein, zum Beispiel cir, und drücken Sie dann die Tabulatortaste. Wenn für den Teilbefehl nur ein Befehl infrage kommt, wird der vollständige Befehl im Text Eingabefeld angezeigt. Wenn für den Teilbefehl mehrere Befehle infrage kommen, wird eine Liste möglicher Befehle im Befehlsfenster angezeigt.

Je nach dem verwendeten Befehl sind möglicherweise in der Befehlszeile oder in der Symbolleiste weitere Eingaben erforderlich.

  • Statt der Enter-Taste kann auch die Leertaste gedrückt werden, wenn folgende Einstellung aktiviert ist: Optionen > Anwendungseinstellungen > Standardwerte > Beim Drücken der Leertaste Befehle auswerten. Dies ist die Standardeinstellung.
  • Statt durch Komma getrennter Koordinaten können Koordinaten auch über das numerische Tastenfeld im Format X..Y eingegeben werden. So ist beispielsweise die Eingabe 10..20 äquivalent zu @10,20. Möglicherweise sind die zwei Punkte schneller einzugeben als das Komma.
  • Durch den Tastencode-Modus kann man Zwei-Buchstaben-Befehle eingeben und muss nach dem Eintippen des Befehls nicht mehr „Enter“ drücken.
  • Man kann c bzw. u eintippen, gefolgt von „Enter“, anstatt close bzw. undo einzutippen.

Arbeiten mit dem Befehlszeilen-Rechner

  1. Geben Sie Cal in der Befehlszeile ein, um den Rechnermodus zu aktivieren.
  2. Geben Sie die erforderlichen Gleichungen ein. Mit dem Rechnermodus kann man außerdem bestimmte Werte wie Pi abrufen. Die Ergebnisse werden im Befehlsfenster angezeigt.
  3. Geben Sie Cal erneut in der Befehlszeile ein, um den Rechnermodus wieder zu verlassen.

Siehe die Liste der Operatoren und Funktionen des Rechners.

  • Die Konstante Pi ist in LibreCAD definiert als 3,14159265359.
  • Trigonometriefunktionen arbeiten mit dem Bogenmaß (Bogenmaß = Winkelgrad*Pi/180).
