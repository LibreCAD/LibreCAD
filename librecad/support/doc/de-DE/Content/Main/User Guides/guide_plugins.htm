Arbeiten mit Plugins

Plugins in LibreCAD sind spezielle Befehle, mit denen man Objekte in einer Zeichnung besser erstellen, ändern oder Informationen darüber abrufen kann. Plugins können neben der Änderung von Objekten noch vielen weiteren Zwecken dienen, zum Beispiel können sie bestimmte Dateitypen unterstützen, wie z. B. das ESRI-Shapefile-Plugin.

 

Name Beschreibung
Ausrichten Ausrichtung ausgewählter Objekte an einer Referenz. Hierzu werden die Endpositionen zweier Anfangspunkte definiert.
ASCII-Punkte lesen

Einlesen von Punkten aus einer Textdatei.

In jeder Zeile der Datei wird ein Punkt definiert. Dazu werden eine ID, eine X-Koordinate, eine Y-Koordinate, eine Z-Koordinate und ein optionaler Code angegeben. Die Felder können durch Komma, Tabulator oder Leerzeichen getrennt sein. Das Dezimaltrennzeichen ist der Punkt (.). Die Punkte können mit einer Linie, ID oder Koordinate verknüpft sein, und Codefelder können als Text gezeichnet werden.
Teilen

Aufteilung einer Linie oder eines Kreises in n Abschnitte.

Am Ende jedes Abschnitts kann eine Gitterlinie erscheinen, die das Ende anzeigt. Die Größe dieser Gitterlinie kann als Prozentsatz der Segmentlänge definiert werden Mit dem Teilen-Werkzeug kann die Linie oder der Kreis am Ende jedes Abschnitts zerlegt werden.
Zahnrad-Plugin Zeichnen eines Zahnrades. Dafür werden der Mittelpunkt des Zahnrads sowie Definitionsparameter wie die Anzahl der Zähne, der Modul usw. angegeben.
ESRI Shapefile

Import einer Formdatei mit Geo- bzw. raumbezogenen GIS-Vektordaten (Karten).

Anmerkung: Bis zum Abschluss des Importvorgangs wird LibreCAD gesperrt. Bei großen Dateien kann dies sehr viel Zeit in Anspruch nehmen.
Objekte auflisten Listet die ausgewählten Objekte mit ihren Eigenschaften wie ID, Ebene, Farbe, Linienart, Linienstärke und Koordinaten auf.
PIC-Datei lesen Import von Diagrammen in der Grafiksprache PIC.
Plot-Plugin

Plotten einer mathematischen oder parametrischen Funktion im Koordinatensystem der Zeichnung.

Hierfür benötigen Sie die Formel, den Startwert, den Endwert und die Schrittweite. Der Plot kann eine Linie/Linien, eine Polylinie oder ein Spline sein.
Gleiche Eigenschaften

Übernahme der Eigenschaften eines Referenzobjekts in ausgewählte Objekte.

Die geänderten Eigenschaften sind Ebene, Farbe, Linienart und Linienstärke.
Beispiel-Plugin Zeichnen einer Linie durch Angabe der X- und Y-Koordinaten der Endpunkte.
