Arbeiten mit Bemaßungen

LibreCAD enthält verschiedene Bemaßungswerkzeuge zur Bemaßung von Linien, Kreisen, Bögen und Winkeln. Unter Aktuelle Zeichnungseinstellungen finden Sie einen Überblick über die Möglichkeiten, die Gestaltung von Bemaßungen zu ändern.

Einstellungen für Bemaßungen konfigurieren

Bevor Sie Bemaßungen in Ihre Zeichnung aufnehmen, sollten Sie prüfen, ob die Einstellungen für Bemaßungen konfiguriert sind.

  1. Klicken Sie auf Optionen.
  2. Klicken Sie auf Aktuelle Zeichnungseinstellungen.
  3. Klicken Sie auf Bemaßungen.
  4. Vergewissern Sie sich, dass alle auf dieser Seite aufgeführten Eigenschaften für Ihre Zeichnung geeignet sind.
  5. Klicken Sie auf OK, um Ihre Änderungen nur für diese Zeichnung zu speichern, oder auf Vorlage aktualisieren, um diese Einstellungen auch für zukünftige Zeichnungen zu übernehmen.

So gewährleisten Sie, dass Bemaßungen in ProNest sichtbar sind

Sie können Ihre Ebenen und ProNest-Einstellungen so konfigurieren, dass die Bemaßungsinformationen auf reinen Anzeigeebenen in ProNest sichtbar sind.

  1. Vergewissern Sie sich, dass in ProNest bei den Einstellungen für das Importieren von Teilen eine reine Anzeigeebene („Nur anzeigen“) vorhanden ist (Einstellungen > Teile importieren > Registerkarte CAD-Import > Nur anzeigen).
  2. Vergewissern Sie sich in LibreCAD, dass die Ebene „Nur anzeigen“ bei den Ebenen Ihrer Zeichnung vorhanden ist. Es könnte sinnvoll sein, die Farbe dieser Ebene zu ändern, damit eine Prozess-Ebene nicht mit der Ebene „Nur anzeigen“ verwechselt werden kann.
  3. Zeichnen Sie die Bemaßungen auf der Ebene „Nur anzeigen“ in LibreCAD.
  4. Importieren Sie das Teil in ProNest.

Informationen aus der Ebene „Nur anzeigen“ erscheinen für ein Teil in der Verschachtelung, sind aber in der Teileliste nicht sichtbar. Außerdem können Sie den Teilebericht für ein einzelnes Teil aufrufen. Klicken Sie dazu mit der rechten Maustaste auf ein Teil in der Teileliste und wählen Sie „Teilebericht“ aus. Hier werden auch Informationen aus der Ebene „Nur anzeigen“ angezeigt.

Zeichnungseinstellungen für Bemaßungen ändern

Sie können die Eigenschaften für Bemaßungstexte, -Linien, -Pfeile und -Gitterlinien in der Zeichnung ändern.

  1. Klicken Sie dazu auf Optionen > Aktuelle Zeichnungseinstellungen > Bemaßungen.
  2. Ändern Sie die gewünschten Parameter.
  3. Klicken Sie auf OK, um Ihre Änderungen zu speichern, oder auf Abbrechen, um sie zu verwerfen.

Die Änderungen an den Zeichnungseinstellungen für Bemaßungen gelten für alle vorhandenen Bemaßungen in der aktuellen Zeichnung und für alle neuen Zeichnungen, die nach dem Speichern der Änderungen erstellt werden.

Horizontale Linien bemaßen

  • Verwenden Sie das Werkzeug Horizontale Bemaßung (  ) und klicken Sie auf die Start- und Endpunkte für das zu bemaßende Objekt.

Aktivieren Sie „Am Objekt fassen“ (  ) oder „An Endpunkten fassen“ (  ) für präzise lineare Bemaßungen.

Vertikale Linien bemaßen (90°)

  • Verwenden Sie das Werkzeug Vertikale Bemaßung (  ) und klicken Sie auf die Start- und Endpunkte für das zu bemaßende Objekt.

Gewinkelte Linien bemaßen

  • Verwenden Sie das Werkzeug Ausgerichtete Bemaßung (  ) und klicken Sie auf die Start- und Endpunkte für das zu bemaßende Objekt.

Bögen und Kreise bemaßen

  • Verwenden Sie das Werkzeug Radiale Bemaßung (  ) oder das Werkzeug Durchmesser-Bemaßung (  ) und klicken Sie auf das zu bemaßende Kreis- oder Bogen-Objekt.

Winkel bemaßen

  • Verwenden Sie das Werkzeug Winkel-Bemaßung (  ) und wählen Sie die beiden Linien aus, zwischen denen der Winkel bemaßt werden soll.

Individuelle Führungslinien hinzufügen

Durch die Bemaßungswerkzeuge werden je nach Ihren aktuellen Zeichnungseinstellungen automatisch Führungslinien eingefügt. Bei Bedarf können Sie mit dem Führungs-Bemaßungswerkzeug (  ) weitere Führungslinien hinzufügen.

Bemaßungstext verschieben

Verschieben der Bemaßungszahl, aber nicht der Linien oder Pfeile der Bemaßung.

  1. Klicken Sie auf das Bemaßungsobjekt.
  2. Klicken Sie auf den blauen Anfasser in der Mitte der Bemaßungszahl und halten Sie die Maustaste gedrückt. Ziehen Sie den Text an eine neue Position und lassen Sie die Maustaste los.

Die Position von Bemaßungen verschieben

Verschieben der Position der gesamten Bemaßung einschließlich Bemaßungszahl, Linien und Pfeile der Bemaßung. Dadurch wird die Bemaßung möglicherweise von der Form getrennt.

  1. Klicken Sie auf das Bemaßungsobjekt.
  2. Klicken Sie unten an der Bemaßungslinie auf den blauen Anfasser, der der Form am nächsten ist, und halten Sie die Maustaste gedrückt. Ziehen Sie die Bemaßung an eine neue Position und lassen Sie die Maustaste los.

Bemaßungslinien verlängern oder verkürzen

Zur Verlängerung oder Verkürzung von Bemaßungslinien. Hierbei bleiben die Bemaßungslinien mit der Form verknüpft.

  1. Klicken Sie auf das Bemaßungsobjekt.
  2. Klicken Sie auf den blauen Anfasser neben einem Bemaßungspfeil und halten Sie die Maustaste gedrückt. Ziehen Sie den Anfasser zur Verlängerung/Verkürzung der Bemaßungslinie an eine neue Position und lassen Sie die Maustaste los.
