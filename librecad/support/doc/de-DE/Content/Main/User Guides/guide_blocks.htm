Arbeiten mit Blöcken

Blöcke sind kleine, wiederverwendbare Zeichnungen häufig verwendeter Objekte wie Schrauben, Titelblöcke usw. Eingefügte Blöcke (auch als Inserts bezeichnet) setzen sich aus geometrischen Formen wie Linien und Bögen zusammen und können auch Text und Bemaßungen enthalten. LibreCAD enthält mehrere Bibliotheken mit Blöcken, die in eine Zeichnung eingefügt werden können. Ebenso können Sie eigene Blöcke erstellen.

 

Zur Verwaltung von Blöcken stehen zwei Andockwidgets zur Verfügung:

  • Mit dem Andockwidget „Blockliste“ werden Blöcke innerhalb der aktuellen Zeichnung verwaltet, und
  • mit dem Andockwidget „Bibliotheks-Browser“ können Sie eine Sammlung verfügbarer Blöcke aufrufen und Blöcke in die aktuelle Zeichnung einfügen.

Weitere Informationen über Andockwidgets finden Sie hier und über den Bibliotheks-Browser finden Sie hier.

 

Beim Einfügen in eine Zeichnung wird ein Block als einzelnes Objekt behandelt. Man kann Blöcke aus der Blockliste heraus mehrmals in die gleiche Zeichnung einfügen, zum Beispiel an verschiedenen Positionen, in verschiedenen Maßstäben und Drehwinkeln. Die eingefügten Blöcke sind alle verknüpft. Änderungen an einem eingefügten Block werden in allen Instanzen des Blocks übernommen.

Einen Block aus einer vorhandenen Zeichnung erstellen

  1. Zeichnen Sie das Objekt, das Sie in einen Block umwandeln.
  2. Wählen Sie alle Objekte aus, aus denen der Block bestehen soll. Klicken Sie dazu auf die einzelnen Objekte oder wählen Sie mehrere Objekte in einem Bereich aus.
  3. Klicken Sie im Andockwidget „Blockliste“ auf Block erstellen (  ).
  4. Geben Sie einen Referenzpunkt an.
  5. Geben Sie dem Block einen Namen und klicken Sie auf OK.

Der Block erscheint jetzt in der Blockliste und kann mehrmals in die Zeichnung eingefügt oder in eine Datei oder die Bibliothek gespeichert werden.

Einen Block aus einem leeren Block erstellen

  1. Klicken Sie im Andockwidget „Blockliste“ auf Leeren Block hinzufügen (  ).
  2. Geben Sie dem Block einen Namen und klicken Sie auf OK.
  3. Klicken Sie auf Aktiven Block in separatem Fenster bearbeiten (  ). Dadurch öffnet sich eine Registerkarte mit einem Zeichendokument für einen leeren Block.
  4. Zeichnen Sie den Block. Der Ursprung der Zeichnung (0,0) wird zum Referenzpunkt für den Block.
  5. Verlassen Sie die Registerkarte mit der Block-Zeichnung. Der Block wird gespeichert und kann nun in eine Zeichnung eingefügt werden.

Blöcke in eine Zeichnung einfügen

  1. Klicken Sie in der Blockliste auf den Block, der eingefügt werden soll.
  2. Klicken Sie im Andockwidget „Blockliste“ auf Aktiven Block einfügen (  ).
  3. Legen Sie die Einfügeposition fest. Klicken Sie dazu irgendwo in die Zeichnung oder geben Sie die Koordinaten in der Befehlszeile an.

Blöcke mit der Symbolleiste (erweitert) in eine Zeichnung einfügen

  1. Klicken Sie in der Blockliste auf den Block, der eingefügt werden soll.
  2. Klicken Sie im Andockwidget „Blockliste“ auf Aktiven Block einfügen (  ).
  3. Legen Sie die Parameter fest, die auf der Symbolleiste erscheinen.

  • Parameter im Zusammenhang mit Blocktransformationen: Winkel für die Drehung und Faktor für die Skalierung.
  • Parameter im Zusammenhang mit Blockmustern: Matrix (Rastergröße) und Abstand zwischen Spalten und Zeilen der Matrix.
  1. Legen Sie die Einfügeposition fest. Klicken Sie dazu irgendwo in die Zeichnung oder geben Sie die Koordinaten in der Befehlszeile an.

Eine Instanz eines Blocks aus einer Zeichnung entfernen

  1. Klicken Sie in der Zeichnung auf den Block.
  2. Klicken Sie auf die Taste „Löschen“.

Einen Block und alle Instanzen aus einer Zeichnung entfernen

Wenn eine Instanz eines Blocks aus einer Zeichnung entfernt wird, wird nur der Block entfernt, der auf einer Ebene platziert wurde, der Block wird aber weiterhin in der DXF-Datei gespeichert. Wenn Sie einen Block vollständig aus der Zeichnungsebene und der DXF-Datei entfernen möchten, gehen Sie so vor:

  1. Klicken Sie in der Blockliste auf den Block.
  2. Klicken Sie in der Blockliste auf Aktiven Block entfernen (  ).

Blöcke auflösen

Wenn Sie irgendeine Geometrie oder einzelne Objekte verändern möchten, müssen Sie Blöcke möglicherweise wieder aufbrechen. Nach dem Einfügen eines Blocks in eine Zeichnung wird der Block zunächst als Ganzes behandelt und kann nur verändert werden, wenn man ihn aufbricht.

  1. Geben Sie in der Befehlszeile explode ein oder klicken Sie bei den Bearbeitungsoptionen auf Aufbrechen (  ).
  2. Klicken Sie auf den Block, der aufgebrochen werden soll.
  3. Drücken Sie die Enter-Taste.

Blöcke ändern

Wenn der Block in eine Datei gespeichert wurde, müssen Sie diese Datei nur öffnen, Ihre Änderungen vornehmen und vor dem Verlassen speichern.

 

Wenn der Block noch nicht in eine bestimmte Zeichnung gespeichert wurde, aber Bestandteil der Zeichnung ist, an der Sie gerade arbeiten, können Sie den Block so ändern:

  1. Klicken Sie in der Blockliste auf den Block, der geändert werden soll.
  2. Klicken Sie in der Blockliste auf Aktiven Block in separatem Fenster bearbeiten (  ).
  3. Nehmen Sie die gewünschten Änderungen an dem Block vor.
  4. Verlassen Sie die Block-Zeichnung.

Der Block wird aktualisiert.

Blöcke aus der Bibliothek einfügen

  1. Klicken Sie im Andockwidget „Bibliotheks-Browser“ auf den Block, der eingefügt werden soll.
  2. Klicken Sie auf Einfügen.
  3. Klicken Sie auf einen Referenzpunkt in der Zeichnung oder geben Sie in der Befehlszeile die Koordinaten ein.

Sie können den Block so oft einfügen, wie Sie möchten. Mit der Esc-Taste oder einem Rechtsklick verlassen Sie den Block-Einfügemodus.

Blöcke in einer Datei speichern

  1. Klicken Sie in der Blockliste auf den Block, der gespeichert werden soll.
  2. Klicken Sie auf Aktiven Block in einer Datei speichern (  ).
  3. Geben Sie einen Namen und einen Speicherort für die Datei an und klicken Sie dann auf OK.
