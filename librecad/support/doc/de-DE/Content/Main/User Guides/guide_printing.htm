Drucken

In dieser Druckanleitung gehen wir davon aus, dass die Zeichnung im Maßstab 1:1 erstellt wurde.

Schnelldruck

  1. Klicken Sie auf Datei > Druckvorschau oder auf Druckvorschau (  ).
  2. Unter Optionen > Aktuelle Zeichnungseinstellungen >  Papier können Sie das Papierlayout einstellen bzw. überprüfen.
  3. Wählen Sie im Druckvorschau-Modus die Schaltfläche An Seite anpassen (  ) aus. Damit wird die Zeichnung so skaliert, dass sie auf die Seite passt und auf der Seite zentriert ist.
  4. Klicken Sie auf Datei >  Drucken oder auf Drucken (  ).
  5. Nehmen Sie gegebenenfalls die notwendigen Einstellungen im Drucker-Dialogfenster vor.
  6. Klicken Sie auf Drucken.

Maßstabsgetreu drucken

Wenn Sie eine Zeichnung ohne Zeichenrahmen-/Titelblock-Vorlage, aber in einem bestimmten Maßstab drucken möchten, sind einige zusätzliche Schritte erforderlich.

  1. Klicken Sie auf Datei > Druckvorschau oder auf Druckvorschau (  ).

  2. Unter Optionen > Aktuelle Zeichnungseinstellungen >  Papier können Sie das Papierlayout einstellen bzw. überprüfen.
  3. Wählen Sie im Druckvorschau-Modus An Seite anpassen (  ) aus. Damit wird der größtmögliche Maßstab für die im vorigen Schritt definierte Papiergröße festgelegt.
  4. Wählen Sie in der Symbolleiste „Druckvorschau“ aus dem Dropdown-Menü den gewünschten Maßstab aus.
  5. Klicken Sie in der Symbolleiste „Druckvorschau“ auf Auf Seite zentrieren (  ).

  6. Durch Verschieben der Seite hinter der Zeichnung kann die Zeichnung auf der Seite verschoben werden. Klicken Sie mit der Maus irgendwo auf die Zeichnungsfläche und halten Sie die Maustaste gedrückt. Ziehen Sie das Papier nun in die gewünschte Position und lassen Sie die Maustaste los. Beim gleichzeitigen Drücken der Umschalttaste kann das Papier nur horizontal bewegt werden, beim Drücken der Strg-Taste nur vertikal.

  7. Klicken Sie auf Datei > Drucken oder auf Drucken (  ).

  8. Nehmen Sie gegebenenfalls die notwendigen Einstellungen im Druck-Dialogfenster vor.
  9. Klicken Sie auf Drucken.

Maßstabsgetreu drucken mit Bemaßungen

Bei einer Zeichnung im Maßstab 1:1 mit Bemaßungen muss die Größe der Bemaßungs-Elemente für die Druckausgabe angepasst werden. Diese Größen hängen vom gewünschten Maßstab der Druckausgabe ab. Wenn sie nicht angepasst werden, gilt die definierte Textgröße, und diese ist möglicherweise für Zeichnungen in großem oder kleinem Maßstab ungeeignet.

Sobald der Druckmaßstab definiert wurde (siehe voriger Abschnitt), kann die Größe der Bemaßungen bei den „Aktuellen Zeichnungseinstellungen“ über den Parameter Allgemeiner Maßstab angepasst werden. Wir empfehlen, den Wert aus dem Druckmaßstab als Eingabewert für den Allgemeinen Maßstab zu verwenden (also 50 bei einem Druckmaßstab von 1:50). Bei einem Druckmaßstab der Zeichnung von 1:50, einem Allgemeinen Maßstab von 50 für die Bemaßungen und einer Standard-Textgröße von 2,5 mm beträgt die Druckgröße des Bemaßungstexts: 1/50 * 50 *2,5 = 2,5 mm.

 

Text, der (mit den Werkzeugen Text oder MText) in die Zeichnung aufgenommen wird und nicht zu einer Bemaßung gehört, kann über die Zeichnungseinstellungen nicht geändert werden. Text muss über die Änderungswerkzeuge bei den Eigenschaften geändert werden.

 

Überprüfen Sie in der Druckausgabe, wie die Bemaßungselemente am Ende aussehen und ob sie Ihren Anforderungen entsprechen.

Maßstabsgetreu drucken mit Rahmen und Titelblock

Sie können eine Zeichnung mit Zeichenrahmen-/Titelblock-Vorlage, aber in einem bestimmten Maßstab drucken. Starten Sie mit einer Zeichnung im Maßstab 1:1.

  1. Klicken Sie auf Datei > Druckvorschau oder auf Druckvorschau (  ).
  2. Unter Optionen > Aktuelle Zeichnungseinstellungen >  Papier können Sie das Papierlayout einstellen bzw. überprüfen.
  3. Klicken Sie im Druckvorschau-Modus auf An Seite anpassen (  ). Damit wird der größtmögliche Maßstab für die im vorigen Schritt definierte Papiergröße festgelegt. Achten Sie darauf, dass noch genug Platz für Rahmen und Titelblock bleibt.
  4. Wählen Sie in der Symbolleiste „Druckvorschau“ aus dem Dropdown-Menü den gewünschten Maßstab aus.
  5. Klicken Sie auf das Kontrollkästchen Fest, damit der Maßstab sich nicht ändern kann.
  6. Fügen Sie den Seitenrahmen und den Titelblock in die Zeichnung ein.
  7. Bestätigen Sie das Layout noch einmal im Modus Druckvorschau.
  8. Klicken Sie auf Datei > Drucken oder auf Drucken (  ).
  9. Nehmen Sie gegebenenfalls die notwendigen Einstellungen im Druck-Dialogfenster vor.
  10. Klicken Sie auf Drucken.

Drucken auf mehreren Seiten

Wenn Sie eine Zeichnung in einem bestimmten Maßstab drucken möchten, der nicht auf das verfügbare Papier passt, müssen Sie sie mehrseitig drucken. In diesem Fall wird die Zeichnung in Teilen ausgegeben, die zur fertigen Zeichnung zusammengesetzt werden können.

  1. Klicken Sie auf Datei > Druckvorschau oder auf Druckvorschau (  ).
  2. Unter Optionen > Aktuelle Zeichnungseinstellungen > Papier können Sie das Papierlayout einstellen bzw. überprüfen.
  3. Wählen Sie in der Symbolleiste „Druckvorschau“ aus dem Dropdown-Menü den gewünschten Maßstab aus.
  4. Klicken Sie auf Anzahl der Seiten berechnen… (  ). Dadurch werden im Modus „Druckvorschau“ mehrere Seiten nebeneinander angezeigt, mit der aktuellen Zeichnung in der Mitte. Über Optionen > Aktuelle Zeichnungseinstellungen > Papier > Anzahl der Seiten: Horizontal und Vertikal kann die Anzahl der Seiten eingestellt werden.
  5. Durch Verschieben der Seite hinter der Zeichnung kann die Zeichnung auf der Seite verschoben werden. Klicken Sie mit der Maus irgendwo auf die Zeichnungsfläche und halten Sie die Maustaste gedrückt. Ziehen Sie das Papier nun in die gewünschte Position und lassen Sie die Maustaste los. Beim gleichzeitigen Drücken der Umschalttaste kann das Papier nur horizontal bewegt werden, beim Drücken der Strg-Taste nur vertikal.
  6. Klicken Sie auf Datei > Drucken oder auf Drucken (  ).
  7. Nehmen Sie gegebenenfalls die notwendigen Einstellungen im Druck-Dialogfenster vor.
  8. Klicken Sie auf Drucken.
