Befehlszeilen-Aliasse

Einige Befehle in LibreCAD haben mehrere Formen. Die Langform ist der native, nicht übersetzte Befehl, und die Kurzform ist der Alias. So lautet der lange, native Befehl für Textform beispielsweise „shapetext“, die Kurzform bzw. der Alias lautet „st“. In der Konfigurationsdatei librecad.alias können für jeden Befehl Aliasse definiert werden.

Format: <alias>[TAB]<command-untranslated>

 

Beispiel: l       Zeile

Die Alias-Datei kann mit einem beliebigen Texteditor wie Notepad bearbeitet werden. Sie finden die Datei hier: C:\Users\{Benutzername}\AppData\Local\LibreCAD\librecad.alias.

  • Sie dürfen nur den Alias ändern, nicht die lange, unübersetzte Form.
  • Damit Änderungen wirksam werden, müssen Sie LibreCAD möglicherweise neu starten.
