Eine Zeichnung einrichten

Wenn Sie gerade erst in die Arbeit mit LibreCAD einsteigen, sollten Sie darauf achten, dass Ihre Zeichnung mit den richtigen Einheiten, dem richtigen Maßstab usw. konfiguriert wird. Dies müssen Sie nur einmal tun, und alle Ihre Änderungen werden in der Standardvorlage gespeichert, die LibreCAD bei der Erstellung einer neuen Zeichnung verwendet.

Maßstab

Die besten Ergebnisse erzielen Sie, wenn Zeichnungen im Maßstab 1:1 erstellt werden. LibreCAD bietet entsprechende Möglichkeiten zur Vergrößerung/Verkleinerung, sodass die gesamte Zeichnung auf den Bildschirm passt, Sie aber auch kleinste Details in der Vergrößerung erkennen können.

 

Drucken

Bei der Ausgabe muss die Zeichnung so skaliert werden, dass sie auf die Seite passt. Die Ausgabe kann eine Druckseite, eine PDF-Datei oder ein anderes Bildformat sein.

 

Die Textgröße für die Bemaßung wird bei den „Aktuellen Zeichnungseinstellungen“ auf der Registerkarte „Bemaßungen“ unter „Allgemeiner Maßstab“ festgelegt. Mit dem Allgemeinen Maßstab werden der Bemaßungstext, Pfeile und verwandte Parameter auf eine geeignete Größe für das erforderliche Seitenformat angepasst.

 

Der Wert für den Allgemeinen Maßstab sollte umgekehrt zum Maßstab für das Drucken angesetzt werden. Wenn der Druckmaßstab beispielsweise 1:10 Zoll beträgt, sollte der Allgemeine Maßstab 10 betragen (10:1). Wenn man den Allgemeinen Maßstab als Kehrwert des Druck-Maßstabs festlegt, werden die Bemaßungstexte auf der gedruckten Zeichnung in der definierten Größe ausgegeben. Die Zeichnung wird so herunterskaliert, dass sie auf die Seite passt, und der Bemaßungstext wird so hochskaliert, dass er lesbar ist. Der Druckmaßstab kann über die Druck-Vorschaufunktion von LibreCAD festgelegt werden. Weitere Informationen finden Sie im Druck-Leitfaden.

 

Verwenden Sie bei der Festlegung der Maßstabswerte folgende Parameter:

  • Optionen > Aktuelle Zeichnungseinstellungen > Papier > Format: Durch die Auswahl von Papiergröße und Orientierung wird der Druckmaßstab für die endgültige Ausgabe festgelegt. Mit dem Druckmaßstab wird der Allgemeine Maßstab festgelegt.
  • Aktuelle Zeichnungseinstellungen > Bemaßungen > Allgemeiner Maßstab: Bei der Festlegung des Allgemeinen Maßstabs werden der Bemaßungstext und verwandte Parameter angepasst.
  • Wenn Sie die Papiergröße festlegen, nachdem Sie das Objekt gezeichnet haben, aber bevor Sie mit der Bemaßung der Zeichnung beginnen, ist es einfacher, den Druckmaßstab, den Allgemeinen Maßstab und anschließend auch den geeigneten Zeilenabstand für Bemaßungen zu ermitteln.
  • Lassen Sie bei der Festlegung des Druckmaßstabs auf der Zeichnung auf jeden Fall genug Platz für Bemaßungslinien und -text.
  • Man kann beim Ausdruck einer Zeichnung zwar einen beliebigen Maßstab verwenden, es gibt für verschiedene Zeichnungsarten jedoch bestimmte gängige Maßstäbe. Im Anhang finden Sie unter Zeichnungsmaßstäbe einige Beispiele.

Ebenen

ProNest nimmt automatisch Ebenen in LibreCAD auf, die in der Registerkarte „CAD-Import“ unter den Einstellungen für das Importieren von Teilen angegeben sind. Sie können die Ebenen noch ändern oder neue erstellen, denken Sie aber daran, dass ProNest beim Import eines Teils aus LibreCAD alle Ebenen ignoriert, die bei Ihren Einstellungen für das Importieren von Teilen nicht angegeben sind (ProNest Einstellungen > Teile importieren > CAD-Import).

 

Mit Ebenen lassen sich Zeichnungen besser organisieren. Die Benutzer können zusammenhängende Objekte damit besser platzieren und verwalten. Man kann sich Ebenen als übereinanderliegende, transparente Zeichenblätter vorstellen. Ebenen haben außerdem den Vorteil, dass die Stift-Attribute für alle Objekte auf einer Ebene geändert werden können. Mithilfe des Andockwidgets „Ebenenliste“ kann man Ebenen hinzufügen, entfernen, verbergen und ändern.

 

Durch das Verbergen von Ebenen kann man die visuelle Darstellung beim Zeichnen vereinfachen und sich besser auf den aktuellen Zeichenvorgang konzentrieren.

 

Ebenen können gesperrt werden. Das verhindert versehentliche Änderungen und beschleunigt die Arbeit mit sehr großen, komplexen Zeichnungen.

Konstruktionsebenen

Eine Konstruktionsebene (  ) ist eine spezielle Ebene, auf der man eine Referenzgeometrie erstellen kann, durch die sich andere Zeichenobjekte besser ausrichten lassen. Für eine Konstruktionsebene gilt:

  • Sie enthält Linien von unbegrenzter Länge,
  • Sie erscheint nicht auf gedruckten Zeichnungen, und
  • Sie erscheint beim Import einer Zeichnung auch nicht in ProNest.

Vorlagen

Vorlagen sind Prototypen von Zeichnungen, bei denen man grundlegende Parameter und Einstellungen speichern kann, sodass man nicht jede neu erstellte Zeichnung wieder neu konfigurieren muss. Zu diesen Parametern und Einstellungen gehören die Zeichnungseinstellungen wie das Papierformat, die Haupt-Maßeinheit und das Hauptformat sowie das Format für Bemaßungen. Vorlagen können außerdem Ebenen und Ebenen-Konfigurationen, Linienarten und -stärke, Stiftfarbe und weitere Zeichnungselemente wie Rahmen enthalten.

 

Diese Einstellungen werden bei allen Zeichnungen übernommen, die mit der Vorlage erstellt werden. Eine einmal vorbereitete Vorlage kann an einem beliebigen Speicherort abgelegt werden, für den Sie Lese-/Schreibberechtigung besitzen.

 

LibreCAD unterstützt die Verwendung mehrerer Vorlagen. Ein LibreCAD-Anwender, der viele ähnliche Zeichnungen erstellen möchte, benötigt möglicherweise nur eine oder zwei Vorlagen. Ein Anwender, der mit mehreren verschiedenen Zeichnungsarten arbeiten möchte, braucht wahrscheinlich mehr Vorlagen. So können beispielsweise für jede verfügbare Papiergröße oder jede Papierorientierung eigene Vorlagen eingerichtet werden.

 

Weitere Informationen über die Arbeit mit Vorlagen finden Sie hier.

 

Standardvorlagen

Wenn LibreCAD erstmals aufgerufen wird, erstellt es eine neue Zeichnung mithilfe einer Standardvorlage und einigen Informationen aus ProNest. Jede in LibreCAD neu erstellte Zeichnung nutzt Ihre Vorlage, und diese wird jedes Mal aktualisiert, wenn Sie Einstellungen aus dem Optionsmenü ändern (Anwendungseinstellungen, Aktuelle Zeichnungseinstellungen, Widget-Optionen und Geräteoptionen).

Alle Ebenen Ihrer Zeichenvorlage, die bei den Einstellungen für den Import von Teilen in ProNest nicht zugeordnet sind, werden ignoriert.

Bei der Installation von LibreCAD wird ein Ressourcenverzeichnis erstellt und eine Standardvorlage namens empty.dxf darin abgelegt. Bei 64-Bit-Betriebssystemen finden Sie dieses Verzeichnis unter C:\Program Files (x86)\Hypertherm CAM\LibreCAD\resources\library\templates und bei 32-Bit-Betriebssystemen unter C:\Program Files\Hypertherm CAM\LibreCAD\resources\library\templates.

 

Alternativ zur vorinstallierten LibreCAD-Vorlage kann unter Anwendungseinstellungen > Pfade eine benutzerdefinierte Vorlage konfiguriert werden. Die dort angegebene Vorlage wird anstelle der LibreCAD-Standardvorlage verwendet, wenn die Anwendung gestartet bzw. neue Zeichnungen erstellt werden.
