Attribute eines Objekts ändern

Nach der Platzierung eines Objekts, zum Beispiel einer Linie, eines Bogens, eines Textes oder einer Bemaßung können Sie bestimmte Aspekte des Objekts noch ändern. Wählen Sie dazu im Menü „Ändern“ die Schaltfläche „Attribute“ .

  • Ebene – Änderung der Ebene, auf der das Objekt sich befindet.
  • Stiftfarbe – Änderung der Farbe des Objekts.
  • Stiftbreite – Änderung der Breite des Objekts.
  • Stift-Linienart – Änderung des Linientyps, mit der das Objekt gezeichnet wird.
  • Attribute in Blocktiefe anwenden – Wenn Sie ein Objekt ändern, das Bestandteil eines Blocks ist, können Sie diese Einstellung auswählen, damit die oben genannten Änderungen auf den gesamten Block angewendet werden.

Mit dem Eigenschaften-Werkzeug können Sie weitere Informationen zu einem Objekt ändern, beispielsweise geometrische Koordinaten, Winkel usw.

Attribute eines Objekts ändern

  1. Klicken Sie im Menü Ändern auf Attribute.
  2. Klicken Sie auf das Objekt, dessen Attribute Sie ändern möchten.
  3. Nehmen Sie die Änderungen vor und klicken Sie auf OK.
