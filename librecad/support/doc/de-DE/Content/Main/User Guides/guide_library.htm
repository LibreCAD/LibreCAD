Arbeiten mit dem Bibliotheks-Browser

Für die von Ihnen erstellten Blöcke oder Zeichnungen können zusätzliche Teilebibliotheken hinzugefügt werden. Jede DXF-Zeichnung aus Ihren Bibliotheksordnern wird als Block eingefügt, der dann in einzelne Geometrien aufgebrochen werden kann.

Einen neuen Bibliotheksordner festlegen

  1. Klicken Sie auf Optionen.
  2. Klicken Sie auf Anwendungseinstellungen.
  3. Geben Sie im Feld Teilebibliotheken einen Ordnerpfad an.

Legen Sie Blöcke nicht direkt im Bibliotheksverzeichnis der übergeordneten Teile an. Blöcke müssen in einem Unterordner des übergeordneten Bibliotheksordners abgelegt werden, damit sie im Bibliotheks-Browser erscheinen.

  1. Klicken Sie auf OK.
  2. Klicken Sie im Bibliotheks-Browser auf Aktualisieren.

Der von Ihnen bei den Anwendungseinstellungen angegebene Ordner sollte im Bibliotheks-Browser erscheinen, außerdem die Standard-Installationsordner und -Zeichnungen.

Dateien in Ihren neuen Bibliotheksordner aufnehmen

Sobald Sie einen neuen Ordner in Ihren Bibliotheks-Browser aufgenommen haben (siehe Anleitung im vorigen Abschnitt), können Sie DXF-Zeichnungen in diesen Ordner aufnehmen:

  • Kopieren Sie Dateien aus dem Datei-Explorer auf Ihrem Computer in einen Unterordner des Ordners, den Sie bei den Anwendungseinstellungen angegeben haben.

Über den Bibliotheks-Browser in LibreCAD kann man keine Dateien aufnehmen. Man muss sie vielmehr über den Datei-Explorer des PCs hinzufügen.

Einen Block aus dem Bibliothek-Browser einfügen

  1. Klicken Sie auf den Block, der eingefügt werden soll.
  2. Klicken Sie auf Einfügen.
  3. Geben Sie in der Befehlszeile einen Referenzpunkt an oder wählen Sie einen Punkt auf der Zeichnung aus.

In diesem Modus können Sie beliebig viele Kopien des Blocks einfügen. Mit der Esc-Taste oder einem Rechtsklick verlassen Sie den Block-Einfügemodus.

Einen benutzerdefinierten Bibliotheksordner entfernen

  1. Entfernen Sie in Ihren Anwendungseinstellungen den Ordnerpfad aus den Teilebibliotheken.
  2. Klicken Sie im Bibliotheks-Browser auf Neu erstellen.

Der Ordner sollte nun nicht mehr in Ihrem Bibliotheks-Browser erscheinen.
