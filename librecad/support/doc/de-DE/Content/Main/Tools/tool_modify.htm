Änderungswerkzeuge

Mit diesen Werkzeugen können Sie vorhandene Geometrie in Ihrer Zeichnung ändern, zum Beispiel verschieben, skalieren, überflüssige Geometrie zuschneiden, Text um Objekte herum formen und Eigenschaften bzw. Attribute betrachten und ändern. Die meisten unten beschriebenen Werkzeuge funktionieren sowohl bei mehreren ausgewählten Objekten als auch nur bei einem Objekt.

 

  • Schauen Sie bei der Verwendung eines Werkzeuges auf das Andockwidget für die Befehlszeile und die Symbolleiste mit den Werkzeugoptionen (oberhalb der Zeichnung). Dort finden Sie Rückmeldungen und gegebenenfalls Eingabeaufforderungen für Parameter.
  • Möglicherweise müssen Sie die Enter-Taste drücken, um einen Werkzeugvorgang abzuschließen.
Werkzeug Symbol Befehl Beschreibung
Reihenfolge   Keiner Sortiert Objekte innerhalb einer Ebene. Die ausgewählten Objekte können an den Anfang bzw. ans Ende sowie über bzw. unter ein anderes Objekt verschoben werden.
Verschieben/Kopieren mv, versch Verschiebt ein ausgewähltes Objekt durch Definition eines Referenzpunkts und eines relativen Zielpunkts. Optional kann das Originalobjekt erhalten bleiben (Kopieren), es können mehrere Kopien erstellt bzw. Attribute und Ebene geändert werden.
Drehen ro, drehen Dreht ein ausgewähltes Objekt um einen Drehpunkt. Dabei wird das Objekt von einem Referenzpunkt zu einem Zielpunkt verschoben. Optional kann das Originalobjekt erhalten bleiben, es können mehrere Kopien erstellt bzw. Attribute und Ebene geändert werden.
Skalierung sz, Skalierung Vergrößert oder verkleinert ein ausgewähltes Objekt, ausgehend von einem Referenzpunkt, um einen definierten Faktor für beide Achsen. Optional kann das Originalobjekt erhalten bleiben, es können mehrere Kopien erstellt bzw. Attribute und Ebene geändert werden.
Spiegeln mi, spiegeln Erstellt ein Spiegelbild eines ausgewählten Objekts bezogen auf eine durch zwei Punkte definierte Achse. Optional kann das Originalobjekt erhalten bleiben und es können Attribute und Ebene geändert werden.
Verschieben und drehen Keiner Verschiebt ein ausgewähltes Objekt durch Definition eines Referenzpunkts und eines relativen Zielpunkts und dreht es dabei um einen bestimmten Winkel. Optional kann das Originalobjekt erhalten bleiben, es können mehrere Kopien erstellt bzw. Attribute und Ebene geändert werden.
Zwei drehen Keiner Dreht ein ausgewähltes Objekt um einen absoluten Drehpunkt. Dabei wird das Objekt um einen relativen Referenzpunkt zu einem Zielpunkt gedreht. Optional kann das Originalobjekt erhalten bleiben, es können mehrere Kopien erstellt bzw. Attribute und Ebene geändert werden.
Richtung umkehren Umkehren Vertauscht Anfangs- und Endpunkte eines oder mehrerer ausgewählter Objekte.
Zuschneiden tm, trim Schneidet ein Linienobjekt an einem schneidenden Objekt ab.
Zwei zuschneiden t2, tm2 Schneidet zwei sich überschneidende Objekte am Überschneidungspunkt ab.
Übermaß zuschneiden tx, tmx, trimex Schneidet ein Objekt ab, das sich mit einem anderen Objekt überschneidet. Durch Bewegung des Mauszeigers wird eine Vorschau für das abgeschnittene Objekt angezeigt.
Verlängern le, verlängern

Verlängert ein Linienobjekt. Wenn Sie auf das Kontrollkästchen klicken, wird das Objekt um die im Längenfeld angegebene Länge gekürzt anstatt verlängert. Zum Verkürzen können Sie alternativ auch einen negativen Wert im Längenfeld angeben.

 

Werkzeugoptionen:
Versatz o, pa, Parallele Kopiert ein ausgewähltes Objekt in einem definierten Abstand in der angegebenen Richtung.
Fase (Abschrägung) ch, Fase

Erstellt eine schräge Kante zwischen zwei sich überschneidenden Liniensegmenten, definiert durch einen Versatz auf jedem Segment.

Anmerkung: Dieser Befehl unterscheidet sich von der Fase in ProNest und verhält sich eher wie eine Abschrägung! Verwenden Sie ihn nicht, wenn Sie das Abfasen in den CNC-Code für ein Teil aufnehmen möchten.

 

Werkzeugoptionen:
Ausrundung fi, Ausrundung

Erstellt einen Eckenradius zwischen zwei sich überschneidenden Liniensegmenten mit einem definierten Radius.

 

Werkzeugoptionen:
Teilen di, cut Teilt bzw. zerlegt alle Linien am ausgewählten Schnittpunkt.
Strecken ss, strecken Verschiebt einen ausgewählten Bereich einer Zeichnung. Dafür werden zuerst zwei Enden der Zeichnung oder des Objekts ausgewählt und dann ein Referenzpunkt und ein relativer Zielpunkt definiert.
Eigenschaften mp, prop, Eigenschaften Zur Änderung der Attribute eines oder mehrerer ausgewählter Objekte, zum Beispiel Ebene, Stiftfarbe, Stiftbreite und Stift-Linienart. Außerdem kann die Geometrie der ausgewählten Objekte geändert werden.
Attribute ma, attr, Atträndern Zur Änderung der gemeinsamen Attribute eines oder mehrerer ausgewählter Objekte, zum Beispiel Ebene, Stiftfarbe, Stiftbreite und Stift-Linienart.
Text in Buchstaben aufbrechen Keiner Teilt eine Zeichenkette in die Objekte für die einzelnen Zeichen auf.
Text formen st, stxt, Formtxt

Formt Text um ein einzelnes Bogen- oder Linienobjekt herum.

 

Geben Sie einen Versatzwert an. Hiermit wird die Textposition in Bezug auf das Objekt, um das er herum geformt wird, festgelegt. Bei einem negativen Versatzwert wird der Text (ausgehend von der Text-Grundlinie) unter dem Formobjekt platziert, bei einem positiven Versatzwert über dem Formobjekt.

 

Mit dem Kontrollkästchen können Sie die Textrichtung vertauschen, was insbesondere dann sinnvoll sein kann, wenn Text um Kreise oder Bögen herum geformt wird.

 

Werkzeugoptionen:
Text von Geometrie lösen ut, utxt, unlinktxt Löst die Verknüpfung geformten Textes von dem formenden Objekt. Wenn der Text um einen Bogen herum geformt ist, wird der Text automatisch in einzelne Buchstaben aufgebrochen; wenn der Text entlang einer Linie geformt ist, bleibt der Text als Zeichenkette erhalten und wird nicht in einzelne Buchstaben aufgebrochen.
Aufbrechen xp, aufbrechen Teilt einen oder mehrere ausgewählte Blöcke oder zusammengesetzte Objekte in einzelne Objekte auf.
Ausgewählte löschen er, löschen

Löscht ein oder mehrere ausgewählte Objekte.
