Linienwerkzeuge

Wenn Sie beim Einfügen von Linien feststellen, dass Sie Linien unbegrenzter Länge sehen, liegt das wahrscheinlich daran, dass es sich bei der Ebene, auf der Sie zeichnen, um eine Konstruktionsebene handelt.

Werkzeug Symbol Befehl Beschreibung

2 Punkte

 l, li, line

Zeichnet eine Linie zwischen zwei angegebenen Punkten.

 

Werkzeugoptionen:
Winkel Keiner

Zeichnet eine Linie von einem angegebenen Punkt, der den Anfang, die Mitte oder das Ende der Linie definiert, und mit einer angegebenen Länge und einem angegebenen Winkel.

 

Werkzeugoptionen:
Horizontal hor, horizontal

Zeichnet eine horizontale Linie von einem angegebenen Punkt, der den Anfang, die Mitte oder das Ende der Linie definiert, und mit einer angegebenen Länge.

 

Werkzeugoptionen:
Vertikal ver, vertical

Zeichnet eine vertikale Linie von einem angegebenen Punkt, der den Anfang, die Mitte oder das Ende der Linie definiert, und mit einer angegebenen Länge.

 

Werkzeugoptionen:
Rechteck rec, rect, rectangle

Zeichnet ein Rechteck. Dazu werden die Punkte zweier diagonal gegenüberliegender Ecken angegeben.
Parallele durch Punkt pp, ptp

Zeichnet eine bestimmte Anzahl von Linien durch einen angegebenen Punkt parallel zu einer ausgewählten vorhandenen Linie. Wenn Sie die Maus näher an den Punkt heranschieben oder weiter davon wegschieben, wird der Abstand zwischen den Linien verringert bzw. erhöht.

 

Werkzeugoptionen:
Parallele o, pa, offset, parallel

Zeichnet eine bestimmte Anzahl von Linien parallel zu einer ausgewählten vorhandenen Linie mit einem bestimmten Abstand zwischen den Linien.

 

Werkzeugoptionen:
Winkelhalbierende bi, bisect

Zeichnet eine bestimmte Anzahl von Linien als Winkelhalbierende zwischen zwei vorhandenen, nicht parallel verlaufenden Linien (zum Beispiel in einem Winkel zueinander, mit oder ohne gemeinsamen Punkt).

 

Werkzeugoptionen:
Tangente (P,K) tanpc, tangentpc Zeichnet eine Linie von einem angegebenen Punkt tangential zu einem vorhandenen Kreis.
Tangente (K,K) Keiner Zeichnet eine Linie tangential zu zwei vorhandenen Kreisen.
Tangente rechtwinklig Keiner Zeichnet eine Linie tangential zu einem vorhandenen Kreis und senkrecht zu einer vorhandenen Linie.
Orthogonal ortho, perp

Zeichnet eine Linie einer bestimmten Länge senkrecht zu einer vorhandenen Linie mit dem Mittelpunkt auf einem angegebenen Punkt.

 

Werkzeugoptionen:
Relativer Winkel Keiner

Zeichnet eine Linie bestimmter Länge und in einem bestimmten Winkel zu einer vorhandenen Linie, mit dem Mittelpunkt der Linie auf einem angegebenen Punkt.

 

Werkzeugoptionen:
Polygon (Mitte,Ecke) pl, polyline

Zeichnet ein Polygon mit einer bestimmten Anzahl von Seiten durch Angabe des Mittelpunkts und eines Eckpunkts.

 

Werkzeugoptionen:
Polygon (Zen,Tan) Keiner

Zeichnet ein Polygon mit einer bestimmten Anzahl von Seiten durch Angabe des Mittelpunkts und des Mittelpunkts einer Seite.

 

Werkzeugoptionen:
Polygon (2 Ecken) poly2, polygon2v

Zeichnet ein Polygon mit einer bestimmten Anzahl von Seiten durch Angabe der zwei Punkte einer Seite.
