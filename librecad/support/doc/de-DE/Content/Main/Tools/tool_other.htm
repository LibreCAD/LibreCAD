Zusätzliche Werkzeuge: Text, Schraffur, Punkt

Werkzeug Symbol Befehl Beschreibung
Text mehrzeilig (MText) mtxt, mtext

Fügt mehrzeiligen Text an einem festgelegten Basispunkt in die Zeichnung ein. Optional können Schriftart, Texthöhe, Winkel, Breitenfaktor, Ausrichtung, Sonderzeichen und Zeichensatz definiert werden.

 

Weitere Informationen über die Arbeit mit mehrzeiligem Text (MText) finden Sie hier.

 
Text txt, text

Fügt einzeiligen Text an einem festgelegten Basispunkt in die Zeichnung ein. Optional können Schriftart, Texthöhe, Ausrichtung, Winkel, Sonderzeichen und Zeichensatz definiert werden.

 

Weitere Informationen über die Arbeit mit Text finden Sie hier.

 
Schraffur ha, hatch

Füllt eine Kontur (Polygon, Kreis, Polylinie usw.) mit einem definierten Muster oder einer festen Füllung. Wenn das Objekt ein offenes Profil hat, wird das Schraffurmuster möglicherweise fehlerhaft gezeichnet. Optional können Skalierung und Winkel definiert werden.

 

Schraffurmuster-Anhang.

 
Punkt po, point Zeichnet einen Punkt. Dazu wählen Sie eine Position in der Zeichnung aus oder geben in der Befehlszeile die Koordinaten ein.
