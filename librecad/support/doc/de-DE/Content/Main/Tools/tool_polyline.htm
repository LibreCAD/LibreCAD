Polylinien-Werkzeuge

Werkzeug Symbol Befehl Beschreibung
Polylinie pl, Polylinie

Zeichnet eine offene oder geschlossene, durchgehende Linie, die aus ein oder mehreren geraden Linien oder Bogenabschnitten besteht, die durch Endpunkte bzw. Radius und Winkel (bei Bögen) definiert sind.

 

Werkzeugoptionen:
Knoten hinzufügen Keiner Fügt einen Knoten zu einer vorhandenen Polylinie hinzu. Tipp: Verwenden Sie „Am Objekt fassen“, wenn Sie einen neuen Knoten an einem Segment platzieren.
Knoten anfügen Keiner Fügt ein oder mehrere Segmente zu einer Polylinie hinzu. Dazu wird die Polylinie ausgewählt und der Endpunkt eines neuen Knotens hinzugefügt.
Knoten löschen Keiner Einen Knoten einer vorhandenen Polylinie löschen.
Löschen zwischen zwei Knoten Keiner Löscht einen oder mehrere Knoten zwischen den ausgewählten Knoten einer vorhandenen Polylinie.
Segmente zuschneiden Keiner Erweitert zwei separate, nicht parallel verlaufende Segmente einer vorhandenen Polylinie so, dass sie sich bei einem neuen Knoten überschneiden.
Abstandsgleiche Polylinien erstellen Keiner

Zeichnet eine bestimmte Anzahl von Polylinien parallel zu einer ausgewählten vorhandenen Polylinie mit einem bestimmten Abstand zwischen den Linien.

 

Werkzeugoptionen:
Polylinien aus vorhandenen Segmenten erstellen Keiner Erstellt eine Polylinie aus zwei oder mehr vorhandenen, separaten Linien- oder Bogensegmenten, die eine durchgehende Linie bilden.
