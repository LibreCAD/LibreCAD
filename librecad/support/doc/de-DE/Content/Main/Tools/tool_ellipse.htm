Ellipsenwerkzeuge

Werkzeug Symbol Befehl Beschreibung
Ellipse (Achse) Keiner Zeichnet eine Ellipse. Hierzu werden ein Mittelpunkt, ein Punkt auf dem Umfang der Hauptachse und ein Punkt auf dem Umfang der Nebenachse angegeben.
Ellipsen-Brennpunkt Keiner Zeichnet eine Ellipse. Hierzu werden zwei Brennpunkte und ein Punkt auf dem Umfang angegeben.
Ellipse 4 Punkte Keiner Zeichnet eine Ellipse. Hierzu werden vier Punkte auf dem Umfang angegeben.
Ellipse Mittelpunkt und 3 Punkte Keiner Zeichnet eine Ellipse. Hierzu werden ein Mittelpunkt und drei Punkte auf dem Umfang angegeben.
Eingeschriebene Ellipse ei, ie, eingeschrEllipse Zeichnet eine Ellipse, die durch vier vorhandene, nicht parallel verlaufende Liniensegmente bestimmt ist.
