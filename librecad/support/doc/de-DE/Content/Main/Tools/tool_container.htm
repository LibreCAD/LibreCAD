Über Zeichen- und Änderungs-Werkzeuge

In LibreCAD können Sie mithilfe von Werkzeugen Geometrie, Bemaßungen und Text in Ihre Zeichnung aufnehmen und alle vorhandenen Objekte in der Zeichnung verändern. Unten finden Sie eine Übersicht für die jeweiligen Werkzeuge.

 

Geometrie

  • Linien
  • Polylinien
  • Kreise
  • Kurven
  • Ellipsen

Zusätzliche Werkzeuge

  • Bemaßungen
  • Text, Punkte, Schraffuren
  • Auswählen
  • Ändern

  • Schauen Sie bei der Verwendung eines Werkzeuges auf das Andockwidget für die Befehlszeile und die Symbolleiste mit den Werkzeugoptionen (oberhalb der Zeichnung). Dort finden Sie Rückmeldungen und gegebenenfalls Eingabeaufforderungen für Parameter.
  • Möglicherweise müssen Sie die Enter-Taste drücken, um einen Werkzeugvorgang abzuschließen.
