Utilisation de la ligne de commande

La ligne de commande offre aux utilisateurs une solution de rechange à l’utilisation de la souris pour la sélection des outils et du dessin. L’utilisation du clavier pour sélectionner les outils et saisir les coordonnées peut rendre plus rapide et plus précise la création de dessins.

 

La ligne de commande offre d’autres fonctionnalités utiles que n’offre pas l’utilisation de la souris :

  • Entrée multicommande
  • Calculatrice

La ligne de commande est affichée dans son propre gadget logiciel d’ancrage et est composée de trois éléments.

  1. Une fenêtre d’invite de commande
  2. Une fenêtre de sortie de commande
  3. Un bouton () affichant un menu déroulant qui comprend :
    • Mode de code de touche on/off
    • Charger un fichier de commande
    • Coller plusieurs commandes

 

Consultez l’annexe pour obtenir une liste complète des commandes de la ligne de commande.

Saisie des fonctions dans la ligne de commande

Les commandes pouvant être utilisées sont affichées sur les pages de référence pour chaque catégorie d’outils. La ligne de commande est activée de différentes façons :

  • Commencez à saisir une commande (par exemple, rect), puis sélectionnez Entrée.
  • Appuyez sur la barre d’espace, saisissez une commande et sélectionnez Entrée.
  • Appuyez sur Ctrl+M, sélectionnez une commande, puis sélectionnez Entrée.
  • Appuyez sur Maj, sélectionnez une commande, puis sélectionnez Entrée.
  • Avec le mode code de touche activé, saisissez une commande de deux lettres (par exemple ci). Il n’est pas nécessaire de sélectionner Entrée.
  • La touche Tab peut être utilisée dans la ligne de commande lors de la saisie des commandes. Entrez une commande partielle, comme cir, puis appuyez sur la touche Tab. Si une seule commande est possible avec cette commande partielle, la commande complète sera affichée dans le champ d’entrée de texte. Si plus d’une commande est possible, une liste des commandes possibles sera affichée dans la fenêtre de commande.

Selon la commande que vous utilisez, une entrée supplémentaire peut être nécessaire dans la ligne de commande ou la barre d’outils.

  • Vous pouvez utiliser la barre d’espace à la place de la touche Entrée lorsque la fonction Options > Paramètres de l’application > Par défaut > Évaluer les commandes lorsque la barre d’espace est actionnée est activée (elle l’est par défaut).
  • En plus des coordonnées séparées par des virgules, les coordonnées peuvent également être saisies à l’aide du clavier numérique en utilisant le format X..Y. Par exemple, saisir 10..20 équivaut à @10,20. Il peut être plus rapide d’utiliser les deux points que de taper la virgule.
  • Le mode code de touche permet l’utilisation de commandes à deux lettres et élimine le besoin d’appuyer sur Entrée après la saisie de la commande.
  • Saisir c ou u, puis appuyer sur Entrée peut être utilisé au lieu de saisir close ou undo.

Utilisation de la ligne de commande Calculatrice

  1. Entrez Cal dans la ligne de commande pour activer le mode Calculatrice.
  2. Entrez des équations au besoin. Le mode Calculatrice peut également être utilisé pour extraire des valeurs, comme pi. Les résultats seront affichés dans la fenêtre de commande.
  3. Quittez le mode Calculatrice en saisissant à nouveau Cal dans la ligne de commande.

Consultez la liste des Opérateurs et fonctions de la calculatrice.

  • La constante pi est définie dans LibreCAD comme 3,14159265359.
  • Les fonctions trigonométriques utilisent des radians (radians = degrés*pi/180).
