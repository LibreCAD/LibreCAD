Entrée multicommande avec fichiers de variables (Avancé)

Les entrées de commande peuvent se combiner et être saisies sur une seule ligne, en séparant les commandes et les autres entrées par un point-virgule. Le fait de saisir li;0,0;10..0;0..10;-10..0;c;k sur la ligne de commande dessinera un carré de 10 x 10. Une liste de commandes, séparées par une nouvelle ligne (retour), peut être copiée et collée en utilisant la fonction « Coller plusieurs commandes » depuis le bouton de ligne de commande (angle inférieur droit de l’ancrage Ligne de commande).

 

L’entrée de commande peut également être chargée à partir des fichiers de texte. Entrez les commandes et les autres entrées dans un fichier de texte, en les séparant par un saut de ligne. Par exemple, créez un fichier de texte et entrez les commandes suivantes :

li

0,0

@10,0

@0,10

@-10,0

c

k

Les commandes ci-dessus dessineront un carré de 10 x 10.

Utilisation de l’entrée multicommande

  1. Créez un fichier de texte contenant l’entrée de commande.
  2. Dans LibreCAD, cliquez sur Charger le fichier de commande depuis le menu déroulant en sélectionnant le bouton Ligne de commande ( ).
  3. Localisez et cliquez sur le fichier, puis cliquez sur Ouvrir.

La géométrie que vous avez indiquée dans le fichier de texte apparaîtra dans le dessin actif.

 

Variables dans l’entrée multicommande

L’entrée multicommande peut être attribuée à une variable, et des variables peuvent également contenir d’autres variables qui sont lues de manière récursive :

a=ci;0,0;10

b=ci;10,0;10

c=\a;\b;kill

\c

Entrez chaque ligne de texte ci-dessus dans la ligne de commande. Lorsque \c est saisi, deux cercles se chevauchant avec un rayon de 10 sont dessinés. Le caractère \ est un caractère d’échappement permettant à la ligne de commande d’interpréter le nom de la variable en tant qu’action. Dans l’exemple ci-dessus, \c devient ci;0,0;10;ci;10,0;10;kill.

Utilisation des variables dans l’entrée multicommande

  1. Créez un fichier de texte d’entrée de variables multicommande.
  2. Chargez le fichier en sélectionnant Options.
  3. Cliquez sur Paramètres de l’application.
  4. Cliquez sur Trajectoires.
  5. Cliquez sur Parcourir ( ) et localisez le fichier de variables.
  6. Cliquez sur OK.
  7. Cliquez de nouveau sur OK pour enregistrer les paramètres de l’application.
  8. Redémarrez LibreCAD.
  9. Entrez \c dans la ligne de commande.

Le fichier sera chargé.
