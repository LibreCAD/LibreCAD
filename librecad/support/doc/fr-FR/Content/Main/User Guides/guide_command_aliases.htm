Alias de lignes de commande

Dans LibreCAD, certaines commandes ont plusieurs formes. La forme longue est la commande d’origine, non traduite, et la forme courte est l’alias. Par exemple, la commande d’origine longue pour Forme de texte est « shapetext », mais la forme courte, l’alias, est « ft ». Les alias de toutes les commandes peuvent être définis dans le fichier de configuration librecad.alias.

Format : <alias>[TAB]<command-untranslated>

 

Exemple : l       line

Le fichier Alias peut être modifié avec n’importe quel éditeur de texte, comme Notepad. Le fichier se trouve à l’emplacement suivant : C:\Users\{UserName}\AppData\Local\LibreCAD\librecad.alias.

  • Ne modifiez que l’alias, non pas la forme longue non traduite.
  • Vous devrez redémarrer LibreCAD pour que les modifications à l’alias soient effectives.
