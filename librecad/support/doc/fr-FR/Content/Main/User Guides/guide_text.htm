Utilisation du texte

Du texte peut être intégré dans des dessins et des symboles artistiques et peut être utilisé pour fournir des informations et des notes, notamment dans les cartouches. Créez du texte sur une ou plusieurs lignes en utilisant les outils Texte ou MTexte.

  • Pour obtenir des résultats optimaux lorsque vous utilisez du texte, il est recommandé d’éclater le texte en géométrie (à l’aide de la commande explode) avant d’envoyer un dessin vers ProNest. Utilisez un alignement À gauche de la ligne de base ( ) pour Texte, ou En bas à gauche pour MTexte afin d’éviter tout problème de géométrie lors de l’envoi d’un dessin vers ProNest.
  • Notez que le texte sera toujours converti en géométrie dans ProNest sauf s’il est sur un calque de traçage, mais les résultats seront meilleurs si le texte est éclaté dans LibreCAD avant d’être exporté vers ProNest.
  • Veillez à ce que Lissage soit activé dans ProNest pour convertir les segments de ligne hachés en arcs et en profils plus lisses avant d’importer une pièce depuis LibreCAD (ProNest > Paramètres > Importation de pièces > activez Lisser les entités et essayez Tolérances de lissage entre 0,001 et 0,005 pour commencer).

Création de texte sur une ligne

  1. Cliquez sur Texte ( ) dans le menu Outils.
  2. Dans le champ Texte, entrez le texte que vous souhaitez voir inscrit sur le dessin.
  3. Réglez les paramètres de police, l’alignement et les symboles et convertissez les polices Microsoft, s’il y a lieu.
  4. Cliquez sur OK.
  5. Cliquez sur le dessin pour placer le texte ou entrez les coordonnées de position dans la ligne de commande (l’alignement du texte est utilisé comme point d’insertion lors de l’indication de coordonnées).

Création de texte sur plusieurs lignes

  1. Cliquez sur MTexte ( ) dans le menu Outils.
  2. Dans le champ Texte, entrez le texte que vous souhaitez voir inscrit sur le dessin. Utilisez le menu déroulant Insérer un symbole pour insérer des caractères spéciaux dans le texte, comme les symboles de degré et de droits d’auteur.
  3. Convertissez les polices Microsoft et définissez les valeurs d’espacement, si nécessaire.
  4. Définissez l’alignement et l’angle d’insertion (l’angle peut être laissé vide pour insérer un texte à 180°).
  5. Cliquez sur OK.
  6. Cliquez sur le dessin pour placer le texte ou entrez les coordonnées de position dans la ligne de commande (l’alignement du texte est utilisé comme point d’insertion lors de l’indication de coordonnées).

Modification du texte

  1. Cliquez sur Propriétés ( ) dans le menu des outils Modifier. La boîte de dialogue de texte s’affichera de nouveau, vous permettant de modifier le texte ainsi que les informations de calque et de couleur.
  2. Cliquez sur le texte à modifier.
  3. Modifiez les propriétés pertinentes.
  4. Cliquez sur OK.

Former le texte autour d’une entité

Avec la fonctionnalité Forme de texte exclusive d’Hypertherm dans LibreCAD, vous pouvez former le texte autour d’une ligne simple, d’un arc ou d’un cercle.

  1. Dessinez une forme.
  2. Placez le texte en utilisant les outils Texte ou MTexte. Dans la boîte de dialogue Texte ou MTexte, assurez-vous de spécifier un alignement. Si vous hésitez quant à l’alignement à utiliser, essayez l’option en bas à gauche.
  3. Entrez dans le mode Forme de texte en cliquant sur Forme de texte () à partir des outils Modifier, ou en saisissant st dans la ligne de commande.
  4. Cliquez sur le texte auquel vous souhaitez donner une forme.
  5. Déplacez votre curseur autour d’une forme. Modifiez la valeur de Décalage et la direction du texte en sélectionnant la case à cocher, si nécessaire. Lorsque le texte se trouve à l’emplacement désiré, cliquez pour placer le texte.

Pour atteindre un réglage fin du texte auquel vous donnez une forme, il est recommandé d’éclater les chaînes de texte en lettres individuelles à l’aide de l’outil de modification Éclater le texte en lettres ().

Alignement du texte formé dans le périmètre de l’entité

  1. Dessinez une forme.
  2. Placez le texte en utilisant les outils Texte ou MTexte. Dans la boîte de dialogue Texte ou MTexte, assurez-vous de spécifier un alignement central.
  3. Entrez dans le mode Forme de texte en sélectionnant Forme de texte ( ) à partir des outils Modifier, ou en saisissant st dans la ligne de commande.
  4. Ajustez le décalage du mode Forme de texte dans la barre d’outils.
    •  Pour un texte sur une ligne, le décalage doit être une valeur négative équivalant à la moitié de la hauteur du texte. Par exemple, pour une hauteur de texte de 1, la valeur de décalage doit être de –0,5.
    • Le décalage peut être ajusté et mis à jour en entrant une valeur et en appuyant sur la touche Entrée.
  5. Cliquez sur le texte auquel vous souhaitez donner une forme.
  6. Déplacez votre curseur autour d’une forme. Lorsque le texte se trouve à l’emplacement désiré, cliquez pour placer le texte.

 

Utilisez l’outil de modification Ajuster l’excédent ( ) pour éliminer rapidement toute géométrie non souhaitée en un clic.
