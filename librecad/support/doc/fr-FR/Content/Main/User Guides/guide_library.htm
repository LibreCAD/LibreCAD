Utilisation du Menu des bibliothèques

Des bibliothèques de pièces supplémentaires peuvent être ajoutées pour les blocs ou les dessins que vous créez. Tout dessin DXF indiqué dans vos dossiers Bibliothèque est inséré en tant que bloc, qui peut être éclaté en formes géométriques individuelles.

Spécification d’un nouveau dossier de bibliothèque

  1. Cliquez sur Options.
  2. Cliquez sur Paramètres de l’application.
  3. Indiquez un chemin d’accès de dossier dans le champ Bibliothèques de pièces.

Ne placez pas les blocs directement dans le répertoire parent de bibliothèque des pièces. Les blocs doivent être placés dans un sous-dossier du dossier parent des bibliothèques pour s’afficher dans le menu des bibliothèques.

  1. Cliquez sur OK.
  2. Cliquez sur Actualiser dans le menu des bibliothèques.

Le dossier que vous avez indiqué dans les Paramètres de l’application doit s’afficher dans le menu des bibliothèques, avec les dossiers et les dessins standards installés.

Ajout de fichiers dans votre nouveau dossier de bibliothèque

Lorsque vous ajoutez un nouveau dossier dans votre menu des bibliothèques (voir les étapes dans la section précédente), vous pouvez ajouter des dessins DXF à ce dossier :

  • Copiez et collez des fichiers à partir de l’Explorateur de fichiers sur votre ordinateur dans un sous-dossier du dossier que vous avez indiqué dans les Paramètres de l’application.

Les fichiers ne peuvent être ajoutés au menu des bibliothèques depuis LibreCAD; ils doivent être ajoutés à l’aide de l’Explorateur de fichiers sur votre ordinateur.

Insertion de blocs depuis le menu des bibliothèques

  1. Cliquez sur le bloc que vous souhaitez insérer.
  2. Cliquez sur Insérer.
  3. Indiquez un point de référence dans la ligne de commande ou sélectionnez un point sur le dessin.

Dans ce mode, vous pouvez insérer autant de copies du bloc que nécessaire. Appuyez sur la touche Échap, ou effectuez un clic droit pour quitter le mode d’insertion de blocs.

Suppression d’un dossier de bibliothèque spécifié par l’utilisateur

  1. Assurez-vous que le chemin d’accès du dossier est supprimé du champ Bibliothèques de pièces dans les Paramètres de l’application.
  2. Cliquez sur Restaurer dans le menu des bibliothèques.

Le dossier ne devrait plus s’afficher dans le menu des bibliothèques.
