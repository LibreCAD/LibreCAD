Utilisation des plugiciels

Dans LibreCAD, les plugiciels sont des commandes spéciales qui peuvent aider à créer, à modifier ou à obtenir des informations à propos des entités dans un dessin. Les plugiciels peuvent avoir plusieurs raisons d’être à part la modification des entités, comme la prise en charge de certains types de fichiers, notamment le plugiciel ESRI Shapefile.

 

Nom Description
Aligner Alignez les entités sélectionnées par rapport à une référence en définissant les positions finales des deux points initiaux.
Lecture des points ASCII

Lire les points depuis un fichier texte.

Chaque ligne du fichier est un point défini par un identifiant, une coordonnée X, une coordonnée Y, une coordonnée Z et un code optionnel. Chaque champ peut être séparé par une virgule, une tabulation ou un espace. Le séparateur de décimales est un point (.). Les points peuvent être connectés avec une ligne, un identifiant ou une coordonnée et les champs de code peuvent être tracés comme texte.
Diviser

Divisez une ligne ou un cercle en n sections.

Une coche peut être placée à la limite de chaque section pour marquer chaque limite. La taille de cette coche peut être définie comme pourcentage de la longueur du segment. La ligne ou le cercle peut être cassé à la limite de chaque section utilisant l’outil Diviser.
Plugiciel d’engrenage Dessinez un engrenage en sélectionnant le centre de l’engrenage et en définissant les paramètres comme le nombre de dents, le module, etc.
ESRI Shapefile

Importez le fichier de forme de données de vecteur géospatial GIS (cartes).

Remarque : Le processus d’importation verrouillera LibreCAD jusqu’à ce qu’il soit terminé et les fichiers volumineux peuvent prendre beaucoup de temps.
Liste des entités Listez les entités sélectionnées ainsi que leurs propriétés comme l’identifiant, le calque, la couleur, le type et l’épaisseur de la ligne ainsi que les coordonnées.
Lire le fichier PIC Importez les diagrammes de langue des graphiques PIC.
Plugiciel de tracé

Tracez une fonction mathématique ou une fonction paramétrique en utilisant le système de coordonnées de dessin.

La formule, la valeur de départ, la valeur de fin et la valeur d’étape sont nécessaires. Le tracé peut être composé de lignes, d’une polyligne ou d’une courbe spline.
Mêmes propriétés

Appliquez les propriétés d’une entité de référence aux entités sélectionnées.

Les propriétés modifiées sont le calque, la couleur, le type de ligne et l’épaisseur de la ligne.
Plugiciel représentatif Dessinez une ligne en spécifiant les coordonnées X et Y des points de fin.
