Utilisation des modèles

Les modèles sont des prototypes de dessins qui permettent d’enregistrer les paramètres de base afin de ne pas avoir à configurer chaque nouveau dessin. Les paramètres incluent les paramètres définis dans les Paramètres du dessin, comme le format du papier, les unités principales de mesure, le format et le format des cotations. Les modèles peuvent également inclure les calques et leur configuration, le type et l’épaisseur des lignes, la couleur du stylet ainsi que d’autres éléments du dessin, comme la bordure. Ces paramètres découlent des dessins créés à partir du modèle.

 

Vous pouvez utiliser un dessin existant en tant que modèle ou vous pouvez utiliser le dessin vide par défaut en tant que modèle.

Ajout d’un cartouche au modèle de dessin vide par défaut

Vous pouvez ajouter un cartouche ou d’autres entités au dessin vide par défaut qui s’affiche lorsque LibreCAD est lancé ou qu’un nouveau dessin est créé. La configuration est utile si certaines entités doivent se retrouver dans tous vos dessins de CAO.

  1. Ouvrez le fichier de modèle DXF par défaut dans LibreCAD : C:\Users\[Windows Username]\AppData\Local\Hypertherm CAM\ProNest 2019\ProNestTemplate.DXF
  2. Apportez les modifications au dessin modèle, s’il y a lieu.
  3. Cliquez sur Enregistrer ().

Le fichier modèle est maintenant mis à jour. Pour vérifier vos modifications, créez un nouveau dessin.

Si vous ne pouvez pas localiser le fichier modèle, le modèle de dessin par défaut est listé dans la ligne de commande lorsqu’un nouveau dessin est créé.

Utiliser un dessin existant en tant que modèle

  1. Cliquez sur Fichier.
  2. Cliquez sur Nouveau depuis modèle.
  3. Sélectionnez le dessin que vous souhaitez utiliser comme modèle.
  4. Cliquez sur Ouvrir.

Modifiez le dessin si nécessaire. Veuillez noter que le dessin modèle ne sera pas modifié et que vous n’aurez pas à enregistrer le nouveau dessin dans un fichier séparé.
