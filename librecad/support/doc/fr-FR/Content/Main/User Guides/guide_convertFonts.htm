Conversion des polices Microsoft et AutoCAD en polices LibreCAD

Polices

Plusieurs options de polices sont offertes dans LibreCAD, dont les polices ProNest. Pour utiliser les polices Microsoft® ou les polices AutoCAD®, vous devez suivre les étapes ci-dessous pour d’abord convertir les polices en un format lisible pour LibreCAD.

Si vous ouvrez des dessins AutoCAD .dwg qui utilisent des polices AutoCAD, vous devrez convertir la police afin de garantir que le texte soit transféré correctement dans LibreCAD. Le processus de conversion de la police convertira toutes les polices AutoCAD et ne devra être réalisé qu’une seule fois.

Conversion des polices

Polices ProNest

LibreCAD comprend les polices ProNest dans la liste des polices. Il s’agit des mêmes polices qui sont indiquées à la page Paramètres des polices de CAO de ProNest, comme la police « no lift ».

  • Pour utiliser les polices ProNest, il suffit de trouver la police dans le menu déroulant Polices dans la boîte de dialogue Texte et MTexte.
Polices Microsoft

Utilisez les polices par défaut de Microsoft et personnalisez les lettres, les mots et les espacements de lignes de ces polices en utilisant la boîte de dialogue Convertir la police de la boîte de dialogue Texte et MTexte.

Conversion des polices Microsoft

  1. Cliquez sur Texte ou MTexte.
  2. Cliquez sur Convertir texte () dans le coin supérieur droit.
  3. Sélectionnez une police que vous souhaitez convertir dans le menu déroulant.
  4. Réglez Lettre, Mot et Espacement de ligne, si nécessaire.
    1. Si vous avez réglé l’espacement, vous pouvez enregistrer les modifications dans la police par défaut en cours ou vous pouvez ajuster le nom du fichier dans le champ Enregistrer sous, ce qui vous permettra ainsi d’enregistrer une copie de la police modifiée sans remplacer la police convertie par défaut.
  5. Cliquez sur OK.

Assurez-vous de sélectionner la police convertie dans le menu déroulant Police dans la boîte de dialogue Texte ou MTexte afin d’utiliser la police lorsqu’elle sera convertie.

Polices AutoCAD

LibreCAD pour ProNest prend en charge les polices AutoCAD (.shx). Si vos polices AutoCAD sont stockées dans le dossier de police par défaut pour AutoCAD <C:\Program Files\AutoDesk\AutoCAD 2019\Fonts>, aucune action n’est nécessaire (veuillez noter que ce chemin de fichier est celui d’AutoCAD 2019 et qu’il peut différer selon votre version d’AutoCAD).

Conversion manuelle des polices AutoCAD

Si des polices AutoCAD (.shx) sont stockées dans un autre endroit, vous devrez réaliser une fois les actions suivantes :

  1. Ouvrir un explorateur de fichier sur votre ordinateur.
  2. Naviguer jusqu’au dossier C:\Program Files (x86)\Hypertherm CAM\LibreCAD et localiser le fichier dumpshp.bat.
  3. Effectuer un clic droit sur dumpshp.bat et cliquer sur Copier.
  4. Naviguer jusqu’au dossier contenant vos polices AutoCAD.
  5. Effectuer un clic droit et cliquer sur Coller pour placer une copie de dumpshp.bat dans le dossier contenant les polices AutoCAD.
  6. Double-cliquer sur dumpshp.bat pour exécuter le fichier. Une fenêtre de commande s’affichera et se fermera automatiquement lorsque l’exécution sera terminée.

Désormais, toute police AutoCAD utilisée dans vos dessins sera automatiquement convertie en un format lisible pour LibreCAD. Vous pourrez utiliser n’importe quelle police AutoCAD dans un nouveau dessin LibreCAD.
