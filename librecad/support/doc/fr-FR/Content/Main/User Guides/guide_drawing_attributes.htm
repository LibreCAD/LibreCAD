Modification des attributs d’une entité

Après avoir placé une entité, comme des lignes, des arcs et du texte, ou des cotations, vous pouvez modifier certains aspects de l’entité en utilisant le bouton Attributs ( ) dans le menu Modifier.

  • Calque – Modifier le calque sur lequel se trouve l’entité.
  • Couleur du stylet – Changer la couleur de l’entité.
  • Largeur du stylet – Changer la largeur de l’entité.
  • Type de ligne du stylet – Changer l’aspect uni de la ligne qui dessine l’entité.
  • Appliquer les attributs à tout le bloc – Si vous modifiez une entité faisant partie d’un bloc, sélectionnez cette option pour appliquer toute modification ci-dessus à la totalité du bloc.

Utilisez l’outil Propriétés pour modifier les autres informations relatives à une entité, comme les coordonnées géométriques, les angles, etc.

Modification des attributs d’une entité

  1. Cliquez sur Attributs dans le menu Modifier.
  2. Cliquez sur l’entité dont vous souhaitez modifier les attributs.
  3. Effectuez les modifications, puis cliquez sur OK.
