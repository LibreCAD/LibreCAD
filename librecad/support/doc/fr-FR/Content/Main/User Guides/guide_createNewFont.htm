Création de nouvelles polices

Avec LibreCAD, vous pouvez facilement créer une nouvelle police en utilisant une police existante comme modèle. Dans LibreCAD, chaque caractère de police est un bloc, et chaque bloc est désigné d’après le code de caractère Unicode.

Création d’une police à partir d’un modèle

Étape 1

Trouvez une police existante à utiliser comme modèle pour votre nouvelle police. Essayez de trouver une police similaire à celle que vous souhaitez créer, ou choisissez une police standard comme Times New Roman.

Si vous ne savez pas quelle police utiliser, vous pouvez consulter un aperçu des polices dans la boîte de dialogue Convertir police depuis Outils > MTexte > Convertir police ( ).

 

Étape 2

Localisez la police à l’aide de l’explorateur de fichier sur votre ordinateur.

 

Si vous utilisez une police que vous avez convertie depuis AutoCAD ou Microsoft, le fichier de la police se trouvera à l’emplacement indiqué par le chemin que vous avez précisé dans Paramètres de l’application > Chemin d’accès > Polices.

 

Si vous utilisez une police LibreCAD, le fichier de la police sera placé dans ce dossier : C:\Program Files (x86)\Hypertherm CAM\LibreCAD\resources\fonts

 

Étape 3

Ouvrez le fichier de la police (*.LFF) dans LibreCAD en sélectionnant Fichier > Ouvrir, puis choisissez le type de fichier Police LFF (*.lff). Accédez ensuite au dossier approprié et cliquez sur le fichier de la police que vous souhaitez ouvrir.

 

Étape 4

Le fichier de la police est maintenant ouvert dans LibreCAD. Veuillez noter que chaque caractère est indiqué dans une ligne avec l’Unicode correspondant sous chaque caractère. Veuillez également remarquer que la Liste de blocs présente chaque caractère, par nom Unicode, en tant que bloc individuel.

 

Pour modifier un caractère :

  1. Localisez le bloc correspondant à ce caractère dans la Liste de blocs.
  2. Cliquez sur le bloc dans la Liste de blocs.
  3. Cliquez sur Éditer le bloc actif dans une nouvelle fenêtre ( ).

Le bloc actif s’ouvrira dans une nouvelle fenêtre.

 

Étape 5

Apportez les modifications au caractère, s’il y a lieu. Lorsque vous avez terminé de modifier un caractère, quittez l’onglet, ce qui vous ramènera au fichier LFF de la police complet. Il n’est pas nécessaire d’enregistrer le bloc dans un fichier individuel; le bloc sera conservé dans le fichier LFF.

 

Répétez cette procédure jusqu’à ce que tous les caractères soient modifiés, si nécessaire.

 

Étape 6

Il est conseillé d’enregistrer la police modifiée dans un fichier différent afin de conserver l’état original de la police.

 

Pour enregistrer la police dans un fichier différent :

  1. Cliquez sur Fichier.
  2. Cliquez sur Enregistrer sous.
  3. Localisez le dossier dans lequel vous souhaitez enregistrer le fichier de la police. Il est conseillé d’utiliser le fichier suivant : C:\Program Files (x86)\Hypertherm CAM\LibreCAD\resources\fonts.
  4. Depuis le menu déroulant de types de fichier, sélectionnez Fichier LFF (*.lff).
  5. Nommez le fichier, mais n’incluez pas d’extension de fichier dans le nom. Par exemple, écrivez « MaPolice » plutôt que « MaPolice.lff ».
  6. Cliquez sur Enregistrer.

Si vous enregistrez la police dans le dossier de police LibreCAD de l’étape 6.3, vous devrez modifier les autorisations de dossier afin de pouvoir enregistrer votre dossier. Si cela n’est pas possible, enregistrez le fichier de la police dans le dossier que vous avez indiqué dans Paramètres de l’application > Chemin d’accès > Polices.

Étape 1 – Italict.lff est la police à utiliser comme modèle.
