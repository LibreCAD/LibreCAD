Utilisation de blocs

Les blocs sont de petits dessins réutilisables d’objets généralement utilisés comme des boulons, des cartouches, etc. Les blocs insérés, également appelés insertions, sont des formes géométriques (lignes, arcs) et peuvent également inclure du texte et des cotations. LibreCAD possède plusieurs bibliothèques de blocs qui peuvent être insérés dans un dessin; vous pouvez également créer votre propre bloc.

 

Il existe deux gadgets logiciels d’ancrage pour gérer les blocs :

  • L’ancrage de la Liste de blocs pour la gestion des blocs dans le dessin actif; et
  • L’ancrage du Menu des bibliothèques qui affiche l’ensemble des blocs à disposition et qui permet aux utilisateurs d’insérer des blocs dans le dessin actif.

Apprenez-en davantage sur les Gadgets logiciels d’ancrage ici et sur le Menu des bibliothèques ici.

 

Lorsqu’il est inséré dans un dessin, un bloc est traité comme une seule entité. Les blocs peuvent être insérés depuis la Liste de blocs plusieurs fois dans le même dessin, à des emplacements et selon des échelles et des angles de rotation différents. Les blocs insérés sont tous liés. Les modifications apportées à un bloc inséré se répercuteront sur toutes les instances du bloc.

Création d’un bloc à partir d’un dessin existant

  1. Dessinez l’objet que vous souhaitez convertir en bloc.
  2. Sélectionnez tous les éléments qui constitueront le bloc. Pour ce faire, cliquez sur les éléments individuels ou sélectionnez plusieurs éléments dans une zone.
  3. Cliquez sur Créer Bloc ( ) depuis l’ancrage Liste de blocs.
  4. Précisez un point de référence.
  5. Nommez le bloc et cliquez sur OK.

Le bloc s’affiche maintenant dans la Liste de blocs et peut être inséré plusieurs fois dans le dessin ou enregistré dans un fichier ou dans la Bibliothèque.

Création d’un bloc à partir d’un bloc vide

  1. Cliquez sur Ajouter un bloc vide ( ) depuis l’ancrage Liste de blocs.
  2. Nommez le bloc et cliquez sur OK.
  3. Cliquez sur Éditer le bloc actif dans une nouvelle fenêtre ( ). Cela ouvrira un onglet de document de dessin de bloc vide.
  4. Dessinez le bloc. L’origine du dessin (0,0) sera le point de référence du bloc.
  5. Quittez l’onglet de dessin du bloc. Le bloc sera enregistré et pourra être inséré dans un dessin.

Insertion de blocs dans un dessin

  1. Dans la Liste de blocs, cliquez sur le bloc que vous souhaitez insérer.
  2. Cliquez sur Insérer le bloc actif ( ) depuis l’ancrage Liste de blocs.
  3. Précisez le point d’insertion en cliquant sur le dessin, ou en indiquant les coordonnées dans la ligne de commande.

Insertion de blocs dans un dessin à l’aide de la barre d’outils (Avancé)

  1. Dans la Liste de blocs, cliquez sur le bloc que vous souhaitez insérer.
  2. Cliquez sur Insérer le bloc actif ( ) depuis l’ancrage Liste de blocs.
  3. Précisez les paramètres qui s’afficheront dans la barre d’outils.

  • Paramètres liés aux transformations du bloc : Angle de rotation et Facteur de mise à l’échelle.
  • Paramètres liés aux modèles de bloc : Matrice (taille de grille) et Espacement entre les colonnes et les lignes de la matrice.
  1. Précisez le point d’insertion en cliquant sur le dessin, ou en indiquant les coordonnées dans la ligne de commande.

Suppression de l’instance d’un bloc dans un dessin

  1. Cliquez sur le bloc sur le dessin.
  2. Appuyez sur la touche Supprimer.

Suppression d’un bloc et de toutes ses instances dans un dessin

La suppression d’une instance d’un bloc dans un dessin supprimera uniquement le bloc qui était placé sur un calque, et le bloc sera toujours stocké dans le fichier DXF. Pour supprimer totalement un bloc à la fois du calque du dessin et du fichier DXF, effectuez les actions suivantes :

  1. Cliquez sur le bloc dans la Liste de blocs.
  2. Cliquez sur Supprimer le bloc actif ( ) dans la Liste de blocs.

Éclatement des blocs

Vous pouvez éclater les blocs afin de modifier leur géométrie ou des entités individuelles. Lorsque les blocs sont d’abord insérés dans un dessin, le bloc est traité comme un tout et ne peut être modifié que s’il est éclaté.

  1. Entrez explode dans la ligne de commande ou cliquez sur Éclater ( ) dans les options Modifier.
  2. Cliquez sur le bloc que vous souhaitez éclater.
  3. Appuyez sur la touche Entrée.

Modification des blocs

Si le bloc est enregistré dans un fichier, il suffit d’ouvrir le fichier, d’effectuer les modifications et d’enregistrer avant de quitter.

 

Si le bloc n’est pas enregistré dans un dessin spécifique, mais qu’il fait partie du dessin actif sur lequel vous travaillez, vous pouvez le modifier en effectuant les actions suivantes :

  1. Dans la Liste de blocs, cliquez sur le bloc que vous souhaitez modifier.
  2. Dans la Liste de blocs, cliquez sur Éditer le bloc actif dans une nouvelle fenêtre (  ).
  3. Apportez les modifications au bloc.
  4. Quittez le dessin du bloc.

Le bloc a été mis à jour.

Insertion de blocs depuis la Bibliothèque

  1. Depuis l’ancrage Menu des bibliothèques, cliquez sur le bloc que vous souhaitez insérer.
  2. Cliquez sur Insérer.
  3. Cliquez sur un point de référence dans le dessin ou entrez les coordonnées dans la ligne de commande.

Le bloc peut être placé autant de fois que nécessaire. Appuyez sur la touche Échap ou effectuez un clic droit pour quitter le mode d’insertion du bloc.

Enregistrement des blocs dans un fichier

  1. Dans la Liste de blocs, cliquez sur le bloc que vous souhaitez enregistrer.
  2. Cliquez sur Enregistrer le bloc actif dans un fichier ( ).
  3. Indiquez un nom et enregistrez un emplacement pour le fichier, puis cliquez sur OK.
