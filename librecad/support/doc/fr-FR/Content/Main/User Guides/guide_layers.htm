Utilisation des calques

Création de calques

  1. Cliquez sur Ajouter un calque ( ) sur l’ancrage Liste des calques.
  2. Nommez le calque. De plus, indiquez si le calque est un calque de construction et réglez les options de stylet par défaut, s’il y a lieu.
  3. Cliquez sur OK.

Suppression de calques

Supprimer un calque effacera le calque et toutes les formes géométriques qui existent sur ce calque.

  1. Cliquez sur le calque que vous souhaitez supprimer sur l’ancrage Liste des calques.
  2. Cliquez sur Supprimer calque (  ), ou effectuez un clic droit et cliquez sur Supprimer calque.
  3. Cliquez sur OK.

Activation d’un calque

  • Il suffit de cliquer sur un calque dans l’ancrage Liste des calques pour le rendre actif.

Modification de la couleur d’un calque et d’autres attributs

  1. Cliquez sur le calque à modifier.
  2. Cliquez sur Modifier calque (  ).
  3. Effectuez les ajustements aux paramètres du calque.
  4. Cliquez sur OK.

Modification du calque d’une entité

  1. Cliquez sur une ou plusieurs entités.
  2. Cliquez sur Attributs ( ) ou sur Propriétés ( ) à partir des outils Modifier.
  3. Cliquez sur un calque différent à partir du menu déroulant Calque.
  4. Cliquez sur OK.

Tri des calques

Les calques sont affichés en ordre alphanumérique dans la liste des calques. Cependant, l’ordre des calques ne reflète pas l’ordre dans lequel les entités apparaissent sur le dessin. Utilisez les commandes dans outils Modifier > Ordre ( ) pour déplacer les entités vers le haut ou vers le bas.

Action Touche Résultat
Déplacer vers le haut [Accueil] Déplace l’entité sélectionnée vers la position la plus haute.
Déplacer vers le bas [Fin] Déplace l’entité sélectionnée vers la position la plus basse.
Placer au-dessus de l’entité [Haut de page] Déplace l’entité sélectionnée d’une rangée vers le haut.
Placer au-dessous de l’entité [Bas de page] Déplace l’entité sélectionnée d’une rangée vers le bas.

Création de calques de construction

Pour définir un calque existant en tant que calque de construction :

  • Cliquez sur Calque de construction ( ) à côté du nom du calque.

Lorsque le bouton est vert, cela signifie que le calque est défini comme un calque de construction. Lorsque le bouton est grisé, le calque n’est pas un calque de construction.

Créez de nouveaux calques qui n’ont pas de processus attribué dans ProNest pour les utiliser comme calques de construction. Cela évite d’importer dans ProNest des formes géométriques non souhaitées provenant de calques de construction.

Par exemple, si un calque de coupe est utilisé en tant que calque de construction pour créer des lignes de construction, les entités de lignes utilisées pour créer les lignes de construction seront reconnues comme une forme géométrique dans ProNest. Cependant, si un nouveau calque est utilisé et qu’il n’est pas reconnu par ProNest, les lignes utilisées pour créer les lignes de construction seront ignorées par ProNest.

Création de lignes de construction

Pour créer des lignes de construction d’une longueur infinie :

  1. Assurez-vous que le calque de construction est actif (  ).
  2. Créez une ligne en utilisant un des outils Ligne ou Polyligne. Une nouvelle ligne d’une longueur infinie sera dessinée en tant qu’aperçu sur cette ligne.

    3. Si vous placez des lignes de construction sur un calque ayant un processus attribué dans ProNest, vous devrez supprimer les entités de lignes utilisées pour créer les lignes de construction avant d’importer le dessin dans ProNest. Il est conseillé de créer de nouveaux calques sans processus attribué dans ProNest à utiliser comme calques de construction pour éviter d’importer dans ProNest des formes géométriques non souhaitées provenant des calques de construction.

  3. Ajustez la ligne, puis cliquez n’importe où sur le dessin pour placer la ligne. Cette sélection deviendra le point de référence pour créer une autre ligne de construction.

  4. Placez autant de lignes de construction que nécessaire, en répétant l’étape 3.

  5. Lorsque vous aurez terminé de dessiner les lignes de construction, effectuez un clic droit ou appuyez sur la touche Échap pour quitter le mode de création de lignes de construction.

Masquage des lignes de construction

Pour rendre les lignes de construction invisibles tout en maintenant toutes les autres formes géométriques visibles sur un calque :

  • Cliquez sur Calque de construction ( ) à côté du nom du calque.

Le bouton du calque de construction deviendra grisé et toutes les lignes de construction seront masquées.

Les lignes de construction que vous avez créées ne seront pas supprimées en activant le bouton du calque de construction; elles seront uniquement masquées.

Suppression des lignes de construction

  1. Cliquez sur une ou plusieurs lignes de construction.
  2. Appuyez sur la touche Supprimer.

En suivant ces étapes, vous pourrez supprimer toute ligne de construction infinie, mais pas les formes géométriques qui ont été utilisées pour créer les lignes de construction (lignes et polylignes).
