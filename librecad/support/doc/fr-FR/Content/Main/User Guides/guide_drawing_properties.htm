Modification des propriétés d’une entité

Après avoir placé une entité, comme des lignes, des arcs et du texte, ou des cotations, vous pouvez modifier les attributs et les propriétés qui sont spécifiques au type d’entité que vous modifiez, comme les coordonnées géométriques, les calques, les styles de texte ou l’information des étiquettes. La boîte de dialogue Propriétés est différente selon le type d’entité. Les segments individuels d’une polyligne sont modifiés individuellement. Généralement, l’outil Propriétés permet de gérer la géométrie de l’entité alors que l’outil Attributs est principalement utilisé pour le stylet et les informations relatives aux calques.

Modification des propriétés d’une entité

  1. Cliquez sur Propriétés( ) dans le menu Modifier.
  2. Cliquez sur l’entité dont vous souhaitez modifier les propriétés.
  3. Effectuez les modifications, puis cliquez sur OK.
