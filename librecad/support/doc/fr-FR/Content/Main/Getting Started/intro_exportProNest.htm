Exportation des dessins vers ProNest

Les dessins peuvent directement être importés dans ProNest sans avoir à passer par la fenêtre Modifier liste des pièces.

  • Cliquez sur le bouton Exporter vers ProNest ( ) lorsque vous êtes prêt à importer votre dessin.

Ouvrez ProNest, puis ouvrez LibreCAD afin que LibreCAD utilise les paramètres d’Importation de pièces (ProNest > Paramètres > Importation de pièces) pour les informations de calque de CAO et d’autres propriétés d’importation. Cela n’a besoin d’être effectué qu’une seule fois, sauf si vous avez apporté des modifications aux paramètres d’Importation de pièces.
