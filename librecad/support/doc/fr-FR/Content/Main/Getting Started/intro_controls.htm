Commandes de souris et de clavier, zoom et panoramique

Commandes de souris

Sélectionner une entité

Cliquez une fois sur une entité pour la sélectionner.

Sélectionner plusieurs entités

Cliquez sur plusieurs entités pour les sélectionner. Vous n’avez pas à maintenir enfoncées d’autres touches.

Sélectionner toutes les entités du dessin

Lorsqu’une entité a été sélectionnée, cliquez à nouveau sur cette entité pour la désélectionner.

Zoomer sur l’étendue du dessin

Double-cliquez sur la roue de défilement pour zoomer automatiquement sur l’étendue du dessin.

Ouvrir un menu des outils récemment utilisés

Effectuez un clic droit sur la zone du dessin pour ouvrir un menu des 15 derniers outils utilisés.

Commandes du clavier

Sélectionner toutes les entités du dessin

Sélectionnez Ctrl+A pour sélectionner toutes les entités du dessin.

Désélectionner toutes les entités

Pour désélectionner toutes les entités sélectionnées, effectuez une des actions suivantes :

  • Sélectionnez Échap.
  • Sélectionnez Ctrl+K.

Basculer les options d’accrochage on/off

Appuyez sur la barre d’espace pour activer/désactiver les options d’accrochage. Si cela ne fonctionne pas, vérifiez les Paramètres de l’application.

Saisir les commandes dans la ligne de commande

Depuis l’écran principal de dessin, vous pouvez commencer à écrire, ce qui remplira automatiquement la ligne de commande, sans avoir à cliquer d’abord dans le champ de texte.

Faire défiler le dessin verticalement

Pour faire défiler le dessin vers le haut ou le bas sans recourir au zoom, effectuez une des actions suivantes :

  • Maintenez la touche Ctrl enfoncée tout en utilisant la barre de défilement
  • Utilisez la barre de défilement

Faire défiler le dessin horizontalement

Pour faire défiler le dessin vers la gauche ou vers la droite sans recourir au zoom, effectuez une des actions suivantes :

  • Maintenez la touche Maj enfoncée tout en utilisant la barre de défilement
  • Utilisez la barre de défilement

Quitter un mode Outil

Appuyez sur Échap pour quitter un mode Outil.

Ouvrir l’aide LibreCAD

Appuyez sur F1 pour ouvrir l’aide contextuelle.

Afficher toutes les commandes de la ligne de commande

Depuis la zone de dessin principale, appuyez sur Tab pour obtenir une liste de toutes les commandes offertes pour la ligne de commande.

Basculer en mode écran complet

Appuyez sur F11 pour entrer/quitter le mode Écran complet.

Zoomer

Vous pouvez effectuer un zoom avant ou arrière sur un dessin avec les actions suivantes :

  • Utilisez la roue de défilement de votre souris.
  • Cliquez sur les boutons Zoom avant (), Zoom arrière () ou Zoom automatique (), qui se trouvent sous les boutons de menu principal en haut de l’écran.

Des options supplémentaires de zoom peuvent être configurées dans les Paramètres de l’application.

Panoramique

Vous pouvez défiler en mode panoramique ou déplacer l’écran de dessin comme suit :

  • Maintenez la roue de défilement jusqu’à ce que vous entendiez un clic, puis faites glisser la souris pour déplacer l’écran.
  • Cliquez sur le bouton Défilement du zoom () qui se trouve sous les boutons de menu principal en haut de l’écran, puis cliquez et faites glisser la souris pour déplacer l’écran.
