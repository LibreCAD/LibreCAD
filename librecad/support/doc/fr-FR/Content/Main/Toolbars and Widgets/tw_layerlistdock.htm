Gadget logiciel d’ancrage de la Liste des calques

Le gadget logiciel d’ancrage de la Liste des calques fournit une liste des calques dans le dessin actif et les fonctions pour gérer ces calques. La ligne d’entrée en haut de l’ancrage vous permet de filtrer par noms de calque (par exemple, « *01 » affichera les noms se terminant par « 01 »).

Fonctions de la Liste des calques

Outil Icône Description
Afficher tous les calques Rend tous les calques du dessin actif visibles.
Masquer tous les calques Masque tous les calques du dessin actif.
Ajouter un calque Ajoute un nouveau calque.
Supprimer le calque actif Supprime le calque sélectionné.
Modifier les attributs du calque Modifie les attributs du calque, y compris le nom du calque.

Fonctions de calque individuel

Icône Description
/ Afficher/masquer le calque.
/ Verrouiller/Déverrouiller le calque aux fins de modification.
Permettre d’inclure le calque dans le dessin imprimé.
/

Basculer les lignes de construction. Lorsque cette fonction est activée, toutes les lignes et tous les segments de ligne s’affichent avec une longueur infinie.

Pour en savoir plus, veuillez consulter Utilisation des calques de construction.

Affiche la couleur du calque actif (noir par défaut).
