Gadget logiciel créateur de menus

LibreCAD vous permet de créer des menus personnalisés pour accéder rapidement à vos outils préférés en utilisant une attribution de clics de souris ou de clavier.

 

Assigner

Cliquez pour attribuer un raccourci de souris/clavier à un menu personnalisé.

Mettre à jour

Cliquez pour enregistrer les modifications apportées à un menu personnalisé existant.

Créer un nouveau menu personnalisé

  1. Ouvrez Gadgets logiciels > Créateur de menus. La fenêtre Créateur de menus s’ouvrira.
  2. Dans le volet de gauche, cliquez sur n’importe quel outil, puis sur la flèche pointant vers la droite (>) pour l’ajouter à votre liste de menus dans le volet de droite. La sélection d’une catégorie dans le menu déroulant de gauche permettra de filtrer les outils qui s’affichent dans le volet de gauche.
    • Faites glisser et déposez les éléments pour les réorganiser dans votre nouveau menu du volet de droite.
    • Supprimez des éléments de votre nouveau menu en sélectionnant l’élément Outil, puis en sélectionnant la flèche pointant vers la gauche (<).
  3. Lorsque vous aurez ajouté tous les outils souhaités, veillez à nommer votre nouveau menu dans le champ Nom.
  4. Cliquez sur Créer.
  5. Cliquez sur Assigner. La fenêtre Attribution de menus s’ouvrira.

  1. Choisissez une option pour l’attribution du menu.
  2. Cliquez sur Enregistrer.

Lorsque vous revenez à l’écran principal du dessin, vous pouvez utiliser l’attribution de menu à l’aide de la souris ou d’une touche pour que le menu s’affiche. Par exemple, si votre attribution de menu est réglée sur Double-clic, vous pouvez double-cliquer n’importe où sur le dessin et votre nouveau menu s’ouvrira.

Modifier un menu personnalisé existant

  1. Ouvrez Gadgets logiciels > Créateur de menus. La fenêtre Créateur de menus s’ouvrira.
  2. Dans le menu déroulant de droite, sélectionnez le menu personnalisé que vous souhaitez modifier.
  3. Apportez les modifications aux éléments de votre menu, changez leur ordre dans le menu ou l’attribution du raccourci.
  4. Une fois vos modifications terminées, cliquez sur Mettre à jour, ou quittez la fenêtre si vous souhaitez ignorer les modifications.

Supprimer un menu personnalisé

  1. Ouvrez Gadgets logiciels > Créateur de menus. La fenêtre Créateur de menus s’ouvrira.
  2. Dans le menu déroulant de droite, sélectionnez le menu personnalisé que vous souhaitez supprimer.
  3. Cliquez sur Détruire.
