Gadget logiciel créateur de barres d’outils

LibreCAD vous permet de créer des barres d’outils personnalisées que vous pouvez ancrer pour accéder rapidement à vos outils favoris.

Créer une nouvelle barre d’outils personnalisée

  1. Ouvrez Gadgets logiciels > Créateur de barre d’outils. La fenêtre Créateur de barre d’outils s’ouvrira.
  2. Dans le volet de gauche, cliquez sur n’importe quel outil, puis sur la flèche pointant vers la droite (>) pour l’ajouter à votre liste de barre d’outils dans le volet de droite. La sélection d’une catégorie dans le menu déroulant de gauche permettra de filtrer les outils qui s’affichent dans le volet de gauche.
    • Faites glisser et déposez les éléments pour les réorganiser dans votre nouvelle barre d’outils du volet de droite.
    • Supprimez les éléments de votre nouvelle barre d’outils en sélectionnant l’élément outil, puis en sélectionnant la flèche pointant vers la gauche (<).
  3. Lorsque vous aurez ajouté tous les outils souhaités, veillez à nommer votre nouvelle barre d’outils dans le champ Nom.
  4. Cliquez sur Créer.

Vous devrez redémarrer LibreCAD pour que votre nouvelle barre d’outils s’affiche dans la liste Gadgets logiciels > Barres d’outils et soit accessible d’un clic droit dans la liste > Barres d’outils.

Modifier une barre d’outils personnalisée existante

  1. Ouvrez Gadgets logiciels > Créateur de barre d’outils. La fenêtre Créateur de barre d’outils s’ouvrira.
  2. Dans le menu déroulant à droite, sélectionnez la barre d’outils personnalisée que vous souhaitez modifier.
  3. Apportez les modifications aux éléments de votre barre d’outils ou changez leur ordre dans la barre d’outils.
  4. Cliquez sur Créer pour enregistrer vos modifications, ou quittez la fenêtre si vous souhaitez ignorer les modifications.

Supprimer une barre d’outils personnalisée

  1. Ouvrez Gadgets logiciels > Créateur de barre d’outils. La fenêtre Créateur de barre d’outils s’ouvrira.
  2. Dans le menu déroulant à droite, sélectionnez la barre d’outils personnalisée que vous souhaitez supprimer.
  3. Cliquez sur Détruire.
