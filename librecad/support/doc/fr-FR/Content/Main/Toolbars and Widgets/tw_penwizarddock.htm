Gadget logiciel d’ancrage de l’assistant Stylet

Le gadget logiciel d’ancrage de l’assistant Stylet vous permet de créer une palette de vos couleurs préférées pour les outils de dessin.

 

Utilisation d’une couleur de l’assistant Stylet

  • Veillez à ce que la couleur fasse partie de votre liste Favoris, puis double-cliquez sur la couleur afin qu’elle soit active.

Tout ce que vous dessinerez sera désormais de la couleur que vous avez sélectionnée.

Ajouter une couleur à la liste Favoris de l’assistant Stylet

  1. Cliquez sur une couleur dans le menu déroulant, ou cliquez sur le bouton de couleur personnalisée ().
  2. Cliquez sur le bouton Ajouter aux favoris ().

La couleur s’affiche maintenant dans la liste Favoris de l’assistant Stylet.

Création d’une couleur personnalisée

  1. Cliquez sur Couleur personnalisée ().
  2. Créez une couleur personnalisée, en saisissant soit les valeurs Hue/Sat/Val ou Red/Green/Blue, soit un code couleur HTML ou en faisant glisser le viseur sur la palette de couleurs et en réglant le curseur sur l’échelle de clarté.
  3. Cliquez sur OK pour créer une seule couleur personnalisée, ou cliquez sur Ajouter aux couleurs personnalisées pour créer plusieurs couleurs personnalisées à la fois.
  4. Cliquez sur OK.

La nouvelle couleur personnalisée que vous avez créée s’affiche dans la liste déroulante de l’assistant Stylet.

Suppression d’une couleur de la liste Favoris

  • Effectuez un clic droit sur la couleur que vous souhaitez supprimer, puis cliquez sur Supprimer.

Réorganiser les couleurs dans la liste Favoris

  • Faites glisser et déposez les couleurs de la liste pour les réorganiser.
