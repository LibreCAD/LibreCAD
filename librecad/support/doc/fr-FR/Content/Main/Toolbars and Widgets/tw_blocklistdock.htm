Gadget logiciel d’ancrage de la Liste de blocs

L’ancrage Liste de blocs fournit les fonctions permettant de gérer une liste de blocs existante dans le fichier du dessin.

 

La partie inférieure de l’ancrage affiche une liste des blocs dans le dessin actif. Les blocs de l’exemple ci-dessus s’appellent « MonBloc1 », « MonBloc2 » et « MonBloc3 ».

Fonctions des blocs

L’accès à certains de ces éléments est également possible en effectuant un clic droit sur un calque.

Outil Icône Description
Afficher/Débloquer tous les blocs Rend tous les blocs du dessin actif visibles.
Masquer/Bloquer tous les blocs Masque tous les blocs du dessin actif.
Créer un bloc Crée un bloc à partir des éléments sélectionnés.
Ajouter un bloc vide Crée un bloc vide qui peut ensuite être modifié dans une fenêtre/un onglet séparé(e) du dessin.
Supprimer le bloc actif Supprime le bloc en surbrillance.
Renommer le bloc actif Renomme le bloc en surbrillance.
Éditer le bloc actif dans une nouvelle fenêtre Ouvre une nouvelle fenêtre de dessin pour modifier un bloc nouveau ou existant.
Enregistrer le bloc actif dans un fichier Enregistre le bloc en surbrillance dans un fichier.
Insérer le bloc actif Insère le bloc en surbrillance dans le dessin actif au point de référence indiqué.
