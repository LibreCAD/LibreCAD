Outils supplémentaires : Texte, Hachure, Point

Outil Icône Commande Description
MTexte mtxt, mtext

Insérez du texte sur plusieurs lignes dans le dessin à partir d’un point de base spécifié. De manière facultative, définissez la police, la hauteur de texte, l’angle, le facteur de largeur, l’alignement, l’angle, les symboles spéciaux et l’ensemble de caractères.

 

Pour en savoir plus, veuillez consulter Utilisation de MTexte.

 
Texte txt, text

Insérez du texte sur une ligne dans le dessin à partir d’un point de base spécifié. De manière facultative, définissez la police, la hauteur de texte, l’alignement, l’angle, les symboles spéciaux et l’ensemble de caractères.

 

Pour en savoir plus, veuillez consulter Utilisation de Texte.

 
Hachure ha, hatch

Remplissez un contour (polygone, cercle, polyligne, etc.) avec un modèle défini ou un remplissage intégral. Si l’entité est un profil ouvert, le modèle de hachure pourrait être mal dessiné. Da manière facultative, définissez l’échelle et l’angle.

 

Annexe relatif au modèle de hachure.

 
Point po, point Dessinez un point en sélectionnant un endroit sur le dessin ou en entrant les coordonnées dans la ligne de commande.
