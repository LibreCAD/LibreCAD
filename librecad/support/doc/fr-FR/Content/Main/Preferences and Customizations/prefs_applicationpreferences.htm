Paramètres de l’application

Vous pouvez décider de l’aspect graphique de LibreCAD pour ProNest, définir les chemins d’accès de fichier pour les polices, les bibliothèques de pièces et ainsi de suite, et définir d’autres paramètres par défaut de LibreCAD pour ProNest dans les Paramètres de l’application (Options > Paramètres de l’application).

Apparence

Modifiez l’apparence visuelle de LibreCAD, y compris les indicateurs d’accrochage, l’espacement des grilles et les couleurs utilisées dans l’application.

 

Lignes de l’indicateur d’accrochage

Affiche des lignes visibles à des fins d’alignement lors de l’utilisation d’un outil.

Viseur

Viseur 2

Isométrique

Toile d’araignée

L’angle des lignes se règle de manière dynamique en fonction de la position du curseur.

Forme de l’indicateur d’accrochage

Affiche la forme du curseur lors de l’utilisation d’un outil.

Cercle

Carré

Point

Anticrénelage

Lorsqu’elle est activée, cette option donne aux lignes dentelées un aspect plus lisse.

Couleurs des graphiques

Entrez la valeur hexadécimale ou le nom de la couleur, ou choisissez une couleur personnalisée dans le sélecteur de couleur ( ) pour obtenir la couleur que vous souhaitez utiliser pour chaque option offerte dans cette section.

Chemins d’accès

Définit les chemins d’accès de fichiers de Traductions, Motifs de hachures, Polices, Bibliothèques de pièces, Modèle de dessin et Fichier de variables. Ces chemins d’accès ne remplacent pas les chemins par défaut dans le logiciel; ils sont plutôt utilisés comme supplément.

Traductions

À utiliser uniquement si vous possédez un fichier de traduction QM, qui gère les traductions de l’anglais vers d’autres langues.

Motifs de hachures

À utiliser uniquement si vous possédez des fichiers de motifs de hachures dans votre ordinateur. Sinon, les motifs de hachures LibreCAD par défaut sont toujours utilisables.

Polices

Précisez le dossier unique que vous utilisez pour enregistrer les polices Microsoft ou AutoCAD converties.

Bibliothèques de pièces

Indiquez le dossier qui contient un autre dossier contenant les dessins de fichiers DXF que vous souhaitez avoir à disposition dans le Menu de bibliothèque. Ces fichiers DXF peuvent être insérés dans un autre dessin en tant que bloc.

Modèle

Le fichier que LibreCAD utilise comme modèle pour les nouveaux dessins.

Fichier de variables

Un fichier de texte de variables qui peut être chargé depuis la ligne de commande lorsqu’un nouveau dessin est créé.

Paramètres par défaut

Configurez les paramètres par défaut de vos dessins, dont les unités, les opérations d’enregistrement automatique, etc.

Paramètres par défaut des nouveaux dessins

Unités

Choisissez les unités dans le menu déroulant.

Paramètres par défaut du programme

Sauvegarde automatique

Lorsqu’elle est activée, LibreCAD créera automatiquement un fichier de sauvegarde qui se mettra à jour en fonction de la durée de sauvegarde automatique.

 

Ne pas utiliser la boîte de dialogue du fichier système

Lorsqu’elle est activée, la fenêtre de boîte de dialogue qui s’affiche à l’ouverture d’un fichier aura la même apparence que l’application LibreCAD, plutôt que celle du système d’exploitation utilisé par votre ordinateur.

 

Inverser le sens

Lorsqu’une option est cochée, la direction du défilement horizontal, du défilement vertical et du zoom est inversée.

Paramètres du clavier

Évalue les commandes lorsque la SPACE BAR est actionnée

Lorsqu’elle est activée, la barre d’espace peut être utilisée pour envoyer les commandes dans la ligne de commande.

 

Bascule vers le mode Alignement libre lorsque la SPACE BAR est actionnée

Lorsqu’elle est activée, la barre d’espace peut être utilisée pour basculer vers le mode d’accrochage libre.

Réinitialiser les paramètres

Mise en page

Lorsque cette option est sélectionnée, toutes les modifications apportées à la disposition de l’application sont annulées et la mise en page revient à son état original.

 

Tout

Lorsque cette option est sélectionnée, toutes les modifications apportées à l’application, y compris la mise en page, sont annulées et l’application revient à son état original d’installation.

Démarrage

Afficher l’écran de chargement

Lorsqu’elle est activée, vous ne verrez pas l’écran de présentation LibreCAD pour ProNest s’afficher au démarrage.

 

Démarrer en mode Onglet

Lorsqu’elle est activée, l’application démarrera en mode Onglet plutôt qu’en mode Fenêtre.

 

Démarrer avec la fenêtre principale maximisée

Lorsqu’elle est activée, la fenêtre de l’application sera maximisée à l’ouverture.

 

Activer les gadgets logiciels d’accrochage de CAO

Lorsqu’elle est activée, les gadgets logiciels d’accrochage peuvent être utilisés.

 

Activer les barres d’outils de CAO

Lorsqu’elle est activée, les barres d’outils peuvent être utilisées.

Paramètres de CAO

Modifier le calque des éléments sélectionnés dès son activation

Lorsqu’elle est activée, toute entité actuellement sélectionnée dans un dessin sera déplacée vers le calque suivant qui est sélectionné.

 

Par exemple, avec une entité sur le calque 0, cliquez sur l’entité, puis sur calque SCRIBE; l’entité sélectionnée sera déplacée sur le calque SCRIBE.

 

Zoom automatique sur le dessin à l’ouverture

Lorsqu’elle est activée, l’application zoomera automatiquement sur l’étendue d’un dessin lorsqu’un dessin est ouvert.
