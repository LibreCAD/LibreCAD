Overview to using LibreCAD for ProNest

Navigating the main screen

LibreCAD for ProNest offers a highly customizable main drawing screen, where you can rearrange and resize toolbars and dock widgets. The default setup is described below.

 

In general, there are six primary areas of the main LibreCAD application, highlighted in blue circles in the image below:

  1. Main menu area – Access File, Options, Edit, View, Plugins, Tools, Widgets, Drawings, and Help menus. Below the main menu area are file option buttons, copy/paste buttons, and zoom buttons. Additionally, the Tool status bar area is where you will find parameter prompts from some tools.
  2. Dock widget area – Library Browser, Block List, and Layer List.
  3. Command Line – Command Line prompt window dock. Monitor this area for prompts when using tools.
  4. Snap and Status Bar area – Snap/restrict buttons, Set and Lock Relative Zero buttons, Toolbar expand options, and Menu and Toolbar creators. Below these buttons are coordinate information and selection information as well.
  5. Tools for quick access – Main drawing tools menus, including Lines, Circles, Curves, Ellipses, Polylines, Select, Dimension, Modify, and Measure tools.
  6. Drawing area – Entities and text are drawn in this area. Image files can also be imported and placed in a drawing.

Importing drawings into LibreCAD

As a standalone application, you can open any of the following file types in LibreCAD:

  • AutoCAD (*.dxf; *.dwg)
  • LibreCAD Font (*.lff)
  • QCad Font (*.cxf)
  • Jww (*.jww)

Saving, Printing, and Exporting to ProNest

Saving

  • Select the Save ( ) or Save As ( ) buttons to save your drawing to a DXF file.

 

If you are exporting parts directly to ProNest, you'll be prompted to save the drawing file before export.

 

Printing

  • Select the Print ( ) and Print Preview ( ) buttons to print your drawing on paper.

Consult the Printing User Guide for best results.

 

Exporting to ProNest

  • Select the Export to ProNest button ( ) when you are ready to directly import your drawing into ProNest.

Open ProNest and then open LibreCAD so that LibreCAD will use your Importing Parts settings (ProNest > Settings > Importing Parts) for CAD layer information and other import properties. This only needs to be done once, unless you have make changes to your Importing Parts settings.
