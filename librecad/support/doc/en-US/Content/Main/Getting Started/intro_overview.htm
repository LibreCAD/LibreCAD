Navigating the main screen

LibreCAD for ProNest offers a highly customizable main drawing screen, where you can rearrange and re-size toolbars and dock widgets. The default setup is described below.

In general, there are six primary areas of the main LibreCAD application, highlighted in blue circles.

  1. Main menu area
    • File, Options, Edit, View, Plugins, Tools, Widgets, Drawings, and Help menus.

    • Below the main menu area are file option buttons (such as new drawing, open drawing), copy/paste buttons, and zoom buttons.

    • Pen options are above the custom toolbar. Pen options allow you to control pen color, thickness, and line type.

    • Tool status bar area is where you will find parameter prompts from some tools. The status bar doesn't display until you begin using a tool, such as Text. When active, it will appear below the Pen options. Monitor this area for prompts when using tools.

  2. Dock widget area – Library Browser, Block List, and Layer List.
  3. Command Line – Command Line prompt window dock. Monitor this area for prompts when using tools.
  4. Snap and Status Bar area
    • Snap/restrict buttons.

    • Set and Lock Relative Zero buttons.

    • Show/hide toolbar options.

    • Menu and Toolbar creators.

    • Below the buttons are information about mouse coordinates, length of selected entities, current layer, and grid status.

  5. Tools for quick access
    • Main drawing tools menus, including Lines, Circles, Curves, Ellipses, Polylines, Select, Dimension, Modify, and Measure tools.
    • Drawing tabs are located between the tools and the main drawing screen. The tabs are stacked vertically.
  6. Drawing area – Entities and text are drawn in this area. Image files can also be imported and placed in a drawing. By default, the background color is black, but this can be changed in the Application Preferences.
