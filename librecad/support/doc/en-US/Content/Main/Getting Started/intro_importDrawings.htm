Importing and opening drawings into LibreCAD

LibreCAD as a standalone application

As a standalone application, you can open any of the following file types in LibreCAD:

  • Drawing Exchange/AutoCAD (*.dxf; *.dwg)
  • LibreCAD Font (*.lff)
  • QCad Font (*.cxf)
  • Jww (*.jww)

Importing drawings from ProNest

Any file type that can be imported into ProNest can be opened in LibreCAD for CAD cleanup, either before or after the part is imported into ProNest.
