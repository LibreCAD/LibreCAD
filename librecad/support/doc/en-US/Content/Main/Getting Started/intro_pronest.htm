LibreCAD integration with ProNest

LibreCAD for ProNest and ProNest are integrated, making it easy to go from CAD to cut. LibreCAD can be accessed as:

  • a standalone CAD application, or
  • an editor to modify existing drawings and parts from ProNest.

LibreCAD for ProNest has a context-sensitive Help system that works just like the ProNest Help. For more information on using LibreCAD, select the F1 key when the application is open or when you're in a particular mode.

 

Any part that is imported into ProNest can be opened and edited in LibreCAD after import. DWG and DXF files can be edited from LibreCAD directly, without first going through the Edit Part List window in ProNest.

 

Once a file is opened and modified in LibreCAD, it must be saved as DXF file.

Installation

LibreCAD for ProNest, a Hypertherm-exclusive CAD program that includes special features such as Shape Text and Trim Excess, will automatically install when you install or update ProNest. If you already have the base LibreCAD program installed on your PC (not LibreCAD for ProNest), the ProNest installer won't overwrite your existing installation and is instead a separate program. Additionally, in this scenario, you can use LibreCAD and LibreCAD for ProNest independently.

Creating and editing drawings from ProNest

Create a new drawing from ProNest

In the Edit Part List window, click the LibreCAD for ProNest button ( ) to launch the program, where you can create a new drawing.

When you are finished, you can click either the Exit button to return to ProNest or the Export to ProNest button ( ) to import a drawing into ProNest without exiting LibreCAD. When using the Export to ProNest button, ensure that ProNest is already running on your computer for best results.

Open CAD files in LibreCAD before part import

  1. Select a CAD file in the Part Sources section of the Edit Part List window.
    • You can also select multiple files to bring into LibreCAD at once by holding down the Ctrl key on your keyboard and clicking multiple files.
  2. In the Preview, click LibreCAD from the dropdown menu.
    • Finish making changes to your drawing and exit LibreCAD before you can continue working in ProNest.
  3. In LibreCAD, make the required changes to the CAD file (modifying geometry, changing layer properties, configuring blocks, setting dimensions, and so on).
  4. Click Exit. You'll be prompted to save your drawing and return to ProNest.
  • Modifying a part in LibreCAD after it has been imported will lose any existing bevels that were added to the part in ProNest.
  • Parts that are chained, bridged, or common line cut aren't editable in LibreCAD.
  • If you have imported multiple quantities of a part (for example, 10 required quantities of Base Plate), all instances of the part will be edited. To edit only one instance of a part, you must add that part to the part list individually.

Open CAD files in LibreCAD after part import

You can make some modifications to a part after it has been imported into ProNest, such as changes to geometry, text, dimensions, and display-only information.

Layer information may be modified, but only new layers can be added that are listed in your ProNest Importing Parts settings (ProNest > Settings > Importing Parts > CAD Import tab). Any new layers added to a drawing that aren't defined in ProNest before the part is imported will be discarded.

  1. From the Part List or nest, right-click on one or more parts.
  2. Click Edit in LibreCAD for ProNest ( ).
  3. In LibreCAD, make the required changes to the CAD file.
  4. Click Exit. You'll be prompted to save your drawing and return to ProNest.
