LibreCAD integration with ProNest

LibreCAD and ProNest are integrated, making it easy to go from CAD to cut. LibreCAD can be accessed as:

  • a standalone CAD application, or
  • an editor to modify existing drawings and parts from ProNest.

LibreCAD has a context-sensitive Help system that works just like the ProNest Help. For more information on using LibreCAD, press the F1 key when the application is open or when you're in a particular mode.

 

Any of the following file types can be edited in LibreCAD from the ProNest Part List:

  • VSP
  • Pipe
  • AutoCAD (.dxf; .dwg)
  • IGES files (.igs; .iges)
  • EPS files (.eps)
  • DGN files (.dgn)
  • DSTV files (.nc; .nc1)
  • PDF files, single-page only (.pdf)
  • Raster image files (.bmp; .jpg; .jpeg; .gif; .png; .tff)
  • Autodesk Inventor files (.iam; .ipt)

    • (Inventor™ Interface module required)

  • SolidWorks files (.sldprt; .sldasm)

    • (SolidWorks® Interface module required)

Once a file is opened and modified in LibreCAD, it must be saved as DXF file.

Installation

A Hypertherm-exclusive version of LibreCAD, which includes special features such as Shape Text and Trim Excess, will automatically install when you install or update ProNest. If you already have LibreCAD installed on your PC, the ProNest install won't overwrite your existing installation.

Creating and editing drawings from ProNest

Create a new drawing from ProNest

In the Edit Part List window, select the LibreCAD button ( ) to launch the program, where you can create a new drawing.

When you are finished, you can select the Exit button to return to ProNest, or select the Export to ProNest button () to import a drawing into ProNest without exiting LibreCAD, so that you can create more drawings.

Open CAD files in LibreCAD before part import

  1. Select a CAD file in the Part Sources section of the Edit Part List window.
    • You can also select multiple files to bring into LibreCAD at once by holding down the CTRL key on your keyboard and selecting multiple files.
  2. In the Preview, select LibreCAD from the dropdown menu.
    • Finish making changes to your drawing and exit LibreCAD before you can continue working in ProNest.
  3. In LibreCAD, make the required changes to the CAD file (modifying geometry, changing layer properties, configure blocks, set dimensions, and so on).
  4. Select the Exit button. You'll be prompted to save your drawing and return to ProNest.
  • Modifying a part in LibreCAD after it has been imported will lose any existing bevels that were added to the part in ProNest.
  • Parts that are chained, bridged, or common line cut aren't editable in LibreCAD.
  • If you have imported multiple quantities of a part (for example, 10 required quantities of Base Plate), all instances of the part will be edited. To edit only one instance of a part, you must add that part to the part list individually.

Open CAD files in LibreCAD after part import

You can make some modifications to a part after it has been imported into ProNEst, such as changes to geometry, text, dimensions, and display-only information.

Layer information may be modified, but only new layers can be added that are listed in your ProNest Importing Parts settings (Settings > Importing Parts > CAD Import tab). Any new layers added to a drawing that aren't defined in ProNest before the part is imported will be discarded.

  1. From the Part List or nest, right-click on one or more parts.
  2. Select the Edit in LibreCAD button ().
  3. In LibreCAD, make the required changes to the CAD file.
  4. Select the Exit button. You'll be prompted to save your drawing and return to ProNest.
