Using Templates

Templates are prototype drawings that provide the means to save basic parameters and settings so a drawing does not have to be configured each time a new one is started. The parameters and settings include the settings defined in the Drawing Preferences, such as the paper format, main unit of measure and format, and dimension format. Templates can also include layers and layer configuration, line type and thickness, pen color, and other drawing elements such as a border. These settings are inherited by the drawings created from the template.

You can use an existing drawing as a template, or you can use the default blank drawing as a template.

Add a title block to the default blank drawing template

You can add a title block, or other entities, to the blank default drawing that appears when LibreCAD is launched or a new drawing is created. This is useful to set up if you have entities that need to be present in all of your CAD drawings.

  1. Open the default DXF template file in LibreCAD: C:\Users\[Windows Username]\AppData\Local\Hypertherm CAM\ProNest 2019\ProNestTemplate.DXF
  2. Make changes to the template drawing as needed.
  3. Click Save ( ).

The template file is now updated. To verify your changes, create a new drawing.

If you can't locate the template file, the default drawing template is listed in the Command Line when a new drawing is created.

Use an existing drawing as a template

  1. Click File.
  2. Click New from Template.
  3. Select the drawing that you want to use as a template.
  4. Click Open.

Modify the drawing as needed. Note that the template drawing won't be modified in any way, and you'll need to save the new drawing as a separate file.
