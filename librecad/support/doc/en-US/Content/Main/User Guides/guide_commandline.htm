Using the Command Line

The Command Line offers users an alternative to using the mouse to select tools and draw. Using the keyboard to select tools and enter coordinates can provide greater speed and accuracy when creating drawings.

 

The command line provides other useful features not available using the mouse:

  • Multi-command input
  • A calculator

The command line is displayed in its own dock widget and consists of three components

  1. Command prompt.
  2. Command output window.
  3. A button ( ) that displays a drop-down menu that includes:
    • Keycode Mode on/off.
    • Load a Command file.
    • Paste multiple commands.

 

See the appendix for a comprehensive list of Command Line commands.

Entering functions into the Command Line

The commands available to use are shown on the reference pages for each tool category. The command line is activated in a variety of ways:

  • Start typing any command (for example, rect) and then press ENTER.
  • Press the Space bar, type any command and then press ENTER.
  • Press CTRL + M, any command and then press ENTER.
  • Press SHIFT, any command and then press ENTER.
  • With the Keycode Mode on, type a two letter command (for example, ci). Pressing ENTER is not required.
  • Tab completion can be used in the command line when entering commands. Enter a partial command, such as cir, then press the TAB key. If only one command is possible from the partial command, the full command will display in the text entry field. If more than one command is possible, a list of possible commands will display in the command window.

Depending on the command you're using, additional input may be required in the Command Line or Toolbar.

  • The SPACE BAR may be pressed instead of the ENTER key when Options > Application Preferences > Defaults > Evaluate commands when SPACE BAR is pressed is enabled (which is the default).
  • In addition to the comma separated coordinates, coordinates can also be entered using the numeric keypad using the X..Y format. For example, entering 10..20 is equivalent to @10,20. Using the two decimals may be faster than typing the comma.
  • The Keycode Mode permits the use of two letter commands and eliminates the need to press ENTER after typing the command.
  • Pressing c or u followed by ENTER can be used instead of typing close or undo.

Using the Command Line Calculator

  1. Enter Cal in the Command Line to enable Calculator Mode.
  2. Enter equations as needed. Calculator Mode can also be used to retrieve values, such as pi. Results will display in the command window.
  3. Exit Calculator Mode by again entering Cal in the Command Line.

See the list of Calculator Operators and Functions.

  • The constant pi is defined in LibreCAD as 3.14159265359.
  • Trigonometric functions use radians (radians = degrees*pi/180).

Multi-Command Input with Variable Files (Advanced)

Command input can be combined and entered on a single line by separating the commands and other input with semicolons. Entering li;0,0;10..0;0..10;-10..0;c;k on the command line will draw a 10 x 10 square. A list of commands, separated by a newline (return), can be copied and pasted using Paste multiple commands from the command line button (lower right corner of the Command Line dock).

 

Command input can also be loaded from text files. Enter the commands and other input into a text file, separating each with a line break. For example, create a text file and enter the following commands:

li

0,0

@10,0

@0,10

@-10,0

c

k

The above commands will draw a 10x10 square.

Using multi-command input

  1. Create a text file containing the command input.
  2. In LibreCAD, select Load Command File from the dropdown menu by selecting the Command Line button ( ).
  3. Locate the file, and select Open.

The geometry you specified in the text file will appear in the current drawing.

 

Variables in multi-command input

Multi-command input can be assigned to a variable and variables can also contain other variables, which are read recursively:

a=ci;0,0;10

b=ci;10,0;10

c=\a;\b;kill

\c

Enter each line of the text above on the command line. When \c is entered, two overlapping circles with a radius of 10 are drawn. The \ character is an escape character that allows the command line to interpret the variable name as an action. In the above example \c expands to ci;0,0;10;ci;10,0;10;kill.

Using variables with multi-command input

  1. Create a variable multi-command input text file.
  2. Load the file by selecting Options.
  3. Select Application Preferences.
  4. Select the Paths.
  5. Select Browse ( ), and locate the variable file.
  6. Select OK.
  7. Select OK again to save the Application Preferences.
  8. Restart LibreCAD.
  9. Enter \c in the Command Line.

The file will be loaded.

Command Aliases

In LibreCAD, some commands have multiple forms. The long form is the native, untranslated command, and the short form is the alias. For example, the long, native command for Shape Text is "shapetext," but the short, alias form is "st." Aliases for any command can be defined in the librecad.alias configuration file.

Format: <alias>[TAB]<command-untranslated>

 

Example: l       line

The alias file can be edited using any text editor, such as Notepad. The file can be found here: C:\Users\{UserName}\AppData\Local\LibreCAD\librecad.alias.

  • Only change the alias, not the long untranslated form.
  • You may need to restart LibreCAD for alias changes to take effect.
