Using the Library Browser

Additional part libraries can be added for blocks or drawings that you create. Any drawing listed in your Library folders are inserted as blocks, but these blocks can be exploded into individual geometry.

Specifying a new library folder

  1. Select the Options tab.
  2. Select Application Preferences.
  3. Specify a folder path in the Part Libraries field.

Do not place blocks directly in the parent parts library directory. Blocks must be placed in sub-directories to the parent libraries directory to appear in the Library browser.

  1. Select OK.
  2. Select Refresh from the Library Browser.

The folder you specified in Application Preferences should appear in the Library Browser, along with the default misc and templates folders.

Inserting a block from the Library Browser

  1. Select the block you want to insert.
  2. Select Insert.
  3. Specify a reference point in the Command Line, or select a point on the drawing.

In this mode, you can insert as many copies of the block as needed. Press ESC or right-click to exit the block insertion mode.

Removing a user-specified library folder

  1. Ensure the folder path is removed from the Part Libraries field in your Application Preferences.
  2. Select Rebuild from the Library Browser.

The folder should no longer appear in your Library Browser.
