Importing Microsoft and AutoCAD fonts to LibreCAD fonts

Many font options are available in LibreCAD, including ProNest fonts. In general, ProNest fonts are already imported in LibreCAD, but you can also import TrueType Fonts (.ttf). To use Microsoft® fonts (including TTF) or AutoCAD® fonts, you'll need to follow steps below to first convert those fonts into a readable format for LibreCAD.

If you are opening AutoCAD .dwg drawings that use AutoCAD fonts, you'll need to convert the font to ensure it comes into LibreCAD properly. The font conversion process will convert all AutoCAD fonts and only needs to be done once.

Importing fonts

ProNest fonts

LibreCAD includes ProNest fonts in the font list. These are the same fonts listed on the ProNest CAD Font Preferences page, such as the no lift font.

  • To use ProNest fonts, simply locate the font from the Font dropdown menu in the Text and MText dialog.

Microsoft fonts

Use default fonts from Microsoft, or any other TTF or OTF font file, and customize letter, word, and line spacing for these fonts using the Convert Font dialog from the Text or MText dialog.

LibreCAD will look for fonts in the default Microsoft font folder on your PC: C:/Windows/Fonts.

Importing Microsoft fonts

  1. Click either Text or MText.
  2. Click Convert Text () in the upper right-hand corner.

  3. Select a font you want to convert from the Font Family dropdown menu. Adjust Letter spacing, Word spacing, and Line Spacing, if needed.

  4. Click OK.

To use the font after it has been imported, be sure to select the converted font from the Font dropdown menu in the Text or MText dialog. Newly converted fonts will appear at the bottom of the list.

If you adjusted any spacing values, you can save the spacing in the current default font, or you can adjust the name of the file in the Save as field, allowing you to save a copy of the adjusted font without overwriting the default spacing values in the converted font.

AutoCAD fonts

LibreCAD for ProNest supports AutoCAD fonts (.shx). If your AutoCAD fonts are stored in the default font folder for AutoCAD <C:\Program Files\AutoDesk\AutoCAD 2019\Fonts>, no action is needed (note that this file path is for AutoCAD 2019 and may differ slightly depending on your version of AutoCAD).

Importing AutoCAD fonts manually

If you have AutoCAD (.shx) fonts stored outside of this location, the following needs to be done only once:

  1. Open a file explorer on your PC.
  2. Navigate to the folder C:\Program Files (x86)\Hypertherm CAM\LibreCAD and locate the file dumpshp.bat.
  3. Right-click on dumpshp.bat and click Copy.
  4. Navigate to the folder containing your AutoCAD fonts.
  5. Right-click and click Paste to put a copy of dumpshp.bat into the folder containing your AutoCAD fonts.
  6. Double-click on dumpshp.bat to run the file. A command window will appear and then automatically close when finished.

Now, any AutoCAD fonts used in your drawings will automatically convert into a readable format for LibreCAD, and you can use any AutoCAD font in a new LibreCAD drawing.
