Setting up a drawing

When you first get started with LibreCAD, you'll want to make sure your drawing is configured with the correct units, scale, and so forth. This only needs to be done once, and any changes you make will save to the default template that LibreCAD uses when you create a new drawing.

Scale

For best results, drawings should be created full-scale (1:1). The zooming abilities of LibreCAD will make the whole drawing fit in the display window or zoom into the fine detail.

Printing

When producing output, the drawing will need to be scaled to fit the page. Output can be a printed page, a PDF, or another image format.

Determining dimension text size is addressed by the General Scale on the Dimensions tab of the Current Drawing Preferences. The General Scale is used to adjust the dimension text, arrows and related parameters to the sizes suitable for the required page format.

The value for the General Scale should be the inverse of the scale used for printing. For example, if the print scale is 1:10”, the General Scale should be 10 (10:1). Setting the General Scale to the inverse of the print scale results in the dimension text being the defined size on the printed drawing. The drawing is scaled down to fit the page and the dimension text is scaled up to be legible. The print scale can be determined by using LibreCAD’s print preview feature. See the Printing Guide for more information.

Use the following parameters when setting the scale values:

  • Options > Current Drawing Preferences > Paper > Format: Selecting a paper size and orientation will determine the print scale used for final output. The print scale is used to determine the General Scale.
  • Current Drawing Preferences > Dimensions > General Scale: Setting the General Scale will adjust the dimension text and related parameters.
  • Setting the paper size after drawing the object but prior to dimensioning a drawing will help determine the print scale, the General Scale, and subsequently the appropriate line spacing for dimensions.
  • Be sure to allow room on the drawing for dimension lines and text when determining the print scale.
  • While any scale can be used when printing a drawing there are commonly used scales for different types of drawings. Refer to Drawing Scales in the appendix for some examples.

Layers

ProNest automatically adds layers to LibreCAD that are in your Importing Parts settings under the CAD Import tab. You can still modify or create new layers, but keep in mind that, when importing a part from LibreCAD to ProNest, ProNest will disregard any layers that aren't listed in your Importing Parts settings (ProNest Settings > Importing Parts > CAD Import).

Layers help organize drawings by allowing users to place and manage related entities. Layers can be thought of as transparent drawing sheets that can be placed on top of one another. Layers have the added advantage in that the pen attributes can be modified for all entities on the layer. Layers are added, removed, hidden and modified using the Layer List Dock.

Hiding layers while drawing reduces the visual complexity and makes it easier to focus on the current drawing efforts.

Layers can be locked, which prevents accidental changes and can improve the performance when working with very large complex drawings.

Construction Layers

A Construction Layer ( ) is a special layer used to create reference geometry to help align other drawing entities. A construction layer:

  • contains lines that are an infinite length,
  • won’t appear on printed drawings, and
  • won't appear in ProNest when you import a drawing.

Read more about using layers.

Templates

Templates are prototype drawings that provide the means to save basic parameters and settings so a drawing does not have to be configured each time a new one is started. The parameters and settings include the settings defined in the Drawing Preferences, such as the paper format, main unit of measure and format, and dimension format. Templates can also include layers and layer configuration, line type and thickness, pen color, and other drawing elements such as a border.

These settings are inherited by the drawings created from the template. Once the template has been prepared, it can be saved to any location where you have read/write permissions.

LibreCAD supports the use of multiple templates. A LibreCAD user that plans on creating similar drawings may require only one or two templates. A user that plans on several different types of drawings may want multiple templates. For example, templates can be set up for each paper size available or for each paper orientation.

Read more about using templates.

Default templates

When LibreCAD is first launched it creates a new drawing using a default template and some information from ProNest. Any new drawing created in LibreCAD will use your template, which updates each time you change any of your preferences from the Options menu (Application Preferences, Current Drawing Preferences, Widget Options, and Device Options).

Any layers added to your drawing template will be disregarded if the layers aren't mapped in your ProNest Importing Parts settings.

When installing LibreCAD, a resource directory is created and includes a default template named empty.dxf found in C:\Program Files (x86)\Hypertherm CAM\LibreCAD\resources\library\templates for 64-bit operating systems, and C:\Program Files\Hypertherm CAM\LibreCAD\resources\library\templates for 32-bit operating systems..

As an alternative to the LibreCAD provided template, a user-specified template can be configured in Application Preferences > Paths. The specified template is used instead of the default LibreCAD template when the application is launched and for new drawings.
