Modifying Properties of an entity

After you've placed an entity, such as lines, arcs, text, or dimensions, you can modify attributes as well as properties that are specific to the type of entity you are modifying, such as geometry coordinates, layers, text styles, or label information. The Properties dialog varies by entity type. Individual segments of a polyline are modified individually. In general, the Properties tool gives more control over entity geometry whereas the Attributes tool is primarily used for pen and layer information.

Modifying Properties of an entity

  1. Select Properties( ) from the Modify menu.
  2. Select the entity for which you want to modify properties.
  3. Make changes, and select OK to apply your changes, or select Cancel to discard changes.
