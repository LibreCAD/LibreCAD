Modifying Properties of an entity

After you've placed an entity, such as lines, arcs, text, dimensions, you can modify attributes as well as properties that are specific to the type of entity you are modifying, such as geometry coordinates, text styles, or label information. The Properties dialog varies by entity type. Individual segments of a polyline are modified individually.

Modifying Properties of an entity

  1. Select the Properties button ( ) from the Modify menu.
  2. Select the entity for which you want to modify properties.
  3. Make changes, and select OK to apply your changes, or select Cancel to discard changes.
