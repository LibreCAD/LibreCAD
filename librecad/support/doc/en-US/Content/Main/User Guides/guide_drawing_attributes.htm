Modifying Attributes of an entity

After you've placed an entity, such as lines, arcs, text, or dimensions, you can modify some aspects of the entity using the Attributes button from the Modify menu.

  • Layer – Change the layer that the entity is on.
  • Pen color – Change the color of the entity.
  • Pen width – Change the width of the entity.
  • Pen line type – Change the solidity of line that draws the entity.
  • Apply attributes Block-deep – If you're modifying an entity that is part of a block, select to apply any of the above changes to the entire block.

Use the Properties tool to modify other information about an entity, such as geometric coordinates, angles, and so forth.

Modifying Attributes of an entity

  1. Click Attributes from the Modify menu.
  2. Click the entity for which you want to modify attributes.
  3. Make changes, and click OK.
