Using Dimensions

LibreCAD is equipped with several dimensioning tools to help you dimension lines, circles, arcs, and angles.

Configuring Dimension preferences

Before you begin adding dimensions to your drawing, ensure that your dimension preferences are configured.

  1. Select Options.
  2. Select Current Drawing Preferences.
  3. Select the Dimensions tab.
  4. Ensure all properties listed on this page are appropriate for your drawing.
  5. Select OK to save your changes for this drawing only, or select Update Template to use these preferences for future drawings.

Ensuring dimensions are visible in ProNest

You can configure your layers and ProNest Settings to ensure that dimension information is visible on display-only layers in ProNest.

  1. In ProNest, ensure that you have a display-only layer mapped in your Importing Parts Settings (Settings > Importing Parts > CAD Import tab > Display Only).
  2. In LibreCAD, ensure that the Display Only layer is mapped in your drawing Layers. It may be helpful to change the color of this layer so a process layer isn't confused with the display only layer.
  3. Draw dimensions on the Display Only layer you've added in LibreCAD.
  4. Import the part into ProNest.

Display-only information appears for a part on the nest but isn't visible from the Part List. Additionally, you can view the Part Report for an individual part (right-click on a part from the Part List, and select Part Report), which shows display-only information.

Dimensioning horizontal lines (180°)

  • Use the Horizontal dimension tool ( ), and select start and end points for the entity you are dimensioning.

Enable Snap on Entity ( ) or Snap on Endpoints ( ) to ensure your linear dimensions are accurate.

Dimensioning vertical lines (90°)

  • Use the Vertical dimension tool ( ), and select start and end points for the entity you are dimensioning.

Dimensioning angled lines

  • Use the Aligned dimension tool ( ), and select start and end points for the entity you are dimensioning.

Dimensioning arcs and circles

  • Use the Radial dimension tool ( ) or the Diametric dimension tool ( ), and select the circle or arc entity you are dimensioning.

Dimensioning angles

  • Use the Angular dimension tool ( ), and select the two line between which you are dimensioning the angle.

Adding individual leader lines

Dimensioning tools automatically insert leaders based on your Current Drawing Preferences. If needed, you can add additional leader lines using the Leader dimension tool ( ).
