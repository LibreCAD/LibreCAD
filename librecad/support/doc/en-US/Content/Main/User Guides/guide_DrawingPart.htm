Drawing a part walkthrough

Like with any CAD program, LibreCAD offers various ways to approach drawing a part. This article covers a few different walkthrough examples on how to draw a part.

  • Check the tool status bar above the drawing area and the Command window, in case you are prompted to set parameters.
  • You can use construction layers to create display only guide lines.
  • You can use relative coordinates instead of absolute coordinates in the Command Line.

Using the Command Line

You can draw a part using only the command line.

Example 1: Notched rectangle using command prompts

  1. Enter rect into the Command Line (this is the shortcut for Rectangle).
  2. Enter coordinates for the first corner: 0,0.
  3. Enter coordinates for the second corner: 6,3.

  4. Enter cd into the Command Line (this is the shortcut for the Center, Diameter circle tool).
  5. In the tool status bar, set the Diameter to 1.
  6. Enter coordinates for the first circle center point: 0,1.5.
  7. Since we are still in Center, Diameter circle mode, continue entering center points to draw circles on each side of the rectangle.
    • 3,3
    • 6,1.5
    • 3,0

  8. Next, clean up unwanted geometry using the Trim Excess tool. Enter tx into the Command Line.
  9. Click on the geometry you want to remove, which should be the halves of circles that are outside of the rectangle and the line segments that intersect the circles.

The drawing is complete.

Using buttons and the drawing area

Instead of entering commands in the Command Line, click on any of the drawing tools to begin drawing.

The Command window will prompt you to enter coordinates, which you can enter by clicking in the drawing area to place a shape or point, or you can enter coordinates in the Command Line.

Example 2: Notched rectangle using buttons and the drawing area

  1. Click the Rectangle ( ) button, which is docked on the custom Hypertherm toolbar just above the drawing area.
    • The Rectangle button is also available in Tools > Line > Rectangle.
  2. Click the 0,0 location in the drawing area to place the first corner.

    Tip: Turn on Snap on Grid for precise placement.

  3. Click the 6,3 location in the drawing area to place the second corner.

  4. Click the Center, Diameter ( ) circle button, which is also docked in the Hypertherm toolbar.
    • The Center, Diameter button available in Tools > Circle > Center, Diameter.
  5. In the tool status bar, set the Diameter to 1. The diameter will already be 1 if you changed in Example 1.
  6. Turn off all snapping options in the snapping toolbar below the drawing area. Turn on the Snap to Middle option, which will allow us to snap the center of the circle to the middle of each line segment.
  7. Click near the center of each line segment of the rectangle to place a circle.

  8. Click the Trim Excess ( ) button, which is docked on the custom Hypertherm toolbar just above the drawing area. Click the extruding half-circles and line segments that go through each circle, to remove them from the drawing.
    • The Trim Excess button is also available in Tools > Modify > Trim Excess.

The drawing is complete.

Using the Offset tool

The Offset tool in LibreCAD allows you to create drawings without needing to know coordinates. This method may be helpful if you are starting the drawing with dimension information but not necessarily coordinate information. This approach uses both the Command Line and the drawing area.

Example 3: Slotted rectangle using the Offset tool

  1. Turn on Snap on grid ( ). All other grid options can be disabled.
  2. Use the Two Points line tool ( ) to draw two lines perpendicular to each other:

  3. Turn off Snap on grid.
  4. Use the Offset tool by entering offset in the Command Line.
    1. In the Offset tool status bar, set the Distance to 10 and Number to 1.
    2. Then, click near the bottom line so that an offset line is drawn above it, and click near the left line so that an offset line is drawn on the right side. This will result in a square drawing.

    3. While still in Offset mode, change the Distance to 5 to draw a horizontal and vertical center guide line for the interior cutout.

    4. Change the Distance to 2.5 to draw one additional horizontal line above and below the middle horizontal line.
    5. Change the Distance to 1.5 to draw one additional vertical line to the left and to the right of the middle vertical line. The lines from steps 4 and 5 are guides for drawing slots.

    6. Turn on Snap Intersection ( ).
    7. Click the 2 Points circle ( ) button.
      1. Click the intersection of the horizontal line that is second from the top and the second vertical line from the right, to draw the first circle.
      2. Click the intersection of the horizontal line that is fourth from the top and the second vertical line from the right, to draw the second circle.

    8. Next, start to clean up unwanted geometry using the Trim Excess tool. Enter tx into the Command Line, then click geometry you want to remove.

    9. Continue using Trim Excess to clean up the remainder of guide lines to finish the part.

The drawing is complete.
