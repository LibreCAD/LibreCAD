Multi-Command Input with Variable Files (Advanced)

Command input can be combined and entered on a single line by separating the commands and other input with semicolons. Entering li;0,0;10..0;0..10;-10..0;c;k on the command line will draw a 10 x 10 square. A list of commands, separated by a newline (return), can be copied and pasted using Paste multiple commands from the command line button (lower right corner of the Command Line dock).

Command input can also be loaded from text files. Enter the commands and other input into a text file, separating each with a line break. For example, create a text file and enter the following commands:

li

0,0

@10,0

@0,10

@-10,0

c

k

The above commands will draw a 10x10 square.

Using multi-command input

  1. Create a text file containing the command input.
  2. In LibreCAD, click Load Command File from the dropdown menu by selecting the Command Line button ( ).
  3. Locate and click the file, and click Open.

The geometry you specified in the text file will appear in the current drawing.

Variables in multi-command input

Multi-command input can be assigned to a variable and variables can also contain other variables, which are read recursively:

a=ci;0,0;10

b=ci;10,0;10

c=\a;\b;kill

\c

Enter each line of the text above on the command line. When \c is entered, two overlapping circles with a radius of 10 are drawn. The \ character is an escape character that allows the command line to interpret the variable name as an action. In the above example \c expands to ci;0,0;10;ci;10,0;10;kill.

Using variables with multi-command input

  1. Create a variable multi-command input text file.
  2. Load the file by selecting Options.
  3. Click Application Preferences.
  4. Click the Paths.
  5. Click Browse ( ), and locate the variable file.
  6. Click OK.
  7. Click OK again to save the Application Preferences.
  8. Restart LibreCAD.
  9. Enter \c in the Command Line.

The file will be loaded.
