Using text and fonts

Text can be integrated into artistic drawings and signs, and it can be used to provide information and notes, including title blocks. Create single- or multi-line text using the Text or MText tools.

  • For best results, when using scribe text, it is recommended to explode text into geometry (using the explode command) before sending a drawing to ProNest. Use an alignment of Baseline Left ( ) for Text, or Bottom Left for MText to help prevent any geometry issues when sending a drawing to ProNest.
  • Note that text will always be converted to geometry in ProNest unless it is on a scribe layer, but results will be better if text is exploded in LibreCAD prior to exporting to ProNest.
  • Ensure Smoothing is enabled in ProNest to convert choppy line segments to arcs and smoother profiles before importing a part from LibreCAD (ProNest > Settings > Importing Parts > enable Smooth entities, and try Smooth tolerances between 0.001 and 0.005 to start).

Creating single-line text

  1. Select Text () from the Tools menu.
  2. In the Text field, type the text that you want to appear in the drawing.
  3. Adjust Font Settings, Alignment, and Symbols, and convert Microsoft fonts, if necessary.
  4. Select OK.
  5. Select on the drawing to place text, or enter placement coordinates into the Command line (the text Alignment is used as the insertion point when specifying coordinates).

Creating multi-line text

  1. Select MText () from the Tools menu.
  2. In the Text field, type the text that you want to appear in the drawing. Use the Insert Symbol dropdown menu to insert special characters into your text, such as degree symbols and copyright symbols.
  3. Convert Microsoft fonts and set spacing values, if necessary.
  4. Set the Alignment and insertion Angle (the Angle may be left blank to insert text at 180°).
  5. Select OK.
  6. Select on the drawing to place text, or enter placement coordinates into the Command line (the text Alignment is used as the insertion point when specifying coordinates).

Modifying text

  1. Select Properties ( ) from the Modify tools menu. The text dialog will appear again, allowing you to modify the text as well as layer and color information.
  2. Select the text you want to modify.
  3. Modify any relevant properties.
  4. Select OK to save your changes and update text, or Cancel to discard changes.

Shaping text around an entity

With the Hypertherm-exclusive Shape Text feature in LibreCAD, you can shape text around a single line, arc, or circle entity.

  1. Draw a shape.
  2. Place text using the Text or MText tools. When in the Text or MText dialog, be sure to specify an Alignment. If you are unsure of what Alignment to use, try using the lower left option.
  3. Enter Shape Text mode by selecting Shape Text ( ) from the Modify tools, or by entering st into the Command Line.
  4. Select the text that you want to shape.
  5. Move your cursor around a shape. Modify the Offset value and change the text direction by selecting the checkbox, if needed. When the text is in the desired location, select to place the text.

To achieve fine-grain control over text that you are shaping, it is recommended to explode text strings into individual letters using the Explode Text into Letters ( ) modify tool.

Aligning shaped text through the entity perimeter

  1. Draw a shape.
  2. Place text using the Text or MText tools. When in the Text or MText dialog, be sure to specify a middle center Alignment.
  3. Enter Shape Text mode by selecting Shape Text ( ) from the Modify tools, or by entering st into the Command Line.
  4. Adjust the Shape Text mode Offset on the toolbar.
    •  For single-line text, the Offset should be the negative value of half of the text Height. For example, when using a text height of 1, the offset value should be -0.5.
    • The Offset can be adjusted and updated by entering a value and pressing the ENTER key.
  5. Select the text that you want to shape.
  6. Move your cursor around a shape. When the text is in the desired location, select to place the text.

 

Use the Trim Excess modify tool ( ) to quickly remove any unwanted geometry in one click.

Fonts

Many font options are available in LibreCAD, including ProNest fonts. To use Microsoft® fonts or AutoCAD® fonts, you'll need to follow steps below to first convert those fonts into a readable format for LibreCAD.

If you are opening AutoCAD .dwg drawings that use AutoCAD fonts, you'll need to convert the font to ensure it comes into LibreCAD properly. The font conversion process will convert all AutoCAD fonts and only needs to be done once.

Converting fonts

ProNest fonts

LibreCAD includes ProNest fonts in the font list. These are the same fonts listed on the ProNest CAD Font Preferences page, such as the no lift font.

  • To use ProNest fonts, simply locate the font from the Font dropdown menu in the Text and MText dialog.
Microsoft fonts

Use default fonts from Microsoft and customize letter, word, and line spacing for these fonts using the Convert Font dialog from the Text and MText dialog.

Converting Microsoft fonts

  1. Select either Text or MText.
  2. Select Convert Text () in the upper right-hand corner.
  3. Select a font you want to convert from the dropdown menu.
  4. Adjust Letter, Word, and Line Spacing, if needed.
    1. If you adjusted spacing, you can save the spacing in the current default font, or you can adjust the name of the file in the Save as field, allowing you to save a copy of the adjusted font without overwriting the default converted font.
  5. Select OK.

Be sure to select the converted font from the Font dropdown menu in the Text or MText dialog to use the font after it has been converted.

AutoCAD fonts

Converting AutoCAD fonts

If you'll be using LibreCAD to open/modify AutoCAD drawings (.dwg) that have text, you'll need to complete the following steps to ensure that LibreCAD can present the text appropriately. These steps only need to be completed once and will have no impact on any existing fonts or AutoCAD.

  1. Open LibreCAD.
  2. Select Options.
  3. Select Application Preferences.
  4. On the Paths tab, in the Fonts field, specify a new folder on your PC that will be used to store converted fonts. This folder must be different than where your LibreCAD fonts are stored.
  5. Open a file explorer on your PC.
  6. Navigate to the following file path:
  • 64-bit OS: C:\Program Files (x86)\Hypertherm CAM\LibreCAD
  • 32-bit OS: C:\Program Files\Hypertherm CAM\LibreCAD
  1. Locate the file dumpshp.bat. Right-click and select Copy.
  2. Navigate to the new font folder you specified in step 4.
  3. Right-click and select Paste to put a copy of dumpshp.bat into your new font folder.
  4. Double-click on dumpshp.bat. A command window will appear.
  5. When prompted, press any key.

Now, any AutoCAD fonts used in your drawings will automatically convert into a readable format for LibreCAD, and you can use any AutoCAD font in a new LibreCAD drawing.

Creating a font

With LibreCAD, you can easily create a new font by using an existing font as a template. In LibreCAD, each font character is a block, and each block is named after the unicode character code.

Creating a font from a template

Step 1

Identify an existing font to use as a template for your new font. Try to find a font that looks similar to what you want to create, or choose a standard font such as Times New Roman.

If you are unsure of what font to use, you can preview fonts in the Convert Font dialog from Tools > MText > Convert Font ( ).

 

Step 2

Locate the font using the File Explorer on your PC.

 

If you are using a font that you converted from AutoCAD or Microsoft, the font file will be located in the path you specified in Application Preferences > Paths > Fonts.

 

If you are using a LibreCAD font, the font will be located in this folder: C:\Program Files (x86)\Hypertherm CAM\LibreCAD\resources\fonts

 

Step 3

Open the font file (*.LFF) in LibreCAD by selecting File > Open, and set the file type to LFF Font (*.lff). Then, navigate to the appropriate folder, and select the font file you wish to open.

 

Step 4

The font file is now opened in LibreCAD. Notice that each character is listed in a row along with the corresponding unicode below each character. Additionally, notice that the Block List lists out each character, by unicode name, as an individual block.

 

To edit a character:

  1. Locate the block that corresponds with that character in the Block List.
  2. Select the block in the Block List.
  3. Select Edit the active block in a separate window( ).

The active block will open in a separate window.

 

Step 5

Make changes to the character, as needed. When you have finished editing a character, exit the tab, which will return you to the full font LFF file. There is no need to save the block to an individual file; instead, the block is stored in the LFF file.

 

Repeat this process until all characters are modified, as needed.

 

Step 6

It is recommended that you save the modified font to a different file, so that you can preserve the state of the original font.

 

To save the font as a different file:

  1. Select File.
  2. Select Save As.
  3. Locate the folder to which you want to save the font file. It is recommended that you use the following: C:\Program Files (x86)\Hypertherm CAM\LibreCAD\resources\fonts.
  4. From the file type dropdown, select LFF File (*.lff).
  5. Provide a name for the file, but do not include a file extension in the name. For example, type "MyFont" instead of "MyFont.lff".
  6. Select Save.

If you are saving the font to the LibreCAD font folder from Step 6.3, you may need to modify the folder permissions so that you are able to save to that folder. if this can't be done, save the font file to the folder you have specified in Application Preferences > Paths > Fonts.
