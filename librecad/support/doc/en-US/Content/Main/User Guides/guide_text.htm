Using Text and MText

Text can be integrated into artistic drawings and signs, and it can be used to provide information and notes, including title blocks. Create single- or multi-line text using the Text or MText tools.

Creating single-line text

  1. Select Text () from the Tools menu.
  2. In the Text field, type the text that you want to appear in the drawing.
  3. Adjust Font Settings, Alignment, and Symbols, and convert Microsoft fonts, if necessary.
  4. Select OK.
  5. Select on the drawing to place text, or enter placement coordinates into the Command line (the text Alignment is used as the insertion point when specifying coordinates).

Creating multi-line text

  1. Select MText () from the Tools menu.
  2. In the Text field, type the text that you want to appear in the drawing. Use the Insert Symbol dropdown menu to insert special characters into your text, such as degree symbols and copyright symbols.
  3. Convert Microsoft fonts and set spacing values, if necessary.
  4. Set the Alignment and insertion Angle (the Angle may be left blank to insert text at 180°).
  5. Select OK.
  6. Select on the drawing to place text, or enter placement coordinates into the Command line (the text Alignment is used as the insertion point when specifying coordinates).

Modifying text

  1. Select the Properties button ( ) from the Modify tools menu. The text dialog will appear again, allowing you to modify the text as well as layer and color information.
  2. Modify any relevant properties.
  3. Select OK to save your changes and update text, or Cancel to discard changes.

Shaping text around an entity

With the Hypertherm-exclusive Shape Text feature in LibreCAD, you can shape text around any line, polyline, arc, or circle entity.

  1. Draw a shape.
  2. Place text using the Text or MText tools. When in the Text or MText dialog, be sure to specify an Alignment. If you are unsure of what Alignment to use, try using the lower left option.
  3. Enter Shape Text mode by selecting the Shape Text button ( ) from the Modify tools, or by entering st into the Command Line.
  4. Select the text that you want to shape.
  5. Move your cursor around a shape. When the text is in the desired location, select to place the text.

Text can't be shaped around a spline.

Aligning shaped text through the entity perimeter

  1. Draw a shape.
  2. Place text using the Text or MText tools. When in the Text or MText dialog, be sure to specify a middle center Alignment.
  3. Enter Shape Text mode by selecting the Shape Text button ( ) from the Modify tools, or by entering st into the Command Line.
  4. Adjust the Shape Text mode Offset on the toolbar.
    •  For single-line text, the Offset should be the negative value of half of the text Height. For example, when using a text height of 1, the offset value should be -0.5.
    • The Offset can be adjusted and updated by entering a value and pressing the ENTER key.
  5. Select the text that you want to shape.
  6. Move your cursor around a shape. When the text is in the desired location, select to place the text.

 

Use the Trim Excess modify tool ( ) to easily remove any unwanted geometry in one click.

Fonts

Many font options are available in LibreCAD, including ProNest fonts. To use Microsoft® fonts or AutoCAD® fonts, you'll need to follow steps below to first convert those fonts into a readable format for LibreCAD.

If you are opening AutoCAD .dwg drawings that use AutoCAD fonts, you'll need to convert the font to ensure it comes into LibreCAD properly. The font conversion process will convert all AutoCAD fonts and only needs to be done once.

ProNest fonts

LibreCAD includes ProNest fonts in the font list. These are the same fonts listed on the ProNest CAD Font Preferences page, such as the no lift font.

  • To use ProNest fonts, simply locate the font from the Font dropdown menu in the Text and MText dialog.

Microsoft fonts

Use default fonts from Microsoft and customize letter, word, and line spacing for these fonts using the Convert Font dialog from the Text and MText dialog.

Converting Microsoft fonts

  1. Select either Text or MText.
  2. Select the Convert Text button () in the upper right-hand corner.
  3. Select a font you want to convert from the dropdown menu.
  4. Adjust Letter, Word, and Line Spacing, if needed.
    1. If you adjusted spacing, you can save the spacing in the current default font, or you can adjust the name of the file in the Save as field, allowing you to save a copy of the adjusted font without overwriting the default converted font.
  5. Select OK.

Be sure to select the converted font from the Font dropdown menu in the Text or MText dialog to use the font after it has been converted.

AutoCAD fonts

Converting AutoCAD fonts

If you'll be using LibreCAD to open/modify AutoCAD drawings (.dwg) that have text, you'll need to complete the following steps to ensure that LibreCAD can present the text appropriately. These steps only need to be completed once and will have no impact on any existing fonts or AutoCAD.

  1. Open LibreCAD.
  2. Select Options.
  3. Select Application Preferences.
  4. On the Paths tab, in the Fonts field, specify a new folder on your PC that will be used to store converted fonts. This folder must be different than where your LibreCAD fonts are stored.
  5. Open a file explorer on your PC.
  6. Navigate to the following file path: C:\Program Files (x86)\Hypertherm CAM\LibreCAD
  7. Locate the file dumpshp.bat. Right-click and select Copy.
  8. Navigate to the new font folder you specified in step 3.
  9. Right-click and select Paste to put a copy of dumpshp.bat into your new font folder.
  10. Double-click on dumpshp.bat. A command window will appear.
  11. When prompted, press any key.

Now, any AutoCAD fonts used in your drawings will automatically convert into a readable format for LibreCAD, and you can use any AutoCAD font in a new LibreCAD drawing.
