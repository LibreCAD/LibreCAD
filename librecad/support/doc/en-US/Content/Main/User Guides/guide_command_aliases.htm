Command Line Aliases

In LibreCAD, some commands have multiple forms. The long form is the native, untranslated command, and the short form is the alias. For example, the long, native command for Shape Text is "shapetext," but the short, alias form is "st." Aliases for any command can be defined in the librecad.alias configuration file.

Format: <alias>[TAB]<command-untranslated>

Example: l       line

The alias file can be edited using any text editor, such as Notepad. The file can be found here: C:\Users\{UserName}\AppData\Local\LibreCAD\librecad.alias.

  • Only change the alias, not the long untranslated form.
  • You may need to restart LibreCAD for alias changes to take effect.
