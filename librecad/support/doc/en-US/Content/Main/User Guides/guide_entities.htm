About Entities

Create line, arc, text, point, and hatch entities using Tools:

  • Lines
  • Circles
  • Curves
  • Ellipses
  • Polylines
  • Dimensions
  • Single-line text
  • Multi-line text
  • Points
  • Hatch patterns

Some tools may require you to specify parameters, such as the Center, Diameter tool, which asks for a diameter input, while other tools allow you to draw freely, such as the Rectangle tool. Typically, you can specify coordinates for an entity, or you can point-and-click to draw entities.

 

See the section on Tools for reference information for each tool.

 

Circles

Modifying the circle size

  1. Select the circle entity.
  2. Select and hold a blue handle in any quadrant, then drag and drop to resize the circle.

- or -

  1. Select Properties ( ), or enter prop in the Command Line.
  2. Select the circle entity.
  3. Modify the Center (x), Center (y), and Radius values, as needed.
  4. Select OK.

- or -

  • Use the Modify - Scale tool.

Moving circles

  1. Select the circle entity.
  2. Select and hold the blue handle in the center of the circle, then drag and drop to place the circle in a new location.

- or -

  • Use the Modify - Move/Copy tool.

Curves

Modifying curves

  1. Select the curve entity.
  2. Select and hold a blue handle at either end, then drag and drop to place an end of the curve.

- or -

  1. Select Properties ( ), or enter prop in the Command Line.
  2. Select the curve entity.
  3. Modify the Center (x), Center (y), Radius, Start Angle, and End Angle values, as needed. Optionally, select the Reversed option to change the direction of the curve.
  4. Select OK.

- or -

  • Use the Modify - Scale tool.

Moving curves

  1. Select the curve entity.
  2. Select and hold the blue handle in the center of the curve, then drag and drop to place the curve in a new location.

- or -

  • Use the Modify - Move/Copy tool.

Ellipses

Handles on an ellipse

A. Center point

B. Handle to resize ellipse length (major axis), width (minor axis), and orientation

C. Handle to draw only part of the ellipse as an arc

D. Handle to resize the width of the ellipse (minor axis)

E. Handle to resize ellipse length, width, and orientation

 

Reshaping ellipses

  1. Select the ellipse entity.
  2. Select and hold either handle B or E, then drag and drop to reshape the ellipse.

Creating partial ellipses

  1. Select the ellipse entity.
  2. Select and hold handle C, then drag around the ellipse until the preview displays the correct arc you want to draw.
  3. Release the mouse hold and click away to complete the arc drawing.

Scaling ellipses

  • Use the Modify - Scale tool.

Moving ellipses

  1. Select the ellipse entity.
  2. Select the center point at handle A, then drag and drop to place the ellipse in a new location.

- or -

  • Use the Modify - Move/Copy tool.

Lines

Modifying line length or orientation

  1. Select the line entity.
  2. Select and hold a blue handle at either end, then drag and drop to place the end of the line.

All lines in a polygon are considered exploded. To modify the size of an entire polygon, use Modify - Scale; to modify the location of an entire polygon, use Modify - Move/Copy or CTRL+C and CTRL+V to copy/paste. Read more about Modify tools here.

Moving lines

  1. Select the line entity.
  2. Select and hold near the middle of the line, then drag and drop to place the line in a new location.

- or -

  • Use the Modify - Move/Copy tool.

Polylines

Modifying polylines

  1. Select the polyline entity.
  2. Select and hold a blue handle, then drag and drop to place the end of the line.

All lines in a polyline are modified individually. To modify the size of an entire polygon, use Modify - Scale.

Moving polylines

  1. Select the line entity.
  2. Select and hold anywhere on the polyline except the blue handles, then drag and drop to place the polyline in a new location.

- or -

  • Use the Modify - Move/Copy tool.

Points

Moving points

  • Select and hold the point, then drag and drop to place the point in a new location.

- or -

  1. Select Properties ( ), or enter prop in the Command Line.
  2. Select the point.
  3. Enter coordinates in the Geometry - Position (x) and Position (y) values to place the point in that location.

Hatch

Add hatching to an area

  1. Select at least one entity. For best results, select an enclosed shape, such as a circle or polygon.
  2. Select Hatch ( ) from the Tools menu, or enter hatch into the Command Line.
  3. Select a hatch pattern from the dropdown menu, and enter a Scale. Optionally, enter an Angle at which to place the hatch pattern.
  4. Select OK.

Modifying a hatch pattern

  1. Select Properties ( ), or enter prop in the Command Line.
  2. Select the hatch pattern in the drawing.
  3. Modify attributes of the hatch pattern.
  4. Select OK when you have finished making changes.

Removing a hatch pattern

  1. Select the hatch pattern.
  2. Press the DELETE key on your keyboard.
