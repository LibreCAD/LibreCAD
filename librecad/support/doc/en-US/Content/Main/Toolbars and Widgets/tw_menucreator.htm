Menu creator widget

LibreCAD allows you to create custom menus for quick access to your favorite tools using a mouse/keyboard click assignment.

 

Create

Select to create a new custom menu.

Destroy

Select to delete a custom menu.

Assign

Select to assign a mouse/keyboard shortcut to a custom menu.

Update

Select to save changes to an existing custom menu.

Create a new custom menu

  1. Open Widgets tab > Menu Creator. The Menu Creator window will appear.
  2. In the left pane, select any tool then select the right-pointing chevron (>) to add it to your menu list in the right pane. Selecting a category from the left dropdown menu will filter the tools that appear on the left pane.
    • Drag and drop items to re-order them in your new menu in the right-hand pane.
    • Remove items from your new menu by selecting the tool item then selecting the left-pointing chevron (<).
  3. Once you've added all desired tools, be sure to name your new menu in the Name field.
  4. Select Create.
  5. Select Assign. The Menu Assignment window will appear.

  1. Select an option for the menu assignment.
  2. Select Save.

When you return to the main drawing screen, you can use the mouse/keystroke menu assignment to make the menu appear. For example, if your menu assignment is set to Double-Click, you can double-click anywhere in the drawing, and your new menu will appear.

Modify an existing custom menu

  1. Open Widgets tab > Menu Creator. The Menu Creator window will appear.
  2. Select from the dropdown menu on the right-hand side the custom menu that you want to modify.
  3. Make any changes to the items on your menu, their order in the menu, or the shortcut assignment.
  4. Select Update when you have finished making changes, or exit from the window if you wish to discard your changes.

Delete a custom menu

  1. Open Widgets tab > Menu Creator. The Menu Creator window will appear.
  2. Select from the dropdown menu on the right-hand side the custom menu that you want to delete.
  3. Select Destroy.
