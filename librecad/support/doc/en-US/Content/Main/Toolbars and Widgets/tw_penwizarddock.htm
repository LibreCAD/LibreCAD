Pen Wizard dock

The Pen Wizard allows users to create a palette of favorite colors for the drawing tools.

 

Using a Pen Wizard color

  • Ensure that the color is listed in your Favorites list, then double-click on the color to make it active.

Anything you draw will now use the color you have selected.

Adding a color to the Pen Wizard favorites list

  1. Select a color from the dropdown, or select the custom color button ().
  2. Select the Add to favorites button ().

The color now appears in the Pen Wizard Favorites list.

Creating a custom color

  1. Select the custom color button ().
  2. Create a custom color by either entering Hue/Sat/Val or Red/Green/Blue values, entering an HTML color code, or by dragging the crosshair around the color palette and adjusting the light to dark scale.
  3. Select OK to create only one custom color, or select Add to Custom Colors to create multiple custom colors in a row.
  4. Select OK to save your changes, or select Cancel to discard changes.

The new custom color you have created appears in the Pen Wizard dropdown list.

Removing a color from the Favorites list

  • Right-click on the color you wish to remove, and select Remove.

Reorder colors in the Favorites list

  • Drag and drop colors in the list to reorder them.
