Block List dock widget

The Block List Dock provides the functions to manage a list of blocks that exist in the drawing file.

 

The lower portion of the dock shows a list of blocks in the current drawing. The blocks in the example above are named "MyBlock1," "MyBlock2," and "MyBlock3."

Block functions

Some of these items are also available by right-clicking on a layer.

Tool Icon Description
Show/defreeze all blocks Makes all the blocks in the current drawing visible.
Hide/freeze all blocks Hides all blocks in the active drawing.
Create a block Creates a block from the selected items.
Add an empty block Creates an empty block that can then be edited is a separate drawing window/tab.
Remove the active block Deletes the highlighted block.
Rename the active block Rename the highlighted block.
Edit the active block in a separate window Open a new drawing window to edit a new or existing block.
Save the active block to a file Saves the highlighted block to a file.
Insert the active block Inserts the highlighted block in the current drawing at the specified reference point.
