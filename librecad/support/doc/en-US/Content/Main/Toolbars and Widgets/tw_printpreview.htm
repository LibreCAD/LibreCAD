Print Preview toolbar

The Print Preview toolbar is visible when in Print Preview mode (File > Print Preview, or ). It used to set up the print output as desired regardless of output format (PDF or paper). The print preview can be configured after selecting File > Print Preview through the menu. A combination of scale value, color status and drawing position relative to paper allows custom print output.

Item Icon Description
Scale [dropdown menu]

Displays the scale of the drawing based on its size and the paper size selected in Drawing Preferences. It is also possible to adjust the scale to a specific value in the dropdown list or enter a custom value separated by the colon (:).

 

Selecting the Fixed checkbox locks the scale to the set value.
Toggle Black/White Toggles the colors of all entities from color to black/white.
Center to page Centers the drawing to the paper size selected in Drawing Preferences.
Fit to page Fits the drawing to the paper size selected.
Multipages Determines the number of pages needed to print the drawing based on the defined scale and the paper size selected in Drawing Preferences.
