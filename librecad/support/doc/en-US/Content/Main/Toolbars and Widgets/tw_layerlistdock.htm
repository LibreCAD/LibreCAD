Layer List dock widget

The Layer List dock widget provides a list of layers and in the current drawing and functions to manage layers. The entry line on top of the dock allows you to filter by layer names (for example,“*01” would show all names ending by “01”).

Layer List functions

Tool Icon Description
Show all layers Makes all the layers in the current drawing visible.
Hide all layers Hides all layers in the active drawing.
Add a layer Adds a new layer.
Remove the current layer Removes the selected layer.
Modify layer attributes Modify attributes of the layer, including the layer name.

Individual layer functions

Icon Description
/ Show/hide the layer.
/ Lock/unlock the layer for editing.
Allow the layer to be included in the printed drawing.

Toggle construction lines. When enabled, all lines and line segments will appear as having infinite length.
