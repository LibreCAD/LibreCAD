Layer List dock

The Layer List Dock provides the functions to manage layers and a list of layers in the current drawing. The entry line on top of the dock allows to filter by layer names (for example,“*01” would show all names ending by “01”).

Layer List Dock functions

Tool Icon Description
Show all layers Makes all the layers in the current drawing visible.
Hide all layers Hides all layers in the active drawing.
Add a layer Adds a new layer.
Remove the current layer Removes the selected layer.
Modify layer attributes Modify attributes of the layer, including the layer name.

Individual layer functions

Icon Description
/ Show/hide layer.
/ Lock/unlock layer.
Print layer.
/ Toggle construction lines.
Shows the current layer color (default is black).
