Library Browser dock widget

The Library Browser dock widget shows blocks available from the defined libraries. You can add drawings to the Library Browser as well.

LibreCAD includes many blocks in the Library Browser for different drawing applications: general shapes, electronics, decision trees, planning, and power. Additionally, the Library Block includes a sheets folder that contains title block templates for various paper sizes.
