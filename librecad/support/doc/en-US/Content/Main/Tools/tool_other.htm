Additional tools: Text, Hatch, Point

Tool Icon Command Description
MText mtxt, mtext

Insert multi-line text into drawing at a specified base point. Optionally define font, text height, angle, width factor, alignment, angle, special symbols, and character set.

 

Read more about using MText.

 
Text txt, text

Insert single-line text into drawing at a specified base point. Optionally define font, text height, alignment, angle, special symbols, and character set.

 

Read more about using Text.

 
Hatch ha, hatch

Fill a contour (polygon, circle, polyline, and so forth) with a defined pattern or a solid fill. If the entity is an open profile, the hatch pattern may be drawn incorrectly. Optionally define scale and angle.

 

Hatch pattern appendix.

 
Point po, point Draw a point by selecting a place in the drawing or entering coordinates in the Command Line.
