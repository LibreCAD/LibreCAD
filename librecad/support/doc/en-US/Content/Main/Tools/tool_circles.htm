Circle tools

Tool Icon Command

Description

Center, Point

 ci, circle Draw a circle with a given radius by assigning a center point and a point on the circumference.
2 Points c2, circle2 Draw a circle with a given diameter by assigning two opposite points on the circumference.
2 Points, Radius none

Draw a circle with two points on the circumference and with an assigned radius.

Tool options:
Center, Diameter cd

Draw a circle with a given diameter by assigning a diameter and center point.

Tool options:
3 Points c3, circle3

Draw a circle assigning three points on the circumference.
Center, Radius none

Draw a circle with a given radius centered at an assigned point on the circumference.

Tool options:
Tangential, 2 Circles, 1 Point none Draw a circle tangential to two existing circles and assigning a center point to establish the radius.
Tangential, 2 Points none

Draw a circle tangential to an existing circle and define the diameter and placement by assigning two points on the circumference.
Tangential, 2 Circles, Radius none

Draw a circle tangential to two existing circles with a given radius.

Tool options:
Tangential, 3 Circles ct3, tan3 Draw a circle tangential to three existing circles and/or lines.
