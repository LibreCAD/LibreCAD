About Draw and Modify Tools

In LibreCAD, use Tools to add geometry, dimensions, and text to your drawing and to make modifications to any existing entities in the drawing. See below for an overview of each tool.

 

Geometry

  • Lines
  • Polylines
  • Circles
  • Curves
  • Ellipses

Additional tools

  • Dimensions
  • Text, points, hatches
  • Select
  • Modify
  • Monitor the Command Line dock and Tool Options toolbar (above the drawing) for feedback or to enter parameters as you use a tool.
  • You may need to select the Enter key to complete a tool operation.
