About Draw and Modify Tools

In LibreCAD, use Tools to add geometry, dimensions, and text to your drawing and to make modifications to any existing entities in the drawing. Select from the list below for an overview of each tool.

 

Geometry

  • Lines
  • Polylines
  • Circles
  • Curves
  • Ellipses

Additional tools

  • Dimensions
  • Text, points, hatches
  • Select
  • Modify
