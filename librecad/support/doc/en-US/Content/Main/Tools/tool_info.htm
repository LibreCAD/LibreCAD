Informational tools

All distances and areas are calculated and returned in the Command line window.

 

Quickly get total linear length of entities by selecting one or more entities. This will update the "Selected Total Length" quantity and length values at the bottom of the application.

Tool Icon Command Description
Distance Point to Point dpp, dist Provides distance in both Cartesian and polar coordinates between two specified points.
Distance Entity to Point none Provides the shortest distance between the selected entity and a specified point.
Angle between two Lines ang, angle Provides angle between two selected line segments, measured counter-clockwise.
Total length of selected Entities none Provides total length of one or more selected entities (length of line segment, circle circumference, and so forth).
Polygonal Area ar, area Provides area and circumference of polygon defined by three or more specified points.
