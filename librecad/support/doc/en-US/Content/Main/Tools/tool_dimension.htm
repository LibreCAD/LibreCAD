Dimensioning tools

Tool Icon Command Description
Aligned da

Apply dimension lines and text aligned to an existing entity by selecting start and end points on a line segment and a placement point for the text.

Tool options:
Linear dr

Apply dimension lines and text at a defined angle to an entity by selecting start and end points on a line segment and a placement point for the text.

Tool options:
Horizontal dh

Apply dimension lines and text to an entity to label the horizontal length of the entity by selecting start and end points on a line segment and a placement point for the text.

For example, a vertical line at 90째 will have a dimension of length 0, and any line that is not perfectly vertical at 90째 will have a dimension length greater than 0.

Tool options:
Vertical dv

Apply dimension lines and text that are aligned to an entity by selecting start and end points on a line segment and a placement point for the text.

For example, a horizontal line at 180째 will have a dimension of length 0, and any line that is not perfectly horizontal at 180째 will have a dimension length greater than 0.

Tool options:
Radial dimradial

Apply dimension lines and text to a circle or arc radius by selecting an entity and a placement point for the text.

Tool options:
Diametric dimdiameter

Apply dimension lines and text on the diameter of a circle or arc by selecting an entity and a placement point for the text.

Tool options:
Angular dimangular

Apply angular dimension by selecting two existing non-parallel line segments and a placement point for the text.

Tool options:
Leader ld Draw a text leader by selecting start (arrow location), middle and end points.
