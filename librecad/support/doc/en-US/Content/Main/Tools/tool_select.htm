Select all entities on visible layers

 

Keyboard shortcut: CTRL+A
(De-)Select Contour none Select or deselect one or more entities (default cursor action).
Select Window none Select or deselected entities connected by shared points.
Deselect Window none Select one or more entities enclosed by selection window (Left to Right), or crossed by selection window (Right to Left) (default cursor drag action).
Select Intersected Entities none Select one or more entities crossed by selection line.
Deselect Intersected Entities none Deselect one or more entities crossed by selection line.
(De-)Select Layer none Select or deselected all entities on the layer of the selected entity.
Invert Selection none

Select all unselected entities while deselecting all selected entities.
