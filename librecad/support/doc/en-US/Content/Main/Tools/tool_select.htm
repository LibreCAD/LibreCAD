Select tools

Tool Icon Command Description
Select an entity none none

Click on an entity to select it.

An entity will not be deselected until it is selected again or until all entities are deselected (Esc key).
Deselect an entity none none Click on a selected entity to deselect it.
Deselect All tn

Deselect all entities on visible layers

Keyboard shortcut: Ctrl+K or Esc.
Select All sa

Select all entities on visible layers.

Keyboard shortcut: Ctrl+A.
Select Contour none none

Finds the shortest path, along intersecting entities, between the first entity selected and the entity closest to the cursor.

Click the first entity, then hold down the Shift key and move the cursor. When an acceptable path preview is shown, click the last entity in the path to select all entities along the path.
(De-)Select Contour none Select or deselect one or more selected entities.
Select Window none Select one or more entities enclosed by selection window (left to right), or crossed by selection window (right to left) (default cursor drag action).
Deselect Window none Deselect one or more entities enclosed by selection window (left to right), or crossed by selection window (right to left) (default cursor drag action).
Select Intersected Entities none Select one or more entities crossed by selection line.
Deselect Intersected Entities none Deselect one or more entities crossed by selection line.
(De-)Select Layer none Select or deselect all entities on the layer of the selected entity.
Invert Selection none

Select all unselected entities while deselecting all selected entities.
