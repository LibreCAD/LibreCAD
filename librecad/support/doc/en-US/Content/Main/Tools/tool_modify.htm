Modify tools

Use these tools to modify existing geometry in your drawing, including moving, scaling, trimming excess geometry, shaping text around entities, and viewing and modifying properties and attributes. Most of the tools described below work on multiple selected entities or a single entity.

 

  • Monitor the Command Line dock and Tool Options toolbar (above the drawing) for feedback or to enter parameters as you use a tool.
  • You may need to select the Enter key to complete a tool operation.
Tool Icon Command Description
Order   none Order entities within a layer. Selected entities can be moved to top, bottom, raised (moved forward) over another entity, or lowered (moved backwards) behind an entity.
Move/Copy mv Move a selected entity by defining a reference point and a relative target point. Optionally keep the original entity (Copy), create multiple copies and / or alter attributes and layer.
Rotate ro Rotate a selected entity around a rotation point, moving the entity from a reference point to a target point. Optionally keep the original entity, create multiple copies and / or alter attributes and layer.
Scale sz Increase or decrease the size of a selected entity from a reference point by a defined factor for both axes. Optionally keep the original entity, create multiple copies and / or alter attributes and layer.
Mirror mi Create a mirror image of a selected entity around an axis defined by two points. Optionally keep the original entity and / or alter attributes and layer.
Move and Rotate none Move a selected entity by defining a reference point and a relative target point and rotating the entity at a given angle. Optionally keep the original entity, create multiple copies and / or alter attributes and layer.
Rotate Two none Rotate a selected entity around an absolute rotation point, while rotating the entity around a relative reference point to a target point. Optionally keep the original entity, create multiple copies and / or alter attributes and layer.
Revert Direction revert Swap start and end points of one or more selected entities.
Trim tm, trim Cut the length of a line entity to an intersecting entity.
Trim Two t2, tm2 Cut the lengths of two intersecting entities to the point of intersection.
Trim Excess tx, tmx, trimex Cut an entity that intersects with another entity. A hover preview displays showing the entity that will be cut.
Lengthen le

Extend the length of a line entity. Click the checkbox to trim the entity by the value entered in the Length field rather than lengthen the entity. Alternatively, enter a negative value in the Lengthen field to trim.

 

Tool options:
Offset o, pa, offset, parallel Copy a selected entity to a defined distance in the specified direction.
Bevel (chamfer) ch, bevel

Create a sloping edge between two intersecting line segments defined by a setback on each segment.

Note: This command is different than beveling in ProNest and is more like a chamfer! Do not use if you are attempting to add beveling to a part's CNC code.

 

Tool options:
Fillet fi, fillet

Create a corner radius between two intersecting line segments with defined radius.

 

Tool options:
Divide di, div, cut Divide, or break, all lines at the selected cutting point.
Stretch ss Move a selected portion of a drawing by first selecting two ends of the drawing or entity and then defining a reference point and a relative target point.
Properties mp, prop Modify the attributes of one or more selected entities, including Layer, Pen color, Pen width, and Pen Line type. Additionally, modify the geometry of selected entities.
Attributes ma, attr Modify the common attributes of one or more selected entities, including Layer, Pen color, Pen width, and Pen Line type.
Explode Text into Letters none Separate a string of text into individual character entities.
Shape Text st, stxt, shapetext

Shape text around a single arc or line entity.

 

Enter an offset value, which is used to determine the position of text with respect to the entity around which it is shaped. Negative offset values will place text (from the text baseline) below the shape entity, and positive offset values will place text above the shape entity.

 

The checkbox will swap the text direction, which may be particularly useful for text shaped around circles and arcs.

 

Tool options:
Unlink Text from Geometry ut, utxt, unlinktxt Dissociates shaped text from the shaping entity. If text is shaped around an arc, the text will automatically explode into individual letters; if text is shaped along a line, the text will remain a text string and will not explode into individual letters.
Explode xp Separate one or more selected blocks or compound entities into individual entities.
Delete Selected er

Delete one or more selected entities.

 

Keyboard shortcut: DEL
