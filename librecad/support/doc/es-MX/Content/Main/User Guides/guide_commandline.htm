Utilizar la Línea de Comandos

La Línea de Comandos ofrece a los usuarios una alternativa a usar el mouse para seleccionar herramientas y dibujar. Usar el teclado para seleccionar herramientas e ingresar coordenadas puede proveer gran velocidad y precisión al crear dibujos.

 

La línea de comandos provee otras características útiles no disponibles al usar el mouse:

  • Entrada multicomando
  • Una calculadora

La línea de comandos se despliega en su propia miniaplicación anclable y consiste en tres componentes

  1. Símbolo del sistema.
  2. Ventana de salida de comando.
  3. Un botón ( ) que despliega un menú desplegable que incluye:
    • Modo de Código Clave encendido/apagado.
    • Cargar un archivo de Comando.
    • Pegar varios comandos.

 

Vea el apéndice para obtener una lista completa de los comandos de la Línea de Comandos.

Ingresar funciones en la Línea de Comandos

Los comandos disponibles para usar se muestran en las páginas de referencia para cada categoría de herramienta. La línea de comandos se activa en varias formas:

  • Comience a escribir cualquier comando (por ejemplo, rect) y luego seleccione Enter.
  • Seleccione la Barra espaciadora, teclee cualquier comando y luego seleccione Enter.
  • Seleccione Ctrl+M, cualquier comando y luego seleccione Enter.
  • Seleccione Shift, cualquier comando y luego seleccione Enter.
  • Con el Modo de Código Clave encendido, teclee un comando de dos letras (por ejemplo, ci). No es necesario seleccionar Enter.
  • Puede utilizar completar con tabulador en la línea de comandos al ingresar comandos. Ingrese un comando parcial, tal como cir, luego seleccione la tecla Tab. En caso de que solamente sea posible un comando desde el comando parcial, el comando completo se desplegará en el campo de entrada de texto. En caso de que más de un comando sea posible, una lista de posibles comandos se desplegará en la ventana de comandos.

Dependiendo del comando que esté utilizando, puede ser necesaria información adicional en la Línea de Comandos o Barra de herramientas.

  • La Barra espaciadora puede seleccionarse en lugar de la tecla Enter cuando se haya habilitado Opciones > Preferencias de Aplicación > Predeterminados > Evaluar comandos cuando esté presionada la Barra espaciadora (este es el ajuste predeterminado).
  • Además de las coordenadas separadas por comas, las coordenadas también pueden ingresarse utilizando el teclado numérico usando el formato X..Y. Por ejemplo, ingresar 10..20 es equivalente a @10,20. Utilizar los dos decimales puede ser más rápido que escribir la coma.
  • El Modo de Código Clave permite el uso de comandos de dos letras y elimina la necesidad de seleccionar Enter después de teclear el comando.
  • Se puede teclear c o u seguido de Enter en lugar de teclear close o close.

Usar la Calculadora de la Línea de Comandos

  1. Ingrese Cal en la línea de comandos para habilitar el Modo Calculadora.
  2. Ingrese las ecuaciones que necesite. El Modo Calculadora también puede usarse para obtener valores, tales como Pi. Los resultados se desplegarán en la ventana de comandos.
  3. Salga del Modo Calculadora ingresando de nuevo Cal en la Línea de Comandos.

Vea la lista de Operaciones y Funciones de Calculadora.

  • La constante Pi se define en LibreCAD como 3,14159265359.
  • Las funciones trigonométricas utilizan radianes (radianes = grados*Pi/180).
