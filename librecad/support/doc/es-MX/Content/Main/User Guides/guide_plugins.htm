Utilizar Plugins

En LibreCAD, los Plugins son comandos especiales que pueden ayudarle a crear, modificar u obtener información acerca de objetos en un dibujo. Los Plugins pueden tener distintos propósitos además de modificar objetos, tales como compatibilidad con ciertos tipos de archivo, por ejemplo el plugin ESRI Shapefile.

 

Nombre Descripción
Alinear Alinee los objetos seleccionados a una referencia definiendo las posiciones finales de 2 puntos iniciales.
Leer puntos ASCII

Lea puntos de un archivo de texto.

Cada línea del archivo es un punto definido por un DI, coordenada X, coordenada Y, coordenada Z y un código opcional. Cada campo puede separarse por una coma, tabulador o espacio. El separador decimal es un punto (.). Los puntos pueden conectarse con una línea, DI o coordenada y los campos de código pueden trazarse como texto.
Partir

Divida una línea o círculo en n secciones.

Puede colocarse una marca en el límite de cada sección para mostrar cada límite. El tamaño de la marca puede definirse como un porcentaje de la longitud del segmento. La línea o el círculo pueden romperse en el límite de cada sección usando la Herramienta partir.
Plugin de engrane Dibuje un engrane seleccionando el centro del engrane y definiendo parámetros tales como cantidad de dientes, módulos y similares.
ESRI Shapefile

Importe archivos de forma de información geoespacial de vectores GIS (mapas).

Nota: El proceso de importación bloqueará LibreCAD hasta que se haya completado y los archivos grandes pueden tomar mucho tiempo.
Lista de objetos Enumere los objetos seleccionados junto con sus propiedades tales como DI, capa, color, tipo de línea, grosor de línea y coordenadas.
Leer archivo PIC Importe diagramas de lenguaje gráfico PIC.
Plugin de trazado

Trace una función matemática o una función paramétrica utilizando el sistema de coordenadas del dibujo.

Se necesitan la fórmula, el valor de inicio, el valor final y el valor de paso. El trazado pueden ser líneas, una polilínea o una curva.
Mismas propiedades

Aplique las propiedades de un objeto de referencia a los objetos seleccionados.

Las propiedades modificadas son capa, color, tipo de línea y grosor de línea.
Plugin de muestra Dibuje una línea al especificar las coordenadas X e Y de los puntos finales.
