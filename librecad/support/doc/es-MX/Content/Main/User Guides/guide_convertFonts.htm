Convertir tipos de letra Microsoft y AutoCAD a tipos de letra LibreCAD

Tipos de letra

Hay muchas opciones de tipos de letra disponibles en LibreCAD, incluyendo los tipos de letra de ProNest. Para utilizar los tipos de letra Microsoft® o tipos de letra AutoCAD®, será necesario que siga los pasos a continuación para convertir esos tipos de letra a un formato legible por LibreCAD.

Si abre dibujos .dwg AutoCAD que utilizan tipos de letra de AutoCAD, será necesario que convierta el tipo de letra para asegurar que llegue a LibreCAD correctamente. El proceso de conversión de tipos de letra convertirá todos los tipos de letra de AutoCAD y deberá hacerse solamente una vez.

Convertir tipos de letra

Tipos de letra de ProNest

LibreCAD incluyen los tipos de letra de ProNest en su lista de tipos de letra. Estos son los mismos tipos de letra listados en la página de Preferencias de tipo de letra de CAD de ProNest, tal como el tipo de letra no lift.

  • Para utilizar los tipos de letra de ProNest, encuentre el tipo de letra en el menú desplegable Tipos de letra en los diálogos Texto y MTexto.
Tipos de letra de Microsoft

Utilizar los tipos de letra predeterminados de Microsoft y personalizar espaciado de letras, palabras e interlineado utilizando el diálogo Convertir Tipos de letra desde los diálogos Texto y MTexto.

Convertir tipos de letra de Microsoft

  1. Haga clic ya sea en Texto o en MTexto.
  2. Haga clic en Convertir Texto () en la esquina superior derecha.
  3. Seleccione un tipo de letra que quiera convertir desde el menú desplegable.
  4. Ajuste el espaciado entre Letras, Palabras e Interlineado, en caso necesario.
    1. En caso de que ajuste el espaciado, puede guardar el espaciado en el tipo de letra predeterminado actual, o puede ajustar el nombre del archivo desde el campo Guardar como sin sobrescribir el tipo de letra predeterminado convertido.
  5. Haga clic en Aceptar.

Asegúrese de seleccionar el tipo de letra convertido desde el menú desplegable Tipo de letra en el diálogo de Texto o de MTexto para usar el tipo de letra una vez convertido.

Tipos de letra de AutoCAD

LibreCAD para ProNest es compatible con tipos de letra de AutoCAD (.shx). Si sus tipos de letra de AutoCAD están almacenados en la carpeta predeterminada de tipos de letra de AutoCAD <C:\Program Files\AutoDesk\AutoCAD 2019\Fonts>, no es necesario llevar a cabo ninguna acción (tenga en cuenta que esta ruta de archivo es para AutoCAD 2019 y puede ser ligeramente distinta dependiendo de la versión de AutoCAD que usted tenga).

Convertir tipos de letra de AutoCAD manualmente

Si tiene tipos de letra de AutoCAD (.shx) almacenados fuera de esta ubicación, será necesario hacer lo siguiente solamente en una ocasión:

  1. Abra el explorador de archivos en su PC.
  2. Busque la carpeta C:\Program Files (x86)\Hypertherm CAM\LibreCAD y ubique el archivo dumpshp.bat.
  3. Haga clic con el botón derecho sobre dumpshp.bat y haga clic en Copiar.
  4. Vaya a la carpeta que contiene sus tipos de letra de AutoCAD.
  5. Haga clic con el botón derecho y clic en Pegar para poner una copia de dumpshp.bat en la carpeta que contiene sus tipos de letra de AutoCAD.
  6. Haga doble clic en dumpshs.bat para correr el archivo. Aparecerá una ventana de comandos y se cerrará automáticamente cuando termine.

Ahora, cualquier tipo de letra de AutoCAD que use en sus dibujos será convertido automáticamente a un formato legible por LibreCAD y podrá utilizar cualquier tipo de letra AutoCAD en un dibujo LibreCAD nuevo.
