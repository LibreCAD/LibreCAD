Bienvenido a LibreCAD para ProNest

LibreCAD para ProNest es un editor CAD dinámico que está integrado en su software de anidamiento Hypertherm. Con LibreCAD, puede fácilmente crear o modificar una geometría, incluyendo texto, cotas, y bloques, así como configurar capas, enviar dibujos a ProNest o editar piezas importadas desde ProNest ¡y más! Utilice LibreCAD como un editor en sí mismo, o úselo para modificar dibujos antes o después de importarlos a ProNest.
