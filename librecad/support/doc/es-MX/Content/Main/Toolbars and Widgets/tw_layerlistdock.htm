Miniaplicación anclable para Lista de Capas

La miniaplicación anclable Lista de Capas provee una lista de todas las capas en el dibujo actual y las funciones para manejar las capas. La línea de entrada sobre la miniaplicación anclable le permite filtrar por nombres (por ejemplo, ,“*01” le mostrará todos los nombres que terminen con “01”).

Funciones de la Lista de Capas

Herramienta Icono Descripción
Mostrar todas las capas Hace visibles todas las capas en el dibujo actual.
Ocultar todas las capas Oculta todas las capas del dibujo actual.
Añadir una capa Añade una nueva capa.
Eliminar capa actual Elimina la capa seleccionada.
Modificar atributos de capa Modifique los atributos de capa, incluyendo el nombre de capa.

Funciones de capa individual

Icono Descripción
/ Mostrar/ocultar la capa.
/ Bloquear/desbloquear la capa para edición.
Permitir incluir la capa en el dibujo impreso.
