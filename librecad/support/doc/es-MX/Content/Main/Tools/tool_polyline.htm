Herramientas de polilínea

Herramienta Icono Comando Descripción
Polilínea pl, polilínea

Dibuje una línea continua abierta o cerrada que consiste en una o más líneas rectas o segmentos de arco definidos por puntos finales y/o radio o ángulo para los arcos.

 

Opciones de herramienta:
Añadir nodo ninguno Añada un nodo a una polilínea existente. Sugerencia: Utilice “Forzar a Objetos” para posicionar el nodo nuevo en el segmento.
Añadir nodo ninguno Agregue uno o más segmentos a una polilínea existente seleccionando la polilínea y luego añadiendo nuevos nodos de punto final.
Eliminar nodo ninguno Elimine el nodo seleccionado de una polilínea existente.
Eliminar entre dos nodos ninguno Elimine uno o más nodos entre nodos seleccionados para una polilínea existente.
Recortar segmentos ninguno Extienda dos segmentos separados, no paralelos, de una polilínea existente para que se intersequen en el nodo nuevo.
Crear polilíneas equidistantes ninguno

Dibuje una cantidad dada de polilíneas paralelas a una polilínea existente seleccionada con una distancia dada entre líneas.

 

Opciones de herramienta:
Crear polilíneas desde segmentos existentes ninguno Cree una polilínea desde dos o más líneas separadas existentes o segmentos de arco que puedan formar un línea continua.
